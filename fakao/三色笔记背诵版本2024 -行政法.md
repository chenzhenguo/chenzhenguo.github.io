## 行政核心考点

## 考点01．行政法基本原则

|               | 具体内容                                                     | 拓展                                                         |
| ------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 合法行政      | 1．法律优用--法已规定不可违：行政行为不能违反法律规定，消极的，最低标准（法律不禁止）；2．法律保留--法无规定即禁止：行政行为要有法律授权，积极的，最高标准（法律要授权）。 | 1．政法的首要准则，其他基本原则都是其延伸和扩展；2．该原则是行政活动和民事活动的主要区别：（法律保留）民事活动法无禁止即可为，行政活动法无授权即禁止；3．该原则的根据是行政机关在政治制度上对立法机关的从属性；（法律优先）4．属于形式法制的范畴。 |
| 合理行政      | 1．公平公正对待；2．考虑相关（符合立法授权的）因素；3．比例原则。 | 1．属于实质法制的范畴；2．比例原则：符合法律目的→措施手段与结果具有适当性→采用损害最小的方式（依次递进）；3．滥用职权就是典型的不考虑相关因素的行为。 |
| 程序正当      | 1．行政公开；                                                | 阳光是最好的防腐剂，公开的例外：国秘、商秘和个人隐私         |
| 2．公众参与； | 公民参与：获得通知权、陈述权、抗辩权、申请权                 |                                                              |
| 3．回避       | 与行政相对人有利害关系的人应该回避                           |                                                              |
| 高效便民      | 1．行政效率；                                                | 遵守法定时限，积极履行职责，确保行政效率。                   |
| 2．便利当事人 | 应优质服务，不得增加相对人的程序负担，方便相对人。           |                                                              |
| 诚实守信      | 1．诚实：行政机关公布的信心应当全面、准确、真实；2．信赖利益保护：（内容如右述）违法赔偿，合法补偿。 | 1．非因法定事由并经法定程序，行政机关不得撤销、变更已生效的行政决定；2．因国家、公共利益或其他法定事由需要撤回／变更行政决定的，应当依法定权限和程序进行，并对相对人的信赖利益损失予以补偿；3．政机关违反程序／非因法定事由违法撤销已生效决定的，应当对相对人因此遭受的损失承担赔偿责任。 |
| 权责一致      | 1．行政效能                                                  | 法律法规应当赋予行政机关执法手段，保证政令有效。             |
| 2．行政责任   | 行政机关违法或不当执法时，应当承担法律责任。                 |                                                              |

## 考点02．行政主体资格判断标准

| 权   | 自己拥有并行使行政职权；1．拥有：排除受委托组织；2．行使：才能成为特定行为的行政机关，一行为中不行使职权，就不是此事的行政机关。--有权无名不一定是行政主体 |
| ---- | ------------------------------------------------------------ |
| 名   | 以自己名义实施行政活动；从其签名盖章上体现。                 |
| 责   | 必须能够独立承担因行政活动而产生的法律责任（复议被申请人、诉讼被告、赔偿义务机关） |

## 考点03．中央行政机关

| 国务院的组成                   | 组成部门                                                     | 1．履行国务院基本行政管理职能；2．全部具有行政主体资格；3．全部享有规章制定权； |
| ------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
|                                |                                                              | 4．设立撤销合并：由国编办提出方案→国务院常委会讨论通过→总理提请全国人大／常委会决定。 |
| 直属机构                       | 1．主管国务院某项专门业务；具有独立的政府管理职能；2．除参事室和机关事务管理局外，绝大多数具有行政主体资格和规章制定权；3．特设直属机构（国资委）；4．设立撤销合并：由国编办提出方案→报请国务院决定。 |                                                              |
| 直属事业单位                   | 1．原则上无行政主体资格；2.3个被授权的具有行政主体资格和规章制定权（证监会、银保监会、气象局）。 |                                                              |
| 办公机构                       | 1．无行政主体资格；2．无规章制定权；3．设立撤销合并：由国编办提出方案→报请国务院决定。 |                                                              |
| 办事机构                       | 1．无行政主体资格；2．无规章制定权。3．设立撤销合并：由国编办提出方案→报请国务院决定。 |                                                              |
| 议事协调机构                   | 1．只议事不办事；2．四无：无机构、无人员、无编制、一般无措施（紧急情况经国务院批准可采取临时措施）；3．除学位委员会、防总、抗震救灾指挥部外，一般无行政主体资格；4．无规章制定权；5．设立撤销合并：由国编办提出方→报请国务院决定。 |                                                              |
| 部委管理国家局                 | 1．由国务院组成部门管理，主管特定业务，行使行政管理职能；2．无规章制定权；3．设立撤销合并：由国编办提出方案→报请国务院批准。 |                                                              |
| 内设机构                       | 1．司级内设：部门提出方案→国编办审核→国务院批准；2．处级内设：部门决定→按年度报请国编办备案。 |                                                              |
| 国务院的行政机构设置和编制管理 | 设置                                                         | 1．机构设置的内容：机构的名称、机构的职能；2．国务院机构的设立撤销合并司级内设机构：部门提出方案→国编办审核→国务院批准；3．国务院机构的设立撤销合并处级内设机构：部门决定→按年度报国编办备案。 |
| 编制                           | 1．机构设置的内容：机构名称、机构职能；2．编制管理的内容：人员数量定额、领导职数；3．国务院行政机构编制的增加和减少：由国务院机构编制管理机关审核方案→报请国务院批准；4．国务院议事协调机构不单独确定编制，所需编制由承担具体工作的国务院行政机构解决。 |                                                              |

## 考点06．公职管理

| 公职取得   | 考录                                                         | 1．范围：担任一级主任科员以下及其他相当职务层次的非领导职务公务员。2．法定条件：具有中国国籍；年满18周岁等。禁止情形：（1）因犯罪受过刑事处罚的；（2）被开除中共党籍的；（3）被开除公职的；（4）依法被列为失信联合惩戒对象。3．体检的项目和标准由中央公务员主管部门会同国务院卫生健康行政部门确定。4．拟录用的公务员名单，公示期不少于5个工作日。5．试用期：1年。6．特殊职位：经省级以上公务员主管部门批准，可以简化程序或采用其他测评办法。 |                                                              |

## 考点07．公务员回避

| 近亲回避 | 一般亲属回避                                                 | 夫妻、直系血亲、三代以内旁系血亲、近姻亲，不得在同一机关担任：1．双方直接隶属于同一领导人员的职务；2．有直接上下级领导关系的职务；3．不得在其中一方担任领导职务的机关从事组织、人事、纪检、监察、审计和财务工作。 |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 特殊回避 | 不得在其配偶、子女及其配偶经营的企业、营利性组织的行业监管或主管部门担任领导职务。 |                                                              |
| 地域回避 | 担任乡级、县级和设区的市级机关及其有关部门的主要领导的。     |                                                              |
| 公务回避 | 1．涉及本人利害关系的；2．涉及亲属关系的；3．其他可能影响公正执行公务的。 |                                                              |
| 离职回避 | 1．辞去公职或者退休的，领导成员和县处级以上领导职务的在离职3年内；2．其他公务员在离职2年内，不得到与原工作业务直接相关的企业或其他营利性组织任职，不得从事与原工作业务直接相关的营利性活动。 |                                                              |

## 考点10．行政法规的制定程序

| 立项     | 1．授权立法：或续或转。（1）国务院根据全国人大及其常委会的授权，可以就法律事项制定行政法规，授权期限一般不得超过5年；（2）期限届满的6个月前，向授权机关报告，并提出是否需要制定有关的法律意见；需要继续授权的，可以提出相关意见，由全国人大及其常委会决定；2．由国务院有关部门报请立项；3．由国务院法制机构拟定年度立法计划报请党中央、国务院审批后向社会公布；4．立法计划中法律项目应当与全国人大常委会的立法规划和年度立法计划相衔接。 |                                                              |

## 考点19．行政处罚的实施规则

| 不处罚的情形   | 1．不满14周岁的；2．违法行为轻微并及时纠正，没有造成危害结果的；3．精神病人；4．超过处罚时效的。【注意】处罚时效的计算终点：违法行为在2年内未被发现的，不再给予处罚。（治安管理处罚是6个月，税收处罚是5年） |
| -------------- | ------------------------------------------------------------ |
| 减免处罚的情形 | 应当从轻或减轻处罚：1．已满14，不满18；2．主动消除或减轻违法行为危害后果的；3．受他人胁迫的；4．配合行政机关查处违法行为有立功表现的。 |
| 刑罚相抵       | 1．行政拘留折抵拘役或有期徒刑；2．罚款折抵罚金。             |
| 一事不再罚     | 当事人的同一违法行为，只违反一个法律规制，不得给予两次以上的行政处罚。 |
| 处罚不成立     | 1．处罚决定前未告知当事人处罚的事实、理由和依据；2．处罚决定前拒绝听取当事人陈述、申辩的。 |

## 考点48．判决

| 一审判决               | 判决结果                                                     | 判决类型                                                     | 具体情形 |
| ---------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | -------- |
| 被告胜诉               | 驳回原告诉讼请求                                             | 1．起诉作为：主体合法＋证据确凿＋适用法律法规正确＋符合程序＋无滥权、越权明显不当的；2．起诉不作为：被告无义务、被告已履行、履行期未满、履行条件不具备。 |          |

| 审理方式             | 一审应当公开审理，但是为了避免涉及国家秘密、商业秘密、个人隐私或法律规定的其他应当保密的政府信息泄露，法院在审理政府信息公开行政案件时，应当视情况采取适当的审理方式。 |                                                              |                                                              |
| 裁判类型             | 积极不作为的判决                                             | 1．应当公开而拒绝或部分拒绝公开的：应当撤销或部分撤销被诉不予公开决定，并判决被告在一定期限内公开→尚需被告调查、裁量的，判决其在一定期限内重新答复。2．内容或形式不当的公开行为：应当判决被告按照申请人要求的内容或法律、法规规定的适当形式提供。3．不予公开可区分处理信息行为：应当判决被告限期公开可以公开的内容。4．不更正信息记录：应当判决被告在一定期限内更正→尚需被告调查、裁量的，判决其在一定期限内重新答复→被告无权更正的，判决其转送有权更正的行政机关处理。 |                                                              |
| 消极不作为的判决     | 无正当理由逾期不予答复的，应当判决被告在一定期限内答复。     |                                                              |                                                              |
| 反信息公开之诉的裁判 | 1．被告公开信息涉及原告商业秘密、个人隐私不存在公共利益等法定事由的，应当判决确认违法，并可以责令被告采取补救措施；造成损害的，根据原告请求依法判决被告承担赔偿责任。政府信息尚未公开的，应当判决行政机关不得公开。2．诉讼期间，原告申请停止公开涉及其商业秘密、个人隐私的政府信息，经审查认为公开该信息会造成难以弥补的损失，并且停止公开不损害公共利益的，可裁定暂时停止公开。 |                                                              |                                                              |
| 驳回诉讼请求         | 1．不属于政府信息、政府信息不存在、依法属于不予公开范围或依法不属于被告公开的；2．申请公开的政府信息已经向公众公开，被告已经告知申请人获取该政府信息的方式和途径的；3．起诉被告逾期不予答复，理由不成立的；4．以政府信息侵犯其商业秘密、个人隐私为由反对公开，理由不成立的；5．要求被告更正逾期自身相关的政府信息记录，理由不成立的；6．无法按照申请人要求的形式提供政府信息，且被告已通过安排查阅资料、提供复制件等适当形式提供的。 |                                                              |                                                              |

## 考点57．司法赔偿的范围

| 刑事司法赔偿                           | 赔偿                                                         | 侵犯人身自由权                                               | 1．错误拘留：违法拘留／合法但超期＋决定撤销案件／不起诉／判决宣告无罪的；2．错误逮捕：采取逮捕措施后→决定撤销案件／不起诉／判决宣告无罪的／其他终止追究刑责的情形； |
| -------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
|                                        |                                                              |                                                              | 【注意】①错误逮捕不区分违法逮捕还是合法逮捕，只要最终无罪了即可；②只要是有罪，即使轻罪重判，也不予赔偿；③针对部分有罪部分无罪的混合情形，对于无罪部分如果符合条件（数罪中部分罪被宣告罪名不成立＋实行监禁的期限超出再审判决确定的期限的），应当予以赔偿；3．错误判决：依照审监程序再审改判无罪（无罪错判），原判刑罚已经执行的；4．免罪的两类人，判决生效之后朋羁押的，应当予以赔偿。 |
| 侵犯生命健康权                         | 1．对有罪公民不应判处死刑而判处死刑并已执行的；2．刑讯逼供或殴打、虐待等行为造成伤亡的；3．违法使用武器、警械的造成伤亡的。 |                                                              |                                                              |
| 侵犯财产权                             | 1．依法查封、扣押、冻结、追缴等措施的；2．依照审判监督程序再审改判无罪，原判罚金、没收财产已经执行的。 |                                                              |                                                              |
| 不予赔偿                               | 1．公民故意虚伪供述／伪造证据／自身行为致害；2．工作人员个人行为。 |                                                              |                                                              |
| 民事、行政司法赔偿范围                 | 赔偿范围                                                     | 1．违法采取排除妨害诉讼强制措施的；2．违法采取保全措施的；3．违法采取先予执行措施的；4．错误执行判、裁定和其他生效法律文书的；5．民事、行政诉讼中司法工作人员侵权的。 |                                                              |
| 不予赔偿的范围                         | 1．申请保全错误；2．申请先予执行后申请人败诉；3．错误执行回转；4．执行标的物错误；5．保管人侵权；6．个人侵权；7．因不可抗力、正当防卫和紧急避险造成损害后果。 |                                                              |                                                              |
| 民事、行政司法赔偿与民事赔偿责任的分担 | 1．数个原因造成同一损害后果的非刑事司法赔偿案件，采取作用力为主，过错程度等其他因素为辅的原则来确定国家赔偿份额；2．受害人对损害结果存在过错的，可减轻赔偿义务机关的责任；3．损失已经在民事诉讼、行政诉讼中获得过赔偿、补偿的，法院对该部分损失不承担国家赔偿责任。 |                                                              |                                                              |

## 考点60．法院赔委会重新处理程序

| 法院重启   | 赔委会赔偿决定生效后，如发现确有错误的，经法院院长决定或上级法院指令，赔委会应当在2个月内重新审查并作出决定，上一级法院赔委会也可直接审查决定。 |
| ---------- | ------------------------------------------------------------ |
| 检察院重启 | 最高检对各级法院赔委会的决定，上级检察院对下级法院赔委会的决定，发现确有错误的，应当向同级法院赔委会提出意见，同级法院应在2个月内重新审查决定。 |
| 当事人申诉 | 请求人或赔偿义务机关可以向上一级法院赔委会提出申诉。         |
