
# 刑诉核心考点

## 考点1．刑事诉讼的基本理念

| 惩罚犯罪与保障人权 | 1．惩罚犯罪：犯罪嫌疑人、被告人；保障人权：犯罪嫌疑人、被告人等诉讼参与人。2．两者是既对立又统一的矛盾。发生冲突时，应当侧重保障人权。 |
| ------------------ | ------------------------------------------------------------ |
| 程序公正与实体公正 | 1．程序公正是指过程的公正，实体公正是指结果的公正。2．诉讼公正包含实体公正和程序公正，应当并重。3．两者是既对立又统一的矛盾。发生冲突时，程序公正优先。 |
| 诉讼效率与司法公正 | 1．公正和效率是诉讼中两大价值目标，公正是首要价值目标；2．公正优先，兼顾效率。 |

## 考点2．刑事诉讼构造

| 特征       | 1．一个国家特定时期的刑诉目的与诉讼构造具有内在的一致性；2．二者均受刑诉法价值观的深刻影响。诉讼价值→影响诉讼目的→决定诉讼构造 |
| ---------- | ------------------------------------------------------------ |
| 弹劾式     | 1．原被告地位平等＋处于主导地位；2．神明裁判。               |
| 纠问式     | 1．法官包揽所有；2．刑讯合法，侦查审判秘密进行；3．原被告是被拷问、追究的对象 |
| 职权主义   | 1．国家专门机关在诉讼中起主导作用；2．利于提高效率，发现真相，程序正义略显不足。 |
| 当事人主义 | 1．当事人在诉讼中居于主导地位；2．适用程序上保障人权的诉讼目的；3．法官消极中立，控辩双方积极主动、平等对抗；4．利于保障民主和程序公正，但容易造成效率低下。 |
| 混合式     | 混合当事人主义和职权主义：以日本为代表。                     |
| 我国       | 1.1979年：超职权主义；2.1996年：吸收当事人主义，保留了职权主义；3.2012年：控辩式庭审改革。 |

## 考点26．强制措施的变更和解除

| 变更             | 1．犯罪嫌疑人、被告人及其法定代理人、近亲属或辩护人有权申请变更强制措施。法院、检察院和公安机关收到申请后，应当在3日以内作出决定；不同意变更强制措施的，应当告知申请人，并说明不同意的理由。2．犯罪嫌疑人、被告人被羁押的案件，不能在本法规定的侦查羁押、审查起诉、一审、二审期限内办结的，对犯罪嫌疑人、被告人应当予以释放；需要继续查证、审理的，对犯罪嫌疑人、被告人可以取保候审或监视居住。 |
| ---------------- | ------------------------------------------------------------ |
| 申请解除主体     | 犯罪嫌疑人、被告人及其法定代理人、近亲属或辩护人             |

### 考点27．附带民事诉讼成立的条件

| 原告           | 1．因犯罪行为遭受物质损失的公民、企业、事业单位、机关、团体等；2．被害人死亡或丧失行为能力的，其法定代理人、近亲属；4．当被害人是未成年人或限制行为能力人时，其法定代理人可以代为提起附带民事诉讼；5．国家财产、集体财产遭受损失＋且受损单位没有提起→检察院在提起公诉时＋可提起附带民事诉讼。 |                                            |

## 考点29．期间

| 重算计算 | 1．在侦查期间，发现另有重要罪行的，自发现之日起重新计算。【注意】另有重要罪行是指与逮捕时的罪行不同种的重大犯罪和同种的影响罪名认定、量刑档次的重大犯罪。2．补充侦查完毕移送检察院或法院后，重新计算审查起诉期限；3．改变管辖的案件，从改变后的机关收到案件之日起重新计算审理期限；4．二审发回原审的案件，从收到发回的案件之日起，重新计算审理期限；5．简易程序转为普通程序审理，从决定转为普通程序之日起重新计算。（区别民诉）【注意】重新计算期间仅适用于公安司法机关的办案期限。 |
| -------- | ------------------------------------------------------------ |
| 不计入   | 1．犯罪嫌疑人不讲真实姓名、住址，身份不明的期间；2．中止审理的期间；（延期审理的期间一般要计入，特殊的如补侦延期不计入）3．对犯罪嫌疑人作精神病鉴定的期间不计入办案期限；（其他的鉴定期间要计入）2．二审法院通知检察院阅卷，检察院应当在1个月内查阅完毕；检察院阅卷时间不计入审理期限。 |

## 考点30．立案条件

| 公诉案件 | 1．有犯罪事实；2．需要追究刑事责任。                         |
| -------- | ------------------------------------------------------------ |
| 自诉案件 | 1．属于自诉案件范围；2．属于本院管辖的；3．被害人告诉；【提示】如果被害人死亡、丧失行为能力或固受强制、威吓等无法告诉，或是限制行为能力人以及固年老、患病、盲、聋、哑等不能亲自告诉，其法定代理人、近亲属告诉或代为告诉的，法院应依法受理。4．有明确的被告人、具体的诉讼请求和证明被告人犯罪事实的证据。 |

## 考点31．立案程序

| 时间         | 拘留逮捕                                                     | 已被拘留或逮捕的犯罪嫌疑人，应当在拘留或逮捕后24小时内讯问。 |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 传唤拘传     | 不得超过12小时，案情特别重大、复杂，需要采取拘留、逮捕措施的，不得超过24小时。 |                                                              |

| 聋哑人       | 应当找个懂手语的翻译。                                       |                                                              |
| 笔录         | 1．讯问笔录应当交犯罪嫌疑人核对，对于没有阅读能力的，应当向他宣读；2．如果记载有遗漏或差错，犯罪嫌疑人可以提出补充或改正；3．犯罪嫌疑人承认笔录没有错误后，应当签名或盖章；侦查人员也应当在笔录上签名；4．犯罪嫌疑人请求自行书写供述的，应当准许，但不得以自行书写代替讯问笔录。 |                                                              |

| 管辖权错误      | 1．审查起诉管辖权错误：案卷材料移送有管辖权的检察院＋通知公安机关。2．侦查管辖权错误：（1）检察院误查了监察委的：及时协商；（2）检察院误查了公安机关的：符合起诉条件的，可以直接起诉；事实不清、证据不足的，应当及时移送；3．检察院审查发现公安和监察委之间误查：（1）如果符合起诉条件，且经征求双方意见后，无不同意见的，可以直接起诉；（2）如果提出不同意见，或事实不清、证据不足的，应当将案件退回移送案件的机关并说明理由，建议其移送有管辖权的机关办理。 |                                                              |

| 发现新犯罪      | 1．退回2次补侦后发现还有新罪：应当移送公安／监察机关侦查；2．已经查清的部分，应直接公诉。 |                                                              |
| 无犯罪事实      | 1．公安移送的                                                | 发现无犯罪事实／刑诉法16条情形之一的＋检察长／检委会决定→应作出不起诉决定。 |
| 2．检察院自侦的 | 发现无犯罪事实／刑诉法16条情形之一的应退回侦查部门，建议撤销案件。 |                                                              |
| 3．监察移送的   | 发现有不起诉情形的＋经上一级检察院批准→不起诉决定，监察机关不服的，可以向上一级检察院复议。 |                                                              |
| 程序倒流        | 1．检察院发现公安机关／监察机关移送的案件的犯罪嫌疑人无犯罪事实／符合刑诉法第16条规定的情形之一的＋经检察长批准→应作出不起诉决定；2．发现犯罪事实非嫌疑人所为＋需要重新调查或侦查的→应作出不起诉决定＋书面说明理由＋将案卷材料退回监察机关或公安机关＋建议公安机关重新调查或侦查；3．检察院负责捕诉部门发现负责侦查部门移送起诉的案件，有上述第1点中情形的，应当退回本院负责侦查的部门，建议撤销案件。 |                                                              |
| 认罪认罚        | 1．犯罪嫌疑人认罪认罚的：检方告知诉讼权利和认罪认罚的后果＋听取犯罪嫌疑人、辩护人、值班律师、被害人及其诉讼代理人的意见＋记录在案；2．犯罪嫌疑人自愿认罪＋同意量刑建议和程序适用的→应当在辩护人或值班律师在场的情况下签署认罪认罚具结书，盲聋哑、半疯傻以及有异议的除外。3．认罪认罚的，检方应当提出量刑建议＋随案移送认罪认罚具结书。 |                                                              |
| 证据调取        | 公安移送                                                     | 1．检察人员审查时可以调取公安机关讯问犯罪嫌疑人的录音、录像并审查相关的录声、录像；2．对于重大、疑难、复杂案件，必要时可以审查全部录音录像。 |

|          | 监察移送                                                     | 认为需要调取有关录音、录像的，可以商监察机关调取。 |
| -------- | ------------------------------------------------------------ | -------------------------------------------------- |
| 检察自侦 | 1．发现侦查部门未按规定移送录音、录像或移送不全的，应当要求补充移送；2．对取证合法性、讯问笔录真实性等产生疑问的，应当针对性审查；3．对于重大、疑难、复杂案件，可以审查全部录音录像。 |                                                    |
| 证据排除 | 1．经审查录音、录像后发现，公安机关、本院负责侦查部门询问不规范，存在违法行为，录音、录像内容与讯问笔录不一致等情形的，应当向公安机关、侦查部门书面提出，要求予以纠正、补正或书面作出合理解释；2．如果发现讯问笔录与讯问犯罪嫌疑人录音、录像内容有重大实质性差异或公安机关、本院负责侦查部门不能补正或合理解释的，该讯问笔录不能作为批准或决定逮捕、提起公诉的依据。 |                                                    |

## 考点45．不起诉

| 分类                                                         | 法定不起诉                                                   | 无犯罪事实或刑诉法第16条的情形。                             |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 酌定不起诉                                                   | 适用条件                                                     | 1．行为已经构成犯罪；2．犯罪情节轻微，依刑法规定不需要判处刑罚或免除刑罚。 |
| 不需判处刑罚或免除刑罚常见情形                               | 1．犯罪嫌疑人在我国领域外犯罪，依照我国刑法规定应当负刑事责任，但在外国已经受过刑事处罚的；2．犯罪嫌疑人又聋又哑或直的；3．犯罪嫌疑人因防卫过当或紧急避险超过必要限度，并造成不应有危害而犯罪的；4．为犯罪准备工具，制造条件的；5．在犯罪过程中自动中止或自动有效地防止犯罪结果发生的；6．在共同犯罪中，起次要或辅助作用的；7．被胁迫、被诱骗参加犯罪的；8．犯罪嫌疑人自首或在自首后有立功表现的。 |                                                              |
| 存疑不起诉                                                   | 1.1次补充侦查后，证据不足，可以作出不起诉决定；2.2次补充侦查后，仍不具备起诉条件的，应当不起诉。 |                                                              |
| 程序                                                         | 宣布与送达                                                   | 1．凡是不起诉的案件，检察院都应当制作不起诉决定书，应当公开宣布，自宣布之日生效；2．送达对象：被害人或近亲属及其诉代；被不起诉人及其辩护人以及被不起诉人的单位；公安机关。 |
| 涉案财物的处理                                               | 对侦查中扣押、冻结的财物解除扣押、冻结，应书面通知解除扣押、冻结。 |                                                              |
| 解除强制措施                                                 | 1．被不起诉人在押的，应当立即释放；2．被采取其他强制措施的，应当通知执行机关解除。 |                                                              |
| 救济                                                         | 公安                                                         | 向原决定机关复议，向上一级复核。（与不批捕的救济程序一样）   |
| 被害人                                                       | 可以向上一级检察院申诉，对申诉不服的可以提起自诉／也可不经申诉直接自诉。 |                                                              |
| 监察                                                         | 监察机关认为不服的，可以向上一级检察院提请复议。             |                                                              |
| 被不起诉人                                                   | 对酌定不起诉向原机关申诉。                                   |                                                              |
| 【注意】省级以下检察院办理自侦案件以及监察委移送案件，检察院拟作不起诉决定的，应当报请上一级检察院批准；公安机关不需要。--自侦案件、监察委案件不起诉的特殊流程 |                                                              |                                                              |

## 考点50．开庭审判前的准备

| 程序               | 1．对有异议的证据，应当在庭审时重点调查；无异议的，庭审时举证、质证可以简化；2．提起附带民事诉讼的，可以调解；庭前会议情况应当制作笔录；3．庭前会议中不直接对非法证据作出排除决定。 |                                                      |
| 出庭要求           | 公诉人                                                       | 检察院应当派员出席法庭支持公诉。                     |
| 刑事被告人         | 普通程序被告人必须到庭，不允许对刑事被告人缺席判决。         |                                                      |
| 被害人、诉讼代理人 | 经传唤或通知未到庭，不影响开庭审理的，法院可以开庭审理。     |                                                      |
| 辩护人             | 经通知未到庭＋被告人同意的→可以开庭审理，但被告人属于应当提供法律援助情形的除外。 |                                                      |

## 考点51．法庭审判

| 开庭审理                   | 1．被告人：必须来；2．辩护人不来：被告同意＋不属于法援案；✓3．被害人、诉代：传唤／通知不来＋不影响开庭审理。 |                                                              |
| -------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 法庭调查                   | 公诉人宣读起诉书                                             | 先由公诉人宣读起诉书；再由附带原告人或其法定代理人、诉讼代理人宣读附带民事起诉状。 |
| 双方当事人陈述             | 被告人、被害人就起诉书指控的犯罪事实分别陈述。               |                                                              |
| 讯问、发问当事人           | 1．公诉人可以就起诉书指控的犯罪事实讯问被告人；2．经审判长准许，其他人员可以向被告人发问；3．经审判长准许，控辩双方可以向被害人、附民原告人发问；4．审判人员可以讯问被告人，向被害人、附民当事人发问。 |                                                              |
| 出示、核实证据（先人后物） | 1．询问证人、鉴定人、专门知识的人；（谁提请通知的，谁先问）2．出示、宣读证据。（谁出示谁先说明，对方再发表意见）【注意】举证方应当向法庭出示物证，让当事人辨认，对未到庭的证人的证言笔录、鉴定人的鉴定意见、勘验笔录和其他作为证据的文书，应当当庭宣读。 |                                                              |
| 证据偷袭                   | 1．公诉人申请出示庭前未移送的证据＋辩方提出异议的→应要求公诉人说明理由→理由成立＋有必要出示的→应准许；2．辩方提出需要做辩护准备的→可以宣布休庭并确定准备时间。 |                                                              |
| 调取新证据                 | 庭审中，申请通知新的证人到庭，调取新的证据，请重新鉴定或勘验的→法庭认为有必要的，应当同意，并宣布延期审理；不同意的，应说明理由并继续审理。 |                                                              |
| 补充侦查                   | 1．法庭发现，有自首、坦白、立功情节＋案卷中没有相关证据的：应通知检察院移送；2．审判期间，被告人提出新的立功线索的：可建议补充侦查；3．补充侦查期满后＋法庭通知＋检察院未移送＋未说明理由→可决定按撤诉处理；4．公诉人发现案件需要补充侦查，建议延期审理的，合议庭应当同意，但建议延期审理不得超过2次。 |                                                              |
| 法院调查核实证据           | 1．情形：法庭对证据有疑问→可以告知补充证据／说明理由→必要时可以休庭调查核实；2．方式：勘验、检查、查封、扣押、冻结→可通知控辩双方到场；3．定罪量刑有关的新证据：应当告知，必要时可直接提取＋通知查阅、摘抄、复制；4．控辩双方补充的及庭外调查核实取得的证据：须经当庭质证才可以作立案根据。→例外：庭外征求意见，控辩双方无异议。 |                                                              |
| 量刑情节调查               | 1．被告认罪的案件，在确认被告人了解起诉指控的犯罪事实和罪名，自愿认罪且知悉法律后果的，法庭调查可以主要围绕量刑和其他有争议的问题进行；2．被告不认罪／做无罪辩护的，法庭调查应当在查明定罪事实的基础上，查明有关量刑事实。 |                                                              |
| 法庭辩论                   | 内容：事实、证据＋适用法律问题（定罪、量刑相关）             |                                                              |
| 最后陈述                   | 1．省略、剥夺、他人代替，均不可；2．未成年人陈述后，法定代理人可以补充陈述；3．可以制止的：反复重复；4．应当制止的：涉密涉私、损害他人及社会公共利益、蔑视法庭或公诉人等；5．应当恢复法庭调查的：提出新事实、新证据＋可能影响正确裁判的；6．应当恢复法庭辩论的：提出新的辩解理由＋可能影响正确裁判的。 |                                                              |
| 评议宣判                   | 变更罪名                                                     | 1．判决事实应与起诉事实一致；2．法院发现新事实：可以建议补充、变更起诉→不变更／7日内不回复的：就起诉指控的事实作出判决。 |
| 是否公开                   | 1．评议一律不公开；2．宣判一律公开：当庭的→5日内送达判决书；定期的→当庭送达判决书。 |                                                              |

## 考点52．审理程序的中断

| 延期处理 | 1．需要通知新证人，调取新物证，重新鉴定／勘验；2．检察院要补充侦查，提出建议的；3．由于当事人申请回避而不能进行审判的。4．变更、追加起诉；5．公诉人对证据收集的合法性进行证明，需要调查核实的；6．简转普的案件，公诉人需要为出席法庭进行准备的；7．检察院在二审中查阅或调阅案卷材料不能在法定期限完成的。【注意】延期审理的时间一般计入审限，特殊情形（如补充侦查）不计入。 |
| -------- | ------------------------------------------------------------ |
| 中止审理 | 1．被告人患有严重疾病，无法出庭的；2．被告人脱逃的；3．自诉人患有严重疾病，无法出庭，未委托诉讼代理人出庭的；4．由于不能抗拒的原因。【注意】中止审理的原固消失后，应当恢复审理；中止审理的期间都不计入审理期限。 |
| 终止审理 | 1．犯罪已过追诉时效期限的；2．经特赦令免除刑罚的；3．依照刑法告诉才处理的犯罪，没有告诉或撤回告诉的；4．犯罪嫌疑人、被告人死亡的；5．其他法律规定免予追究刑事责任的。 |

## 考点53．简易程序

| 适用范围 | 积极条件                                                     | 基层＋一审＋证据足、事实对＋同意简易＋认罪                   |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 消极范围 | 盲聋哑、半疯傻、影响大、不认罪有异议（部分被告即可）、不构罪、辩无罪 |                                                              |
| 审理特点 | 启动                                                         | 1．对符合简易程序条件的案件，法院可以主动决定适用简易程序；2．检察院对于符合简易程序条件的案件，可以建议法院适用简易程序。 |
| 审级     | 只适用一审程序                                               |                                                              |
| 管辖     | 基层法院                                                     |                                                              |
| 审判组织 | 1.3年以下：可独任，可合议；                                  |                                                              |
|          |                                                              | 2.3年以上：必须合议制。【注意】独任审判中，发现可能判刑超过3年的，应当转为合议庭。 |
| 审理期限 | 1.3年以下：20日内审结；2.3年以上：可延长至1个半月。          |                                                              |
| 出庭要求 | 1．公诉人出庭：检察院必须派员出庭；2．辩护人出庭：有辩护人的，应当通知辩护人出庭。【注意】辩护人经通知未到庭，被告人同意的，法院可以开庭审理，但应当提供法律援助情形的除外。 |                                                              |
| 程序简便 | 1．公诉人可以摘要宣读起诉书；2．公、辩、审对被告人的讯问、发问可以简化或省略；3．对双方无异议的证据，可以仅就证据的名称及所证明的事项作出说明；对双方有异议，或法庭认为有必要调查核实的证据，应当出示，质证；4．控辩双方对与定罪量刑有关的事实、证据没有异议的，法庭审理可以直接围绕罪名确定和量刑问题进行。【注意】适用简易程序审理案件，判决宣告前应当听取被告人的最后陈述。 |                                                              |
| 简转普   | 法定事由                                                     | 被可能不构成犯罪的／可能不负刑责的／当庭否认犯罪事实的／事实不清、证据不足的／其他情形的 |
| 审理期间 | 简转普后，审理期限应当从决定转为普通程序之日起计算（重新计算）。【区别】民诉中，简转普的，应该连续计算审理期间。 |                                                              |
| 程序要求 | 转为普通程序的案件，公诉人需要为出席法庭准备的，可以建议法院延期审理。 |                                                              |

## 考点54．速裁程序

| 适用条件 | 1．可能判处3年有期徒刑以下刑罚的案件；2．案件事实清楚，证据确实充分；3．认罪认罚；4．被告同意适用速裁程序。 |                                                              |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 禁止范围 | 1．被告人是盲聋哑，半疯傻；2．被告人是未成年人的；3．案件有重大社会影响的；4．共同犯罪中部分被告人对事实、罪名、量刑建议或适用速裁程序有异议的；5．未就刑事附带民事赔偿达成调解／和解协议的；6．其他不宜适用的情形。 |                                                              |
| 审理特点 | 启动方式                                                     | 1．法院依职权决定；2．检察院可建议：（1）公安机关、犯罪嫌疑人及其辩护人建议适用速裁程序→检察院经审查符合条件的→可建议法院适用速裁程序；（2）公安机关、辩护人未建议适用的→检察院经审查认为符合条件的＋犯罪嫌疑人同意适用的＋可以建议法院适用速裁程序。 |
| 审理法院 | 基层法院                                                     |                                                              |
| 审判组织 | 审判员1人独任                                                |                                                              |
| 送达     | 简化：受普通程序的送达期限的限制                             |                                                              |
| 庭审     | 简化：一般不进行法庭调查和法庭辩论                           |                                                              |
|          | 宣判                                                         | 应当当庭宣判                                                 |
| 审限     | 1．受理之日起10日内审结；2．对可能判处有期徒刑超过1年的，可以延长至15日。 |                                                              |
| 是否派员 | 适用速裁程序审理，检察院应当派员出庭。【提示】凡是公诉一审案件，无论简易还是普通程序，均应当派员出庭。 |                                                              |
| 公诉人   | 公诉人出席速裁程序法庭时，可以简要宣读起诉书指控的事实、证据、适用法律及量刑建议，一般不再讯问被告人。 |                                                              |
| 程序转化 | 1．审理过程中发现：不构成犯罪／不应追究刑事责任的／违背意愿认罪认罚的／否认指控或其他不宣适用速裁程序的情形，应重新审理；2．检察院发现不宜适用速裁程序审理的，应当建议法院转为普通程序或简易程序重新审理；转为普通程序审理的，公诉人需要出席法庭进行准备的，可以建议法院延期审理。 |                                                              |

## 考点55．自诉案件的审理程序

| 受理程序         | 起诉条件                                                     | 详见立案条件                                                 |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 不予受理         | 下列情形之一的，应说服撤诉（不撤诉，裁定不予受理）：1．没必要审：过时效、死亡、撤诉后又起诉的、调解结案，反悔再诉的；2．没法审：不属于自诉范围的、缺乏罪证、被告下落不明的。 |                                                              |
| 驳回起诉         | 已经立案＋缺乏罪证＋无法补充证据的→应说服撤诉／裁定驳回【注意】撤诉或被驳回后，又提出新证据的，应当受理。 |                                                              |
| 救济途径         | 对不予受理／驳回起诉不服的，可以上诉→二审查明不予受理／驳回起诉有错误的→应撤销原裁定，指令一审法院受理／审理。 |                                                              |
| 审理程序         | 1．自诉人承担证明责任；2．被告实施两个以上犯罪行为，分别属于公诉和自诉案件，法院可一并审理。【注意】如果正在审理自诉案件，发现尚未起诉的公诉案件，法院只能移送公安立亲侦查。 |                                                              |
| 审理特点         | 可以调解                                                     | 公转自不适用调解。                                           |
| 可以反诉         | 1．公诉转自诉的案件不适用反诉：2．二审期间，自诉案件当事人提出反诉的，应当告知另行起诉。 |                                                              |
| 可以适用简易程序 | 简易程序并不排斥公诉转自诉案件。                             |                                                              |
| 可和可撤         | 所有自诉案件都能和解和撤诉。1．申请撤诉：撤诉如果确属自愿的，应当准许；2．视为撤诉：自诉人经两次依法传唤，无正当理由拒不到庭的，或未经法庭准许中途退庭的，法院应当决定按自诉人撤诉处理。 |                                                              |
| 审限             | 未羁押                                                       | 适用普通程序审理的被告人未被羁押的自诉案件，应当在立案后6个月内宣判。 |
| 已羁押           | 如果被告人被羁押的，审理期限与公诉案件的相同。（2＋1＋3＋x） |                                                              |
| 自诉案件的可分性 | 被告人可分                                                   | 1．明知有其他共同侵害人，但只对部分侵害人提起自诉的，法院应当受理；2．判决后，又对其他侵害人以同一理由起诉的，不予受理。 |
| 自诉人可分       | 1．共同被害人中只有部分人告诉的，被通知人接到通知后表示不参加诉讼或不出庭的，视为放弃告诉；2．一审宣判后，被通知人就同一事实又提起自诉的，法院不予受理。 |                                                              |

## 考点56．单位犯罪案件的审理程序

| 单位被告人的诉讼代表人 | 谁当诉讼代表人                                               | 原则                                                         | 被告单位的法定代表／主要负责人。 |
| ---------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | -------------------------------- |
| 例外                   | 法定代表人或主要负责人被指控为单位犯罪直接负责的主管人员或因客观原因无法出庭→应由被告单位委托其他负责人／职工（有作证义务的除外） |                                                              |                                  |
| 代表人不出庭的后果     | 1．法定代表／主要负责人拒不到庭的：（1）无正当理由拒不出庭的，可以拘传其到庭；（2）因客观原因无法出庭的，应当要求检察院另行确定诉讼代表人。2．其他人作为诉讼代表人不来的：不论原因，应当要求检察院另行确定诉讼代表人出庭。 |                                                              |                                  |
| 遗漏单位当事人的处理   | 单位犯罪＋检方只作为自然人犯罪起诉→法院应建议对犯罪单位补充起诉→检方仍坚持的，法院应当受理。 |                                                              |                                  |
| 单位财产的处理         | 违法的                                                       | 违法所得及其孳息，应当决定追缴或查封、扣押、冻结。           |                                  |
| 合法的                 | 可以先行查封、扣押、冻结／由被告单位提出担保。               |                                                              |                                  |
| 注销、变更的处理       | 被注销                                                       | 审判期间＋被撤销、注销、吊销营业执照或宣告破产的→直接负责的主管人员和其他直接责任人员应当继续审理。 |                                  |
| 变更的                 | 审判期间＋合并、分立的→应将原单位列为被告单位，并注明合并、分立情况。【提示】对被告单位所判处的罚金以其在新单位的财产及收益为限。 |                                                              |                                  |

## 考点57．上诉和抗诉

|      | 上诉                                                         | 抗诉                                                         |
| ---- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 主体 | 1．独立上诉主体：除被害人以外的所有当事人＋法定代理人；2．非独立的上诉主体：被告的辩护人、近亲属，经其同意。【注意】被害人没有上诉权，被害人及其法定代理人不服地方各级法院一审的判决的，自收到判决书后5日内，有权请求检察院抗诉。 | 一审法院同级的地方检察院【注意】申请抗诉的主体：公诉案件的被害人及其法代（针对判决）。 |
| 理由 | 无需理由，只要不服一审判决即可                               | 一审判决确有错误                                             |
| 形式 | 书面／口头                                                   | 书面                                                         |
| 途径 | 上诉状提交原法院／上一级法院                                 | 抗诉书只能提交原法院                                         |
| 期限 | 判决10日，裁定5日。【注意】附带民诉上诉、抗诉期按照刑诉计算。 |                                                              |
| 撤回 | 1．上诉期内：应当准许；是否再次提出，以期限届满前最后一次意思表示为准；2．上诉期满后：应由二审法院审查，有错的不准。【注意】被判处死立执的被告，在二审开庭后宣告裁判前申请撤回上诉的，应当不予准许，继续按照上诉案件审理。 | 1．抗诉期内：撤回抗诉的，一审法院不再移送案件；2．抗诉期满后：二审法院宣判前撤回的，二审法院可以裁定准许，并通知一审法院和当事人。 |
| 效力 | 1．期满前撤回的一→上诉、抗诉期满后一审裁判生效；2．期满后撤回的→准许裁定书送达上诉人／抗诉机关之日起生效。 |                                                              |

## 考点58．二审审判的原则

| 全面审查原则   | 内涵                                                         | 应当就一审判决认定的事实和法律进行全面审查，不受上诉或抗诉范围的限制。 |                                                              |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 表现           | 共同犯罪                                                     | 1．只有部分被告人上诉的，或检察院只就部分被告人提出抗诉的，二审法院应当对全案进行审查，一并处理； |                                                              |
|                |                                                              |                                                              | 2．如果提出上诉的被告人死亡，其他被告人没有提出上诉，二审法院仍应当对全案进行审查。【注意】死亡的被告人不构成犯罪的，应当宣告无罪；审查后认为构成犯罪的，应当宣布终止审理；对其他同亲被告人仍应当作出判决或裁定； |
| 附带民事诉讼   | 不管对哪个部分上诉或抗诉，既要审查刑事部分，又要审查附带民事部分。 |                                                              |                                                              |
| 上诉不加刑原则 | 内涵                                                         | 二审法院审判只有刑事被告方上诉的案件，不得加重被告人的刑罚。【注意】控方上诉、抗诉的，可以加重被告人的刑罚。 |                                                              |
| 对象           | 谁被检察院抗诉或谁被自诉人上诉的案件，谁就可以被加重刑罚。   |                                                              |                                                              |
| 不能加刑       | 指的是不能加刑期、不能加刑种、不能变方式、数罪不能加某罪。【注意】①罪名不当的，在不加重原判刑罚的情况下，可以改变罪名；②二审法院不能直接加重被告人的刑罚，也不得假借事实不清或证据不足发回原审加重刑罚；（不能变相加刑）③二审法院发回重审的案件，原审法院原则上不得加重被告人的刑罚（除非新事实，检方补充起诉的）。 |                                                              |                                                              |
| 纠错           | 仅被告人上诉后，必须依法改判的，应在二审判决、裁定生效后，按照审判监督程序。 |                                                              |                                                              |

## 考点59．二审审理

| 二审方式       | 应开庭                                                       | 1．一审上诉案件对认定的事实、证据提出异议，可能影响定罪量刑的；2．被告人被判处死立执的上诉案件；3．检察院抗诉的案件。 |                                  |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | -------------------------------- |
| 不开庭         | 应当讯问被告人，听取其他当、辩、诉代人的意见，合议庭全体成员应阅卷，必要时应当提交书面阅卷意见。 |                                                              |                                  |
| 审理程序       | 地点                                                         | 案件发生地或原审法院所在地进行（也可在二审法院）。           |                                  |
| 阅卷           | 决定开庭审理后及时通知检察院阅卷；检察院应当在1个月以内查阅完毕。--不计入审限 |                                                              |                                  |
| 出庭           | 只要二审开庭审理的公诉案件，同级检察院都必须派检察官出庭，检察官助理可以协助出庭，根据需要可配备书记员。--无论是否抗诉 |                                                              |                                  |
| 特殊案件二审   | 共同犯罪                                                     | 1．共犯二审的全面审查问题；2．共犯死亡的处理问题；3．共犯的上诉不加刑问题。 |                                  |
| 死刑案件       | 检察院办理死刑上诉、抗诉案件，应当讯问原审被告人；应当听取辩护人意见。--必经程序 |                                                              |                                  |
| 自诉案件       | 1．二审反诉→法院应当告知其另行起诉；2．二审调解→应当制作调解书，一审裁判视为自动撤销；3．二审和解→法院裁定准许撤回自诉，并撤销一审判决或裁定。 |                                                              |                                  |
| 附带民诉       | 分开生效                                                     | 只有附民上诉＋上诉期满后→一审刑事判决即发生法律效力。【注意】二审附带民事诉讼案件审结前，可以暂缓送监执行。 |                                  |
| 全案审查       | 审理附民的上诉、抗诉案件，应当对全案进行审查。               |                                                              |                                  |
| 有错误的       | 1．刑上民不上（附民生效）：民事有错，对民事按照审判监督予以纠正；2．民上刑不上（刑事生效）：刑事有错，对刑事按照审监程序进行再审， |                                                              |                                  |
|                |                                                              |                                                              | 并将附民部分与刑事部分一并审理。 |
| 增加请求／反诉 | 二审法院可以调解，调解不成的，告知当事人另行起诉。           |                                                              |                                  |
| 二审审限       | 2+2+X                                                        | 1．应当在2个月以内审结；2．死刑案件或附带民事诉讼的案件，以及有本法规定交、集、流、广情形之一的，经高院批准／决定，可以延长2个月；3．因特殊情况还需要延长的，报请最高法批准。 |                                  |

## 考点60．二审审理结果

| 裁定维持原判 | 事实清楚、证据充分、适用法律正确、量刑适当。【注意】一审虽然量刑过轻，但受上诉不加刑原则的限制，维持原判。 |                                       |
| ------------ | ------------------------------------------------------------ | ------------------------------------- |
| 判决改判     | 可以改判                                                     | 原判决事实不清或证据不足的。          |
| 应当改判     | 仅适用法律有错误或量刑不当的。                               |                                       |
| 裁定发回重审 | 可以发回                                                     | 事实不清楚或证据不足的。--只能发回1次 |
| 应当发回     | 可以反复发回：1．违反公开审判规定的；2．违反回避制度的；3．审判组织不合法；4．剥夺、限制当事人的权利，可能影响公正审判的；5．其他违反法定程序的情形，可能影响公正审判的。 |                                       |

## 考点61．死立执的核准

| 报请程序               | 不上不抗就上报                                               | 一审后不上诉不抗诉，上诉期满10日内报高院复核，高院同意的，报请最高法核准。（层层上报） |                                                              |
| ---------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 上诉抗诉走二审         | 一审后上诉、抗诉的，应当按二审程序审理。1．高院二审裁定维持，裁定后10日内报请最高法核准；2．高院二审改判非死刑，二审终审生效。 |                                                              |                                                              |
| 全面审查               | 共同犯罪案件中，部分被告人被判处死刑的，应当对全案进行审查。 |                                                              |                                                              |
| 讯问被告人             | 复核死刑案件，应当讯问被告人，辩护律师提出要求的，应当听取辩护律师意见。 |                                                              |                                                              |
| 检察监督               | 最高检可以向最高法提出意见；最高法应当将死刑复核结果通报最高检。 |                                                              |                                                              |
| 复核结果               | 裁定核准死刑                                                 | 原判事实、法律正确、量刑适当、诉讼程序合法的。【注意】死刑并无不当＋认定某一事实／引用的法律等不够准确→纠正后＋作出核准死刑的裁判。 |                                                              |
| 裁定不予核准，发回     | 1．原判事实不清、证据不足的；（事实证据问题）2．复核期间出现新的影响定罪量刑的事实、证据的；（新事实证据问题）3．原判认定事实正确，但依法不应当判处死刑的；（量刑问题）4．发现原审违反法律规定的诉讼程序，可能影响正确判决的。（程序问题） |                                                              |                                                              |
| 判决改判（1／N不该死） | 数罪并罚（都是死刑）                                         | 一人犯数个死罪：部分死刑核准＋部分实施正确，但不该判处死刑的改判。（1／n不该死→改判）【注意】部分死罪事实不清、证据不足的，对全亲不予核准，并撤销原判，全亲发回重审。（1／n不清楚→全案发回） |                                                              |
|                        |                                                              | 共同犯罪（都是死刑）                                         | 一个案件有数个死刑被告人：应当死刑的核准＋不该死刑的改判。（1／n不该死→改判）【注意】部分被告人死刑事实不清、证据不足的，对全亲裁定不予核准，并撤销原判，全案发回重新审判。（1／n不清楚→全案发回） |
| 发回重审               | 发回哪里                                                     | 发回二审／一审法院                                           |                                                              |
| 要不要开庭             | 1．发回一审法院：一律开庭；2．发回二审法院的：可以不开庭，但是需查清事实、核实证据、纠正程序违法等必须开庭。 |                                                              |                                                              |
| 重组合议庭             | 发回重新审判的案件，原则上应当另行组成合议庭进行审理。【例外】复核期间有新的事实、证据＋影响定罪量刑／原判事实正确，依法不应判死刑的→发回重审可以不另组合议庭。 |                                                              |                                                              |

## 考点62．再审申诉

| 主体             | 当事人及其法定代理人、近亲属                                 |                                                              |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 对象             | 已经生效的判决、裁定                                         |                                                              |
| 受理机关         | 法院、检察院                                                 |                                                              |
| 效力             | 1．不停止判决、裁定的执行；2．不必然引起审判监督程序。       |                                                              |
| 时间             | 最迟在刑罚执行完毕后2年内提出申诉。【例外】超过2年申诉，具有下列情形之一，应当受理；①可能对原审被告人宣告无罪的；②原审被告人在规定的期限内向法院提出申诉，法院未受理的；③属于疑难、复杂、重大案件的。 |                                                              |
| 法院申诉的程序   | 原则找终审                                                   | 1．原则：终审法院处理；2．例外：撤回上诉的，对一审判决可由一审法院审查。 |
| 越级怎么办       | 向上一级申诉的：可以告知／直接转交／可以直接审查（疑难、复杂、重大） |                                                              |
| 死刑找谁         | 原核准法院审查／（原审法院审查→原审法院应当提出意见后层报原核准法院处理） |                                                              |
| 两级申诉         | 对驳回申诉不服，可以向上一级法院申诉→经审查不符合再审条件的，应说服撤回申诉→仍然坚持申诉的，裁定驳回／通知不予重审。 |                                                              |
| 检察院申诉的程序 | 原则                                                         | 1．由作出生效裁判的法院的同级检察院刑事申诉检察部门办理；2．直接向上级检察院申诉→可以交由作出生效判决、裁定的法院的同级检察院受理；案情重大、疑难、复杂的，上级检察院可以直接受理；3．检察院审查决定不予抗诉＋继续申诉的→上一级检察院应当受理。 |
| 例外             | 对于法院死刑终审判决、裁定尚未执行的申诉，由刑事执行检察机构办理。 |                                                              |

## 考点63．提起审判监督程序的主体

| 法院                     | 本院                                                         | 本院院长对本院已经生效的裁判，发现确有错误，提交审判委员会讨论决定。 |          |
| ------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | -------- |
| 上级法院                 | 提审                                                         | 对下级法院已经生效的裁判，如果发现确有错误，有权提审。       |          |
| 指令再审                 | 对下级法院已经生效的裁判，如果发现确有错误，有权指令再审。【注意】指令再审的，一般应指令原审法院以外的下级法院审理；由原审法院审理更有利，可指令原审法院审理。 |                                                              |          |
| 检察院                   | 最高检对各级法院／上级检察院对下级法院→已经生效的裁判，如果发现确有错误，有权向同级法院提起再审抗诉。 |                                                              |          |
| 二审抗诉和再审抗诉的对比 |                                                              | 二审抗诉                                                     | 再审抗诉 |
| 对象不同                 | 尚未生效的一审裁判                                           | 已经生效的裁判                                               |          |
| 抗诉机关不同             | 原审法院的同级检察院                                         | 原审法院的上级检察院／最高检                                 |          |
| 接受抗诉机关不同         | 抗诉检察院的上一级法院                                       | 抗诉检察院的同级法院                                         |          |
| 提起期限不同             | 二审抗诉期限法定：10／5天                                    | 未作明确规定                                                 |          |
| 效力不同                 | 必然导致一审不发生法律效力                                   | 再审抗诉不会停止原生效裁判的执行                             |          |

## 考点64．再审程序

| 审理程序         | 1．原来一审的，应当按照一审程序进行；2．原来是二审的，或提审的，应当按照二审程序进行。（不能上诉） |                                                              |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 审判组织         | 另行组成合议庭                                               |                                                              |
| 再审决定书       | 除检察院抗诉以外，法院应制作再审决定书。                     |                                                              |
| 再审对执行的影响 | 原则                                                         | 不停止原裁判的执行。                                         |
| 例外             | 可能再审改无罪、再审减刑后刑期届满的→可以决定中止执行的原裁判，必要时采取取保候审、监视居住。 |                                                              |
| 再审的开庭       | 应当开庭                                                     | 按一审程序审、按二审程序需要对事实／证据审理、检方抗诉的、可能加重刑罚的。 |
| 可不开庭         | 被告人、自诉人已死亡／丧失行为能力。                         |                                                              |
| 再审的出庭       | 检察院                                                       | 开庭审理的，同级检察院应派员出庭。                           |
| 同案犯           | 再审决定书／抗诉书，只针对部分被告人的＋其他同案犯不出庭不影响审理的→可不出庭参诉。 |                                                              |
| 中止与终止       | 1．原审被告人（原审上诉人）下落不明或收到抗诉书后未到庭的，法院应当中止审理；2．原审被告人（原审上诉人）到案后，恢复审理；3．如果超过2年仍查无下落的，应当裁定终止审理。 |                                                              |
| 强制措施         | 1．法院决定再审的案件，需要对被告人采取强制措施的，由法院依法决定；2．检察院抗诉的再审案件，需要对被告人采取强制措施的，由检察院依法决定。 |                                                              |
| 审限             | 应当在作出提审、再审决定之日起3个月以内审结，需要延长期限的，不得超过6个月。 |                                                              |
| 再审不加刑       | 除检察院抗诉的以外，再审一般不得加重原审被告人的刑罚。       |                                                              |

## 考点66．罚金、没收财产判决的执行

| 执行主体               | 原则                                                         | 一审法院                                                     |
| ---------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 例外                   | 财产在异地的，可以委托财产所在地的同级法院代为执行。         |                                                              |
| 执行时间               | 罚金                                                         | 1．期内未足额缴纳的，法院应当在期满后强制缴纳；2．确因不能抗拒的原因难以缴纳的，经法院裁定可以延期缴纳、酌情减少或免除。 |
| 没收                   | 对没收财产的执行，法院应当立即执行。                         |                                                              |
| 执行范围               | 1．刑事裁判生效时的合法所有财产；2．保留被执行人及其所扶养家属的生活必需费用。 |                                                              |
| 执行措施               | 1．侦查机关已经采取查封、扣押、冻结→法院应在期满前及时续行；2．侦查机关已查封、扣押、冻结的财产，法院可直接裁定处置，不需通知侦查机关解除。 |                                                              |
| 执行顺序               | 1．判处罚金／没收财产＋又承担民事责任→先履行民事责任部分；2．判处财产型之前＋所负正当债务＋需要偿还的＋经债权人请求→应当偿还；3．被执行人在执行中同时承担刑责和民责，其财产不足以支付的，按照下列顺序执行：（1）人身损害赔偿中的医疗费用；（2）退赔被害人的损失；（3）其他民事债务；（4）罚金；（5）没收财产。 |                                                              |
| 赃款赃物的追缴         | 1．赃款赃物及其收益，应一并追缴；2．赃款赃物投资或置业形成的财产及其收益；3．将赃款赃物与其他合法财产共同投资或置业，形成的财产中与赃款赃物对应的份额及其收益。 |                                                              |
| 涉案财物认定错误的救济 | 向执行法院提出书面异议：1．可通过裁定补正的，执行机构应当将异成材料移送刑事审判部门处理；2．不能补正的，告知通过审判监督程序纠正。 |                                                              |
| 清偿转让的追缴         | 1．第三人明知是涉案财物而接受的；2．第三人无偿或以明显低于市场的价格取得涉案财物的；3．第三人通过非法债务清偿或违法犯罪活动取得涉案财物的；4．第三人通过其他恶意方式取得涉案财物的。 |                                                              |
| 执行回转               | 财产刑被撤销，已执行的财产应当全部或部分返还被执行人；无法返还的，应赔偿。 |                                                              |
| 执行中止               | 1．执行标的物系法院或仲裁机构正在审理案件的争议标的物，需等待该案件审理完毕确权；2．案外人对执行标的物提出异议的。 |                                                              |
| 终结执行               | 1．据以执行的判决、裁定被撤销的；2．被执行人死亡或被执行死刑，且无财产可供执行的；3．被判处罚金的单位终止，且无财产可供执行的；4．依照刑法第53条规定免除罚金的。 |                                                              |

## 考点67．监外执行

| 对象 | 判处有期徒刑、拘役的罪犯                                   |
| ---- | ---------------------------------------------------------- |
| 条件 | 1．严重疾病需要保外就医；2．怀孕／正在哺乳自己婴儿的妇女； |

|          | 3．生活不能自理，适用监外执行不致于危害社会的。              |                                                              |                                                              |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 程序     | 决定                                                         | 执行前                                                       | 交付执行的法院决定。                                         |

| 决定主体 | 法院决定暂予监外执行的，依法应当予以收监的，在法院作出决定后，由公安机关送交执行。 |                                                              |                                                              |

| 死亡     | 监外执行期间死亡的，执行机关应及时通知原关押监狱／其他执行机关。 |                                                              |                                                              |

## 考点68．减刑、假释

| 对象           | 减刑                                                         | 被判处管制、拘役、有期、无期的罪犯。                         |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 假释           | 被判处有期、无期的罪犯。                                     |                                                              |
| 条件           | 减刑条件                                                     | 1．认真遵守监规，接受教育改造，确有悔改表现的，或有立功表现的，可以减刑；2．有重大立功表现的，应当减刑。 |
| 假释条件       | 1．有期徒刑执行了原判刑期1／2以上，无期徒刑实际执行13年以上；2．认真遵守监规，接受教育改造，确有悔改表现，释放后不致再危害社会。 |                                                              |
| 管辖           | 高院                                                         | 无期徒刑的减刑、假释案件由服刑地高院管辖。                   |
| 中院           | 其余刑期的减刑、假释案件由服刑地中院管辖。                   |                                                              |
| 审理           | 公告程序                                                     | 法院应当在立案后5日内向社会公示；公示期限为5日。             |
| 审判组织       | 法院审理减刑、假释案件，应当依法由审判员或由审判员和陪审员组成合议庭。 |                                                              |
| 开庭审理       | 1．重大立功报减刑；2．时间幅度不符司法解释一般规定；3．公示期有意见（检方异议／投诉）；4．职务、黑、金、重 |                                                              |
| 法院纠错       | 1．法院发现本院生效的减刑、假释裁定有误→应当依法重组合议庭审理并裁定；2．上院发现下院生效的减刑、假释裁定有误→应当指令下院另行组成合议庭审理／自行组成合议庭进行审理。 |                                                              |
| 监督           | 事前监督                                                     | 1．应当依法予以减刑、假释的时候，由执行机关提出建议书，报请法院审核裁定，并将建议书副本抄送检察院； |
|                |                                                              | 2．检察院可以向法院提出书面意见。                            |
| 事后监督       | 检察院认为法院减刑、假释裁定不当，应当在收到裁定书副本后20日内，向法院提出书面意见；法院应当在收到意见后1个月以内重组合议庭进行审理，作出最终裁定。 |                                                              |
| 假释考验的后果 | 撤销假释                                                     | 1．假释考验期限内犯新罪或发现漏罪，应当撤销假释；2．假释考验期限内违反相关管理规定，应当撤销假释。【注意】原裁判法院裁定撤销；撤销假释的裁定一经作由，立即生效。 |
| 执行完毕       | 如果无上述情形，假释考验期满，原判刑罚视为执行完毕，并公开予以宣告。 |                                                              |

## 考点69．未成年人案件诉讼程序

| 侦查讯问           | 1．讯问／审判时：（1）应当通知法定代理人到场；（2）法定代理人无法通知／不能到场／是共犯的：可以通知其他成年亲属／所在学校、单位、居住地的基层组织／未成年保护组织的代表到场，并将有关情况记录在案；（3）明确拒绝上述人员到场的，且理由正当的可以准许，应当通知其他合适成年人到场；（4）到场的法定代理人或其他人员认为办案人员在讯问、审判中侵犯未成年合法权益的，可以提出意见。2．讯问女性未成年犯罪嫌疑人，应当有女性工作人员参加；3．询问未成年被害人、证人，适用上述规定。 |                                                              |                                                              |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 附条件不起诉       | 适用条件                                                     | 1．对于犯罪时已满14周岁不满18周岁的未成年人；2．涉嫌刑法分则第四章、第五章、第六章规定的犯罪；3．根据具体犯罪事实、情节，可能被判处1年有期徒刑以下刑罚；4．犯罪事实清楚，证据确实、充分，符合起诉条件；5．具有悔罪表现；6．检察院可以作出附条件不起诉决定。 |                                                              |
| 程序要求           | 听取意见                                                     | 1．检察院在作出附条件不起诉的决定以前，应当听取公安机关、被害人、未成年犯罪嫌疑人的法定代理人、辩护人的意见；2．考验期届满，检察人员应当制作附条件不起诉考察意见书，提出起诉与否的意见，报请检察长决定。考验期届满作出不起诉决定以前，应当听取被害人意见。 |                                                              |
| 送达程序           | 附条件不起诉决定后，应当制作附条件不起诉决定书，并在3日以内送达公安机关、被害人或其近亲属及其诉讼代理人、未成年犯罪嫌疑人及其法定代理人、辩护人。 |                                                              |                                                              |
| 变更措施           | 附条件不起诉后，检察院应当作出释放或变更强制措施的决定。     |                                                              |                                                              |
| 备案程序           | 应在10日内将附条件不起诉决定书报上级检察院主管部门备案。     |                                                              |                                                              |
| 异议方式           | 公安机关                                                     | 可以要求复议、提请复核。                                     |                                                              |
| 被害人             | 1．可以向上一级检察院申诉，但不可以向法院起诉；2．考验期满作出不起诉决定，被害人仍然只能申诉，不能自诉。 |                                                              |                                                              |
| 被不起诉人         | 1．未成年犯罪嫌疑人及其法定代理人对检察院决定附条件不起诉有异议的，检察院应当提起公诉；未成年犯罪嫌疑人及其法定代理人提出无罪辩解的，经审查理由成立的，应当法定不起诉；2．未成年犯罪嫌疑人及其法定代理人对附条件不起诉无异议，仅对所附条件及考验期有异议的，检察院可以采纳合理意见，对考察内容、方式、时间等进行调整；意见不利于帮教的，检察院不予采纳，应当释法说理； |                                                              |                                                              |
|                    |                                                              |                                                              | 3．作出起诉前，犯罪嫌疑人及其法定代理人撤回异议的，可以作出附条件不起诉决定。 |
| 附条件不起诉的考验 | 考验机关                                                     | 检察院                                                       |                                                              |
| 考验时间           | 6个月以上1年以下，从决定之日起计算。                         |                                                              |                                                              |
| 考验义务           | 1．遵守法律法规，服从监督；2．按规定报告自己的活动情况；3．离开所居住的市、县或迁居，应当报经批准；4．按照要求接受矫治和教育。 |                                                              |                                                              |
| 案件结果           | 起诉                                                         | 1．实施新罪的；2．发现决定附条件不起诉以前还有漏罪；3．违反治安管理规定，造成严重后果或多次违反治安管理规定的；4．违反监督管理规定，造成严重后果或多次违反有关附条件不起诉的监管规定的。 |                                                              |
| 不起诉             | 考验期内无上述情形＋考验期满的，检察院应当不起诉。           |                                                              |                                                              |

## 考点70．当事人和解的公诉案件诉讼程序

| 可和解范围   | 1．因民间纠纷引起＋涉嫌刑法分则第四、五章规定的犯罪＋可能判处3年以下刑罚；2．除渎职犯罪以外的＋可能判处7年以下刑罚＋过失犯罪案件。 |                                                              |                                                              |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 和解的条件   | 积极条件                                                     | 1．犯罪嫌疑人真诚悔罪，向被害人赔偿损失、赔礼道歉等；2．被害人明确表示对犯罪嫌疑人予以谅解；3．双方当事人自愿和解，符合有关法律规定；4．属于侵害特定被害人的故意犯罪或有直接被害人的过失犯罪；5．案件事实清楚，证据确实、充分。 |                                                              |
| 消极条件     | 5年以内曾经故意犯罪的，不适用本章规定的程序。                |                                                              |                                                              |
| 和解的主体   | 被害方                                                       | 1．被害人死亡→法定代理人、近亲属；2．被害人系无行为能力人、限制行为能力人的→法定代理人。 |                                                              |
| 被告方       | 1．被告人在押的，其近亲属经被告人同意，可以代为和解；2．被告人系限制行为能力人的，其法定代理人可以代为和解。 |                                                              |                                                              |
| 可以和解阶段 | 侦查                                                         | 公安机关可以向检察院提出从宽处理的建议。（不可以撤销）       |                                                              |
| 起诉         | 1．检察院可以向法院提出从宽处罚的建议；2．对于犯罪情节轻微，不需要判处刑罚的，可以作出不起诉的决定。（酌定不起诉） |                                                              |                                                              |
| 和解         | 和解事项                                                     | 可协商                                                       | 可以就赔偿损失、赔礼道歉等民事责任事项＋是否要求／同意从宽处理进行协商。 |
| 不可协商     | 不得对事实认定、证据采信、法律适用和定罪量刑等依法属于公、检、法职权范围的事宜进行协商。 |                                                              |                                                              |
| 和解协议     | 1．双方和解的：公检法应听取当事人＋其他相关人员意见，审查自愿性、合法性，并主持制作和解协议书；2．和解协议书的主持制作：（1）公安制作的：双方当事人及其他参与人签字；（2）检察院制作的：双方当事人签字（检察院人员不签字，检察院不盖章）；（3）法院制作的：双方当事人＋审判人员签字（法院不盖章）。 |                                                              |                                                              |

|                | 和解协议的异议                                               | 1．经审查，和解自愿、合法的，予以确认，无需重新制作和解协议书；2．和解不具有自愿性、合法性的，应当认定无效。 |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 和解协议的反悔 | 已全部履行＋当事人反悔的→法院不予支持（有证据证明违反自愿、合法原则的除外） |                                                              |
| 和解协议的履行 | 和解协议约定的赔偿损失内容，被告人应当在协议签署后即时履行。 |                                                              |

考点71．缺席审判程序

| 适用条件   | 贪污贿赂犯罪／（严重危害国安、恐怖活动犯罪＋需要及时审判＋经过最高检核准的）＋犯罪嫌疑人、被告人在境外＋监察机关、公安机关移送审查＋检察院提起公诉＋法院审查符合缺席审判程序的→缺席审判 |                                                              |                                                              |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 审判管辖   | 地域管辖                                                     | 犯罪地、被告人离境前居住地或最高法指定的法院                 |                                                              |
| 级别管辖   | 中院                                                         |                                                              |                                                              |
| 审判组织   | 合议庭审理                                                   |                                                              |                                                              |
| 权利保障   | 域外送达                                                     | 法院应当通过国际条约规定的或外交途径提出的司法协助方式或被告所在地法律允许的其他方式，将传票和起诉书副本送达被告人。 |                                                              |
| 辩护权     | 委托辩护                                                     | 法院缺席审判的，被告有权委托辩护人，其近亲属可以代为委托。   |                                                              |
| 强制辩护   | 被告人和近亲属未委托辩护人的，法院应当通知法援机构指派律师为其辩护。 |                                                              |                                                              |
| 救济权     | 独立上诉                                                     | 被告／近亲属不服判决的，有权向上一级法院上诉。               |                                                              |
| 非独立上诉 | 辩护人经被告／近亲属同意，可以向上一级法院上诉。             |                                                              |                                                              |
| 抗诉权     | 检察院认为法院的判决有误的，应当向上一级法院提出抗诉。       |                                                              |                                                              |
| 程序转化   | 缺转普                                                       | 审理中                                                       | 1．审理中，被告自动投案／被抓获的，应当重新审理。2．到案后的处理：（1）审查起诉期间，自动投案或被抓获的，检察院应当重新审查；（2）严重危害国家安全犯罪、恐怖活动犯罪案件报请核准期间，自动投案或被抓获的，报请检察院应当及时撤回报请，重新审查；（3）提起公诉后，法院拟重新审理的，检察院应当商法院将案件撤回并重新审查。 |
| 生效后     | 1．罪犯在裁判发生法律效力后到案的，法院应当交付执行刑罚；2．交付前，法院应当告知罪犯有权对裁判提出异议，提出异议的，应当重新审理。 |                                                              |                                                              |
| 普转缺     | 疾病                                                         | 被告严重疾病无法出庭＋中止审理超6个月＋仍无法出庭＋被告、法定代理人、近亲属申请或同意恢复审理的→法院可以缺席审理。 |                                                              |
| 死亡       | 1．被告死亡的，法院应当裁定终止审理；有证据证明无罪的，法院缺席审理确认的，依法作出无罪判决；2．法院按照审监程序审理的案件：被告人死亡的，法院，依法判决。 |                                                              |                                                              |

## 考点72．违法所得的没收程序

| 适用条件 | 逃匿的 | 贪污贿赂、恐怖活动等重大犯罪（无期以上的）＋逃匿通缉1年后不能到案。 |
| -------- | ------ | ------------------------------------------------------------ |
|          |        |                                                              |

|          | 死亡的                                                       | 犯罪嫌疑人、被告死亡的，依法应当追缴其违法所得及其他涉案财产的。 |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 没收对象 | 实施犯罪所                                                   | 取得财物及其孳息，以及非法持有的违禁品、供犯罪所用的本人财物。 |
| 启动     | 公安                                                         | 公安认为有上述情形的，应当写出没收违法所得意见书，移送检察院。 |
| 检察院   | 可以直接启动违法所得的没收程序。                             |                                                              |
| 法院     | 1．法院不能主动开启违法所得没收程序；2．应当由检察院提出没收违法所得的申请。 |                                                              |
| 审理程序 | 法院管辖                                                     | 犯罪地或犯罪嫌疑人、被告人居住地的中院                       |
| 审理组织 | 组成合议庭                                                   |                                                              |
| 法院公告 | 应当发出公告。（公告期为6个月）                              |                                                              |
| 申请参诉 | 1．时间：公告期间提出；2．嫌疑被告近亲属／其他关系人：有权申请／委托诉讼代理人参加。 |                                                              |
| 开庭与否 | 1．可以不开庭审理，但利害关系人参加诉讼的，法院应当开庭审理。2．开庭审理的，检察院应当派员出席法庭。 |                                                              |
| 举证责任 | 检察院应当承担举证责任。                                     |                                                              |
| 财产保全 | 法院在必要的时候，可以查封、扣押、冻结申请没收的财产。       |                                                              |
| 裁判结果 | 没收                                                         | 属于违法所得及其他涉案财产，除依法返还被害人以外，应当裁定没收。 |
| 驳回     | 不属于应当追缴财产的，应当裁定驳回申请，解除查封、扣押、冻结措施。 |                                                              |
| 裁判效力 | 嫌疑人、被告人、近亲属和其他利害关系人或检察院可以提出上诉、抗诉。 |                                                              |
| 审理期限 | 参照公诉案件一审普通程序和二审程序的审理期限执行。【注意】公告期间和请求刑事司法协助的时间不计入审理期限。 |                                                              |
| 程序转化 | 审理过程                                                     | 1．在没收程序审理过程中，在逃的犯罪嫌疑人、被告人自动投案或被抓获的，法院应终止审理。2．检察院向原受理申请的法院提起公诉的，可以由同一审判组织审理。 |
| 审判阶段 | 1．犯罪嫌疑人、被告人逃匿的--法院应当中止审理；2．犯罪嫌疑人、被告人死亡的--法院应当终止审理。检察院可以向法院提出没收违法所得的申请。检察院向原受理案件的法院提出申请的，可以由同一审判组织审理。 |                                                              |
| 救济     | 人到案                                                       | 1．原裁定正确的，予以维持；不再对涉案财产作出判决；2．原裁定确有错误的，应当撤销原裁定，并在判决中对有关涉案财产一并处理。 |
| 人未到案 | 法院生效的没收裁定确有错误的，应当依照审判监督程序予以纠正。 |                                                              |

## 考点73．强制医疗程序

| 适用条件   | 1．实施了暴力行为，危害公共安全或严重危害公民人身安全；2．经鉴定依法不负刑事责任的精神病人；3．有继续危害社会可能的。 |                                                              |                                                              |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 启动方式   | 公安机关                                                     | 发现符合强制医疗条件的，应当写出强制医疗意见节，移送检察院。 |                                                              |
| 检察院     | 主管部门                                                     | 申请、监督均由检察院捕诉部门办理。                           |                                                              |
| 审查决定   | 1．接到公安移送强制医疗意见书后，30日内作出是否提出申请的决定；2．公安机关移送：审查不符合条件的→应作出不提出强制医疗申请的决定＋向公安机关书面说明理由；3．需要补充证据的→应书面要求公安机关补充证据，必要时也可自行调查。 |                                                              |                                                              |
|            |                                                              | 启动监督                                                     | 发现公安应当启动而不启动的→可要求公安7日以内书面说明理由→公安机关不启动理由不能成立的，应当通知公安启动程序。 |
| 法院       | 法院审理中发现被告人符合强制医疗条件的，可以主动作出强制医疗的决定。 |                                                              |                                                              |
| 审理程序   | 管辖                                                         | 基层法院管辖，以行为地为主，居住地为辅。                     |                                                              |
| 审理方式   | 应当组成合议庭                                               |                                                              |                                                              |
| 强制要求   | 1．审理检察院申请强制医疗的案件，应当会见被申请人；2．应当通知被申请人或被告人的法定代理人到场；3．没有诉讼代理人的，法院应当通知法援机构指派律师为其提供法律帮助。 |                                                              |                                                              |
| 处理决定   | 对于符合强制医疗条件的，应当在1个月以内作出决定。            |                                                              |                                                              |
| 救济途径   | 对强制医疗决定不服的，可以向上一级法院申请复议。复议期间不停止执行强制医疗的决定。 |                                                              |                                                              |
| 检察院监督 | 检察院认为法院作出的强制医疗决定不当，应当在收到决定书后20日以内向法院提出书面纠正意见。 |                                                              |                                                              |

## 考点74．涉外程序与司法协助

| 特殊问题           | 法律适用                                                     | 适用我国刑诉法的规定。 |
| ------------------ | ------------------------------------------------------------ | ---------------------- |
| 权利同等           | 外国人享有我国刑诉法规定的诉讼权利和承担相应义务。           |                        |
| 委托律师           | 外国当事人须委托中国律师辩护／代理。【注意】外国人不能以律师身份辩护，但是可以非律师身份担任辩护人，条件是：被告人的近亲属、监护人。 |                        |
| 会见、探视、旁听   | 1．探视会见的：驻华使领馆官员／监护人、近亲属→受理法院所在地的高院申请→高院安排探视／会见；2．旁听：驻华使领馆官员要求旁听的→可以向受理法院所在地高院申请，法院应当安排。 |                        |
| 翻译问题           | 1．审判使用语言文字：中国语言文字→应当提供翻译；2．诉讼文书：（1）中文为准；（2）不懂的应当提供外文译本＋不加盖印章；3．外国人拒绝翻译／不要外文译本的：本人出具书面说明。 |                        |
| 限制／暂缓出境     | 1．限制出境：针对被告人→书面通知＋扣留证件；2．暂缓出境：必须到庭的证人。 |                        |
| 跨国委托书程序要求 | 外籍当事人从域外寄交或托交给中国律师或中国公民的委托书，必须经所在国公证机关证明＋所在国中央外交机关或其授权机关认证＋我国驻该国使、领馆认证，但我国与该国之间有互免认证协定的除外。 |                        |
| 刑事司法协助       | 主体                                                         | 公检法                 |
| 依据               | 国际条约、协议、互惠原则                                     |                        |
| 内容               | 1．狭义的：代为送达法律文书、询问证人和鉴定人、物品移交以及提供有关法律咨询；2．广义的：引渡。 |                        |
