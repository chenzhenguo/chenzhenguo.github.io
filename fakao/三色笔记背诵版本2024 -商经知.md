## 考点04．公司的设立方式

## 考点22．合伙的产生

| 合伙人资格 | 1．合伙人可以是自然人，也可以是法人或其他组织。【注意】非法人组织也可以。2．普通合伙人数量2个以上，有限合伙人数量2个以上50个以下；3．至少有1个普通合伙人，仅剩有限合伙人的应当解散，仅剩普通合伙人的应当转为普通合伙企业；4．无民事行为能力人和限制民事行为能力人不得成为普通合伙人； |

|            | 5．国有独资公司、国有企业（含全民）、上市公司、公益性事业单位和社会团体不得成为普通合伙人，但可成为有限合伙人。 |
| 合伙人出资 | 1．有合伙人认缴或实缴的出资；【注意】出资要求：不要求形式、实缴、数额，非货币财产需要全员参与确定价值。2．普通合伙人可以劳务出资，其评估办法只能由全体合伙人协商确定；但有限合伙人不得以劳务出资；3．有限合伙人应按期足额缴纳出资，未按期足额缴纳的，应承担补缴义务，并承担违约责任。 |
| 营业执照   | 1．有合伙企业的名称和生产经营场所；2．营业执照签发日期，为合伙企业成立日期；3．登记事项：名称，主要经营场所，执行事务合伙人，经营范围，合伙企业类型，合伙人姓名或名称及住所、承担责任方式、认缴或实缴的出资数额、缴付期限、出资方式和评估方式、合伙期限等。 |
| 合伙协议   | 1．合伙协议依法由全体合伙人协商一致，以书面形式订立；2．修改或补充合伙协议，应当经全体合伙人一致同意，另约除外。 |

## 考点25．合伙的消亡

| 解散                                                         | 1．合伙期限届满，合伙人决定不再经营；2．合伙协议约定的解散事由出现；3．全体合伙人决定解散；4．合伙人已不具备法定人数满30天；5．合伙协议约定的合伙目的已经实现或无法实现；6．依法被吊销营业执照、责令关闭或被撤销；7．法律、行政法规规定的其他原因（如有限合伙企业仅剩有限合伙人）。 |

## 考点26．个人独资企业

| 设立     | 1．仅由一个自然人投资设立；2．投资人以其个人财产对企业债务承担无限责任；仅在企业设立登记时明确以其家庭共有财产作为个人出资的，才依法以家庭共有财产对企业债务承担无限责任；3．不具有独立的法人资格，不缴纳企业所得税。 |

## 考点28．破产原因

| 破产原因         | 不能清偿到期债务＋资不抵债or不能清偿到期债务＋明显缺乏清偿能力 |
| ---------------- | ------------------------------------------------------------ |
| 到期不还         | 债权债务关系依法成立，债务履行期限已经届满，债务人未完全清偿债务。 |
| 资不抵债         | 资产负债表，审计报告，资产评估报告等。                       |
| 明显缺乏清偿能力 | 1．资金严重不足、财产不能变现，无法清偿债务；2．法定代表人下落不明且无他人管理资产，无法清偿债务；3．法院强制执行，无法清偿债务；4．长期亏损，扭亏困难，无法清偿债务。 |
| 异议不成立       | 1．债务人以连带责任人未丧失清偿能力为由，主张不具备破产原因的；2．以申请人未预先交纳诉讼费用为由，对破产申请提出异议的；3．债务人以具有清偿能力或资产超过负债为由提出异议，但不能立即清偿或达成和解的；4．债务人对债权数额提出异议，如果债务人对双方无争议的数额部分已经丧失清偿能力；5．债务人对债权是否存在担保等提出异议，因其不影响破产原因的成立；6．债务人财产市价变化导致破产原因消失的，不影响破产案件的受理与继续审理，法院不得驳回申请。 |

## 考点34．撤销权

| 管理人的撤销权   | 行为发生在破产受理前1年内（唯一标准，不论第三方善恶意），造成债务人财产的不当减少：1．放弃债权；2．无偿转让财产；3．以明显不合理价格交易（互相返还后受让人从债务人处求偿已支付价款→共益债务）；4．对没有财产担保的债务提供财产担保。【注意】事后补充，担保在破产受理前1年内设定，可撤销。5．提前清偿破产受理时仍未到期的债务。【注意】危机期间个别清偿，破产受理前已到期，仍可撤销。 |
| ---------------- | ------------------------------------------------------------ |
| 债权人的撤销权   | 管理人未行使上述撤销权的，债权人可行使撤销权追回破产财产，1年内仍是唯一标准：1．放弃债权；2．无偿转让财产3．以明显不合理价格交易（互相返还后受让人从债务人处求偿已支付价款→共益债务）。 |
| 对外债权诉讼时效 | 1．债务人对外债权的诉讼时效自破产受理之日起中断；2．债务人无正当理由未行使的到期债权在破产受理前1年内超过诉讼时效，自破产受理之日起重新计算。 |
| 危机期间个别清偿 | 1．危机期间（破产受理前6个月内发生破产原因的）仍对个别债权人进行清偿的，管理人有权申请撤销（使债务人财产受益的除外）；2．危机期间债务人对有财产担保的债权人进行个别清偿，管理人不能撤销（担保财产价值低于债权额除外）；3．危机期间因履行生效的法律文书进行的个别清偿，管理人不能撤销（恶意串通除外）；4．危机期间使债务人财产受益的个别清偿不能撤销。 |
| 无效行为         | 应当追回并计入债务人财产，无时间限制。1．债务人为逃避债务而隐匿、转移财产；2．债务人虚构债务或承认不真实债务。 |

## 考点41．票据抗辩及补救

| 票据抗辩                                                     | 对物抗辩                                                     | 又称绝对抗辩或客观抗辩；基于票据这个“物”本身所作的，与票据当事人之间的关系无关，票据债务人可对任何票据债权人所作的抗辩。 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 对人抗辩                                                     | 又称主观抗辩或相对抗辩；基于票据义务人与特定票据权利人之间的一定关系发生的抗辩，抗辩只能对特定票据权利人主张。1．票据债务人可以对不履行约定义务的与自己有直接债权债务关系的持票人进行抗辩； |                                                              |
| 2．票据债务人不得以自己与出票人之间的抗辩事由对抗持票人；【注意】承兑人不得以其与出票人之间的资金关系对抗持票人。3．票据债务人不得以自己与持票人的前手之间的抗辩事由对抗持票人（切断制度），但持票人明知仍受让票据的除外；4．善意的、已付对价的正当持票人可以向任何票据债务人请求付款，不受其前手权利瑕疵和前手相互间抗辩的影响；5．善意的、无对价取得（税收、继承、赠与）票据的持票人仍然享有票据权利，但其享有的票据权利不能优于其前手的权利。【注意】切断制度的例外，义务人对其前手的抗辩理由可以抗辩持票人。 |                                                              |                                                              |
| 票据补救                                                     | 三种救济措施彼此独立且平行，没有前后顺序的要求；票据遗失并不丧失票据权利。 |                                                              |
| 挂失止付                                                     | 1．确定付款人的票据（支票，填明“现金”字样的银行本票，填明“现金”字样和代理付款人的银行汇票，已承兑商业汇票）丢失，适用挂失止付（失票人通知付款人请求其停止票据支付）；2．挂失止付是临时补救措施，只有3天的有效期，票据本身并不因挂失止付而无效；3．挂失止付不是公示催告程序和诉讼程序的必经程序。【提示】挂失止付后承兑行再付款应承担赔偿责任；但承兑行接到出票人直接通知停止付款后仍有付款义务（原因关系不影响票据效力）。 |                                                              |
| 公示催告                                                     | 1．可背书转让的票据丧失，适用公示催告；现金支票、现金本票、现金汇票不得背书转让，不能申请公示催告；2．公示催告程序只是起到催告期间止付的效力，并不直接影响票据的效力；3．票据付款地法院，3日内公告，公告期间不少于60日，届满次日1个月内申请除权，判决1年内可撤销；4．公告期满无人申报债权，经申请人申请法院作出除权判决后，票据才丧失效力，申请人可持除权判决书行使票据权利；5．有人就同一票据申报权利，法院裁定终结公示催告程序，申请人可向对方提起普通民事诉讼。 |                                                              |
| 民事诉讼                                                     | 失票人请求出票人补发票据或请求债务人付款被拒时，以有债权债务关系的出票人、拒绝付款的付款人或承兑人为被告，向被告住所地或票据支付地法院起诉。 |                                                              |

| 受益人的指定   | 1．投保人指定受益人须经被保险人同意，被保险人可以单独指定受益人；2．受益人约定为法定或法定继承人的，以法定继承人为受益人；3．受益人仅约定为身份关系，投保人与被保险人为同一主体的，根据保险事故发生时与被保险人的身份关系确定受益人；4．受益人仅约定为身份关系，投保人与被保险人为不同主体的，根据保险合同成立时与被保险人的身份关系确定受益人；5．受益人的约定包括姓名和身份关系，保险事故发生时身份关系发生变化的，认定为未指定受益人。 |                                                              |
| 受益人的变更   | 1．投保人变更受益人时须经被保险人同意，被保险人可以单独变更受益人（无须投保人同意）；2．变更需要在保险事故发生之前作出；不通知保险人，对保险人不生效。 |                                                              |
|                | 顺序份额                                                     | 未约定受益顺序，未约定受益份额：由其他受益人平均享有。       |
| 受益不能       | 保险金作为被保险人的遗产：1．没有指定受益人，或受益人指定不明无法确定的；2．受益人故意造成被保险人的死亡、伤残，依法丧失受益权，没有其他受益人的；3．受益人放弃受益权，没有其他受益人的；4．受益人先于被保险人死亡，没有其他受益人的。【注意】推定死亡：受益人与被保险人在同一事件中死亡，且不能确定死亡先后顺序的，推定受益人死亡在先。 |                                                              |
| 死亡险         | 宣告死亡特殊规则                                             | 被保险人被宣告死亡之日在保险责任期间之外，但有证据证明下落不明之日在保险责任期间之内，保险人也应当承担赔付责任。 |
| 被保险人的限制 | 1．无行为能力人不入死亡险；2．父母可以为其未成年子女投保死亡险；3．其他履行监护职责的人经父母同意可以为未成年人投保死亡险。 |                                                              |
| 被保险人同意   | 1．被保险人同意是死亡险合同生效的前提；2．被保险人的同意可以在合同订立时作出，也可以在合同订立后追认（未反对即同意）；3．保单再流转（转让、质押）需被保险人书面同意；【注意】父母为未成年子女投保人身险不受限制。4．被保险人以书面形式通知保险人和投保人撤销其同意意思表示的，可认定为保险合同解除。 |                                                              |
| 中止及复效     | 1．投保人自保险人催告之日起超过30日未支付当期保险费，或超过约定的期限60日未支付当期保险费的，合同效力中止，或由保险人按照合同约定的条件减少保险金额；2．保险合同中止后，2年内，经保险人与投保人协商并达成协议，在投保人补交保险费后，合同效力恢复；保险人收到效力恢复的申请后，应于30日内作出答复，否则视为同意恢复效力。 |                                                              |

## 考点46．财产保险合同

| 第三者责任保险                                               | 一般流程                                                     | 被保险人请求，保险人对第三者直接赔偿。                       |                                                              |
| 特殊流程                                                     | 被保险人怠于请求的，受损害的第三者直接索赔。                 |                                                              |                                                              |
| 连带责任                                                     | 1．被保险人因共同侵权依法承担连带责任，保险人不得以该连带责任超出被保险人应承担的责任份额为由，拒绝赔付保险金；2．保险人承担保险责任后，有权主张就超出被保险人责任份额的部分向其他连带责任人追偿。 |                                                              |                                                              |
| 和解协议                                                     | 1．和解协议经保险人认可，保险人在保险合同范围内依据和解协议承担保险责任；2．和解协议未经保险人认可，保险人有权主张对保险责任范围以及赔偿数额重新予以核定。 |                                                              |                                                              |
| 代位求偿制度                                                 | 保护规则                                                     | 1．被保险人具有选择权，可以找第三者索赔，也可以找保险公司索赔；2．订立保险合同前，被保险人已放弃对第三者请求赔偿的权利，放弃行为合法有效，保险人不能就相应部分主张行使代位求偿权；3．保险合同订立时，保险人就是否存在放弃情形提出询问，投保人未如实告知，导致保险人不能代位行使请求赔偿的权利，保险人有权请求返还相应保险金；4．保险公司赔付之前，如果弃权，此时属于被保险人的私权处分，弃权行为能够产生同时豁免第三者和保险公司的赔付义务；5．保险公司赔付之后，在赔付的金额范围内，被保险人向第三者追偿的权利已经让渡给保险公司，被保险人无权放弃，弃权行为无效；6．保险公司就代位求偿事宜通知第三者前，第三者对被保险人重复赔偿，保险公司不得向第三者主张代位求偿，但是可要求被保险人返还保险金；7．保险公司就代位求偿事宜通知第三者后，第三者对被保险人重复赔偿，保险公司有权向第三者主张代位求偿；8．代位求偿诉讼以被保险人与第三者之间的法律关系确定管辖法院。 |                                                              |
| 例外                                                         | 1．第三者如果是被保险人的家庭成员或组成人员，非故意造成的保险事故，保险公司赔付后不得代位求偿；2．人身保险不适用代位求偿制度。 |                                                              |                                                              |

## 考点48．上市公司收购

| 一般规定                     | 实际控制权                                                   | 1．持股≥50％；2．实际支配表决权＞30％；3．决定董事会≥1／2成员选任；4．对股东大会决议产生重大影响。 |            |

## 考点50．投资人保护

| 普通投资者的特别保护   | 1．普通投资者与证券公司的纠纷，证券公司需自证清白（举证责任倒置）；2．普通技资者与证券公司发生纠纷的，证券公司应当证明其行为符合法律、行政法规以及国务院证券监督管理机构的规定，不存在误导、欺诈等情形；证券公司不能证明的，应当承担相应的赔偿责任。 |
| ---------------------- | ------------------------------------------------------------ |
| 公开征集股东权利       | 1．上市公司董事会、独立董事、持有1％以上有表决权股份的股东或投资者保护机构，可以作为征集人；2．征集人可自行或委托证券公司、证券服务机构，公开请求上市公司股东委托其代为出席股东大会，并代为行使提案权、表决权等股东权利（不包括分红权）。 |

## 考点53．滥用市场支配地位

| 支配地位认定       | 经营者本身情况＋其他经营者情况1．该经营者在相关市场的市场份额，以及相关市场的竞争状况；2．该经营者控制销售市场或原材料采购市场的能力；3．该经营者的财力和技术条件；4．其他经营者对该经营者在交易上的依赖程度；5．其他经营者进入相关市场的难易程度；6．与认定该经营者市场支配地位有关的其他因素。 |                                                              |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 市场支配地位的推定 | 1．法律推定市场支配地位的情形：（1）一个经营者在相关市场的市场份额达到1／2的；（2）两个经营者在相关市场的市场份额合计达到2／3的；（3）三个经营者在相关市场的市场份额合计达到3／4的。对（2）、（3）其中有经营者市场份额不足1／10的，不应推定该经营者有市场支配地位。2．被推定具有市场支配地位的经营者，有证据证明不具有市场支配地位的，不应当认定其具有市场支配地位。 |                                                              |
| 法律责任           | 民事责任                                                     | 原告应对被告在相关市场内具有支配地位和其滥用市场支配地位承担举证责任。被告以其行为具有正当性为由进行抗辩的应承担举证责任。 |
| 行政责任           | 责令＋没收＋罚款。                                           |                                                              |

## 考点55．不正当竞争行为

| 混淆行为             | 1．让人（一般消费者）误认为是他人商品或与他人存在特定联系；【注意】蹭热点，傍名牌；即使标明了自己的厂名、厂址、商标，只要引起一般消费者误以为，均构成混淆行为。2．被混淆的对象不要求专利保护、驰名保护等，只要“有一定影响”；3．情节严重的，吊销营业执照。 |
| -------------------- | ------------------------------------------------------------ |
| 商业贿赂行为         | 1．贿赂有影响力的单位或个人，以谋取交易机会或竞争优势；2．未入账；经营者的工作人员进行贿赂的，应当认定为经营者的行为；3．情节严重的，吊销营业执照。 |
| 虚假宣传行为         | 1．经营者不得对其商品作虚假或引人误解的商业宣传，欺骗、误导消费者（美化自己）；2．协助刷单者同等处罚；3．情节严重的，吊销营业执照 |
| 侵犯商业秘密         | 1．不为公众所知悉（秘密性）、具有商业价值（价值性）并经权利人采取相应保密措施（保密性）的商业信息；2．电子侵人；教唆、引诱、帮助；第三人侵权，只要求主观“明知或应知”＋客观“获取、披露、使用或允许他人使用”的行为。 |
| 不正当有奖销售       | 1．抽奖式的有奖销售（不约束附赠式有奖销售），最高奖的金额超过5万元；2．多次抽奖，奖金不累加。 |
| 诋毁商誉             | 1．主体是有竞争关系的经营者；新闻单位被利用和被唆使的，仅构成一般的民事侵权，即属于侵害名誉权行为，而非不正当竞争行为；2．故意编造、传播虚假信息或误导性信息（诋毁对手）；3．商业信誉、商品声誉受损害的竞争对手，可以明确为某一或某几个具体的受害人，也可以为某一类的受害人。 |
| 互联网不正当竞争行为 | 1．经营者不得利用技术手段，通过影响用户选择或其他方式，实施妨碍、破坏其他经营者合法提供的网络产品或服务正常运行的行为（技术干预或控制）；2．插入链接、强制进行目标跳转；误导、欺骗、强迫用户修改、关闭、卸载；不兼容。 |

## 考点58．食品安全法

| 适用范围         | 1．供食用的源于农业的初级产品（食用农产品）的质量安全管理，遵守农产品质量安全法，不适用本法；2．食用农产品的市场销售、有关质量安全标准的制定、有关安全信息的公布和食安法对农业投入品作出规定的（禁止将剧毒、高毒农药用于农业投入品），应当遵守食安法的规定。 |                                                              |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 食品安全标准     | 国家标准                                                     | 1．国务院卫生行政部门（卫健委）会同食品安全监督管理部门制定公布（国标）；2．国务院标准化行政部门提供标准编号。 |
| 地方标准         | 1．省、自治区、直辖市人民政府卫生行政部门制定（省标）；2．适用对象是地方特色食品；3．经国务院卫生行政部门备案后生效；4．食品安全国家标准制定后，该地方标准即行废止。 |                                                              |
| 食品安全控制     | 1．国家对食品生产经营实行许可制度，从事食品生产、食品销售、餐饮服务，应当依法取得许可；但是，销售食用农产品，不需要取得许可；2．生产（销售不管）食品添加剂应当符合法律、法规和食品安全国家标准；3．食品生产经营者应当建立食品安全追溯体系；国家鼓励食品生产经营者采用信息化手段建立该体系；4．患有国务院卫生行政部门规定的有碍食品安全疾病的人员，不得从事接触直接入口食品的工作；5．禁止将剧毒、高毒农药用于蔬菜、瓜果、茶叶和中草药材等国家规定的农作物（农业投入品）；6．食品包装（标签）、说明书和广告，要求真实、全面；7．禁止消费者组织有偿推荐食品；禁止食品安全监督管理部门、食品检验机构、食品行业协会推荐食品；8．使用保健食品原料目录以外原料（新原料）的保健食品和首次进口的保健食品应当经国务院食品安全监督管理部门注册（国家级注册）；9．保健食品的标签、说明书不得涉及疾病预防、治疗功能，并声明“本品不能代替药物”；10．婴幼儿配方食品生产企业应当将食品原料、食品添加剂、产品配方及标签等事项向省、自治区、直辖市人民政府食品安全监督管理部门备案（省级备案）；11．婴幼儿配方乳粉的产品配方应当经国务院食品安全监督管理部门注册（国家级注册）；12．不得以分装方式生产婴幼儿配方乳粉，同一企业不得用同一配方生产不同品牌的婴幼儿配方乳粉。 |                                                              |
| 食品召回制度     | 主动召回                                                     | 1．生产者召回：生产者原因；召回的食品并非绝对不能再上市销售；2．经营者召回：经营者原因。 |
| 责令召回         | 县级以上人民政府食品安全监督管理部门可以责令召回或停止经营。 |                                                              |
| 食品安全事故处置 | 三级应急预案                                                 | 1．国家级：国务院组织制定国家食品安全事故应急预案；2．地方级：县级以上地方政府制定本行政区域的食品安全事故应急预案，并报上一级政府备案；3．企业级。 |
| 机制             | 通报制度                                                     | 1．事故发生单位有义务立即采取措施，防止事故扩大；2．事故单位和接受病人进行治疗的单位有义务及时向事故发生地县级人民政府食品安全监督管理部门、卫生行政部门报告；3．一般按照食品安全事故应急预案逐级上报（特情有绿色通道）。 |
| 食品安全事故调查 | 1．县级以上人民政府应当立即成立事故处置指挥机构，启动应急预案；2．设区的市级以上人民政府食品安全监督管理部门进行事故责任调查；3．涉及两个以上省、自治区、直辖市的重大食品安全事故由国务院食品安全监督管理部门组织事故责任调查。 |                                                              |
| 法律责任         | 行政责任                                                     | 1.1年内累计受到3次非责令停业及吊销许可证的处罚的，责令停业直至吊销许可证；2．企业吊销许可证，相关人员5年禁职；3．个人从事食品安全类犯罪行为，终身禁职；4．（善意免罚）食品经营者履行了进货查验等义务，有充分证据证明其不知道所采购的食品不符合食品安全标准，并能如实说明进货来源的，可以免除（行政）处罚。 |
| 民事责任         | 1．民事赔偿优先；2．消费者因不符合食品安全标准的食品受到损害的，可以向经营者要求赔偿损失，也可以向生产者要求赔偿损失；3．接到消费者赔偿要求的生产经营者，应当实行首负责任制，先行赔付，不得推诿，可以追偿。 |                                                              |
| 惩罚性赔偿       | 1．生产（无过错责任）不符合食品安全标准的食品或经营明知（过错责任）是不符合食品安全标准的食品（不要求损害结果），消费者除要求赔偿损失外，还可以向生产者或销售者要求支付10倍价款或损失3倍（共计4倍）的赔偿金；增加赔偿的金额不足1000元的，为1000元；2．但是，食品的标签、说明书存在不影响食品安全且不会对消费者造成误导的瑕疵的除外。 |                                                              |
| 无效抗辩         | 1．“知假买假”抗辩无效；2．“赠品”抗辩无效；3．有检验合格证抗辩无效。 |                                                              |

## 考点70．劳动法

| 建设项目的环评         | 1．可能造成重大环境影响的，应当编制环境影响报告书（审批），进行全面评价；2．可能造成轻度环境影响的，应当编制环境影响报告表（审批），进行分析或专项评价；3．对环境影响很小、不需要进行环境影响评价的，应当填报环境影响登记表（备案）。 |                                                              |
| 规划与项目环评间的关系 | 1．建设项目的环评，应当避免与规划的环评相重复；2．作为一项整体建设项目的规划（重合），按照建设项目进行环评，不进行规划的环评；3．已经进行了环评的规划包含具体建设项目的，规划的环评结论应当作为建设项目环评的重要依据，建设项目环评的内容应当根据规划的环评审查意见予以简化。 |                                                              |
| 环评程序               | 1．建设项目的环境影响报告书内容中删除了行政主管部门审查同意以及行业部门预审的内容；2．环境影响报告书应当包括预防或减轻不良环境影响的对策和措施；3．涉核、涉密、跨省、国批需国务院级审批；4．建设项目的性质、规模、地点、采用的生产工艺或防治污染、防止生态破坏的措施发生重大变动的，建设单位应当重新报批建设项目的环境影响评价文件。 |                                                              |

## 考点74．环境保护法

| 基本制度           | 三同时                                                       | 建设项目中防治污染的设施，应当与主体工程同时设计、同时施工、同时投产使用。 |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |

|                    | 保护税                                                       | 人；2．向依法设立的污水集中处理、生活垃圾集中处理场所排放应税污染物，或在符合国家和地方环境保护标准的设施、场所贮存或处置固体废物的，不属于直接向环境排放污染物；3．超标或不达标间接排放也需缴纳环境保护税；4．农业生产（不包括规模化养殖）排放应税污染物的，流动污染源（车船航空器）排放应税污染物的，暂予免征环境保护税。 |

## 考点75．森林法

| 权属制度 | 1．森林资源属于国家或集体所有，林木权益属于营造者所有；2．权属争议，个人之间、个人与单位之间由乡镇或县级以上人民政府依法处理，单位之间由县级以上人民政府依法处理；争议解决前，当事人任何一方不得砍伐有争议的林木或改变林地现状。 |
| -------- | ------------------------------------------------------------ |
| 经营管理 | 1．公益林严格保护，商品林自主经营；2．可以合理利用公益林林地资源和森林景观资源，适度开展林下经济、森林旅游等；3．标准内占用林地→县级以上林业部门审批；超标准占用林地→林业＋自然资源主管部门双审批；4．公益林只能进行抚育、更新和低质低效林改造性质的采伐；商品林严格控制皆伐面积，伐育同步 |

规划实施；自然保护区的林木，禁止采伐，实验区的竹林除外；5．采伐林地上的林木应当申请采伐许可证；采伐自然保护区以外的竹林，农村居民采伐自留地和房前屋后个人所有的零星林木，不需要申请采伐许可证。

## 考点76．矿产资源法

| 权属制度     | 1．矿产资源属于国家所有；2．勘查、开采矿产资源，必须依法分别申请、经批准取得探矿权、采矿权，并办理登记；3．开采矿产资源，必须按照国家有关规定缴纳资源税和资源补偿费；4．探矿权转让：最低投入＋批准；5．采矿权转让：合并、分立、合资、合作、资产出售、变更产权＋批准；6．禁止将探矿权、采矿权倒卖牟利。 |
| ------------ | ------------------------------------------------------------ |
| 勘查开发管理 | 1．国家对矿产资源勘查实行统一的区块登记管理制度；2．规划、重要、特定、大型、海域，中央（地质矿产主管部门）审批，其他由省级审批；3．规划、重要、特定，有计划开采；4．允许个人采挖零星分散资源和只能用作普通建筑材料的砂、石、粘土以及为生活自用采挖少量矿产。 |

## 考点77．著作权的主体

| 一般规则     | 1．自动保护原则：中国作者，享受国民待遇的作者；2．出版保护原则：未与中国签订协议或共同参加国际条约的国家的作者以及无国籍人的作品。 |
| ------------ | ------------------------------------------------------------ |
| 合作作品     | 1．著作权由参与实质创作的合作作者共同享有；2．可以分割使用的（词曲），作者对各自创作的部分可以单独享有著作权；3．不可以分割使用的（画），不能协商一致的，任何一方不得阻止他方行使除转让以外的其他权利，但是所得收益应当合理分配给所有合作作者。 |
| 委托作品     | 1．无约定的，著作权属于受托人；委托人在约定的使用范围内享有使用作品的权利；2．领导讲话，著作权归报告人或讲话人；可以支付给执笔人适当的报酬；3．自传体作品，无约定的，著作权归该特定人物；可以支付给执笔人或整理人适当的报酬。 |
| 职务作品     | 1．指公民为完成单位的工作任务而又未主要利用单位物质技术条件所（个人脑力）创作的作品；2．著作权由作者享有（约定无效）；交付作品之日起2年内，单位有权在其业务范围内优先使用。【注意】2年内，未经单位同意，作者不得许可第三人以与单位使用的相同方式使用该作品。 |
| 演绎作品     | 1．演绎者针对演绎（在已有作品的基础上改编、翻译、注释、整理）作品享有单独著作权；2．对作品进行演绎创作应得到作者的同意；3．第三方使用演绎作品须“双许可＋双付费”。 |
| 汇编作品     | 1．汇编者针对汇编（选择或编排，体现独创性）作品享有单独著作权；2．汇编人汇编有著作权的作品时应当经过原作品著作权人的授权，并支付报酬；3．第三方使用汇编作品须“双许可＋双付费”。 |
| 影视作品     | 1．著作权由制片者享有；2．编剧、导演、摄影、作词、作曲等作者享有署名权及获取报酬的权利；3．演员享有表明表演者身份（不享有著作权或署名权）及获取报酬的权利；4．剧本、音乐等可以单独使用的作品的作者有权单独行使其著作权。【提示】将电影改为舞台剧、连环画，需要制片者和剧本原作者双许可，双付费。 |
| 原件所有     | 1．原件的所有权转移，不视为作品著作权的转移；                |
| 权转移的作品 | 2．原件（绘画、书法、雕塑、摄影）的所有权人享有原件物权及原件展览权；3．作者享有除原件展览权之外的其他的著作权（复制权、复制件展览权）。 |

## 考点79．著作权的使用限制

| 合理使用 | 无需许可，无需付费，著作权人不能通过事先声明排除（时事文章、公众集会上的讲话除外）：1．为个人学习、研究或者欣赏，使用他人已经发表的作品（自学）；2．为介绍、评论某一作品或说明某一问题，在作品中适当引用他人已经发表的作品（共赢）；3．免费表演已经发表的作品，该表演未向公众收取费用，也未向表演者支付报酬（并非义演）；4．对设置或陈列在室外公共场所的艺术作品进行临摹、绘画、摄影、录像（非商业目的）；5．将中国公民、法人或其他组织已经发表的以汉语言文字创作的作品翻译成少数民族语言文字作品在国内出版发行。 |
| -------- | ------------------------------------------------------------ |
| 法定许可 | 无需许可，有偿使用，著作权人可通过事先声明排除：1．报纸期刊转载摘编；；2．音像公司翻唱；［并非翻录（盗版）］3．广播电台、电视台播放他人已发表的（文字）作品、已经出版的录音制品（不含影视作品）；4．编写出版国家规划教科书。 |

## 考点81．专利权的程序

| 申请原则         | 先申请原则                                                   | 先到先得，同一日申请的，协商不成的，驳回各方申请，各自作为商业秘密。 |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 禁止重复授权原则 | 1．同样的发明创制造只能授予一项专利权；2．同一申请人同日对同样的发明创造，可以既申请实用新型专利又申请发明专利；（以实用新型占坑）3．先获得实用新型专利权尚未终止，且申请人声明放弃该实用新型专利权的，可以授予发明专利权。（将实用新型腾坑） |                                                              |
| 优先权原则       | 专利申请人第一次提出专利申请后，在法定期限内（发明和实用新型为第一次申请之日起12个月内），又就相同主题的发明创造提出专利申请的，该在后申请被视为是在第一次提出申请的日期提出的；首次提出专利申请的日期为优先权日。 |                                                              |
| 宣告无效         | 1．任何单位或个人在专利权生效之后均可向国家知识产权局专利局申请专利无效；2．宣告无效的专利权视为自始不存在，不具有追溯力；（给过的钱不用退，没给的钱不用给）3．对宣告专利权无效或者维持专利权的决定不服的，可以自收到通知之日起3个月内向法院起诉；（诉讼之前必须经过专利局复审，按行政程序救济）4．法院受理的专利纠纷案件，被告在答辩期间内请求宣告该项专利权无效的，可以不中止诉讼。 |                                                              |

## 考点82．专利权的内容及期限

| 保护期限     | 1．发明专利权自申请日起20年，实用新型专利权和外观设计专利权自申请日起10年；2．临时保护：对于在发明专利（仅限发明专利）申请公布后至专利权授予前（临时保护期间）使用 |                                                              |

|      | 发明而未支付适当费用（不认定为侵权，该费用为补偿性质）的纠纷，专利权人应当在（只能待）专利权被授予之后，请求管理专利工作的部门调解，或直接向法院起诉；诉讼时效自专利权人得知或应当得知他人使用其发明之日起或自专利权授予之日起（孰后）3年。 |

## 考点83．专利侵权

| 善意侵权                             | 1．生产经营目的，使用（使用外观设计除外，因其本身就不侵权）、许诺销售或销售（制造、进口除外，因制造、进口无论是否知情都认定侵权且承担赔偿责任），不知道未经专利权人许可，能证明该产品合法来源的，认定侵权，停止使用／销售并销毁，但无需赔偿，不被行政罚款；2．善意使用者如果支付过合理对价，可以不停止使用。 |                                                      |
| ------------------------------------ | ------------------------------------------------------------ | ---------------------------------------------------- |
| 抗辩                                 | 1．现有技术：                                                | 专利申请日前，相关领域内的公知技术，随便实施不侵权。 |
| 2．先用权原则：                      | 专利申请日前，个别企业在制相同产品或准备，在原范围内继续实施不侵权 |                                                      |
| 3．权利用尽：                        | 由专利权人或经其许可的单位、个人售出后，使用、许诺销售、销售、进口该产品不侵权。【注意】专利权的效力以其在市场上第一次行使权利为限，不得无限期延伸。 |                                                      |
| 4．非商业使用原则：                  | 专为科学研究和实验而使用（如果制造则侵权）有关专利的不侵权。 |                                                      |
| 5．临时过境原则；6．行政审批的需要。 |                                                              |                                                      |

## 考点84．标识

| 组成     | 1．文字、图形、字母、数字、三维标志、颜色组合和声音等；2．同一商品可使用两个以上注册商标。 |
| -------- | ------------------------------------------------------------ |
| 分类     | 商品商标、服务商标、集体商标（集体组织内成员使用，证明成员资格）、证明商标（组织以外的单位或个人使用，证明产品或服务的某些特征）。 |
| 禁用标志 | 有损国家尊严或社会公众利益，有损公序良俗，特殊的地名【提示】作为集体商标、证明商标组成部分or具有其他含义or已经注册使用的除外。 |

## 考点85．驰名商标

| 认定 | 1．驰名商标不一定是注册商标，注册商标也不一定是驰名商标；2．商标局认定or法院司法认定；（仅作为案件事实和判决理由，不写入判决主文）3．禁止将“驰名商标”字样用于宣传。 |
| ---- | ------------------------------------------------------------ |
| 保护 | 1．对未注册的驰名商标“同类保护”；保护措施只包括禁注、禁用、停止侵害等，没有赔偿；2．对已注册的驰名商标“跨类保护”；保护措施包括禁注、禁用、停止侵害等，可以要求赔偿；3．对恶意注册的，驰名商标权利人申请商标无效不受5年的时间限制。--非驰名商标受限 |

## 考点86．商标权的程序

| 注册原则 | 1．申请在先原则：同日使用或均未使用的，不愿协商或协商不成的，以抽签的方式确定；2．自愿注册原则；3．优先权原则：商标注册申请人，自其商标在外国第一次提出商标注册申请之日起6个月内（国际 |
| -------- | ------------------------------------------------------------ |
|          |                                                              |

|          | 优先权），又在中国就相同商品以同一商标提出商标注册申请的，或商标在中国政府主办的或承认的国际展览会展出的商品上首次使用的，自该商品展出之日起6个月内（国内优先权），可以享有优先权。【提示】商标使用许可合同的签订前提是该商标是注册商标（还需向商标局备案），否则合同无效。 |                                                              |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 申请     | 一标多类                                                     | 商标注册申请人可以通过一份申请就多个类别的商品申请注册同一商标。 |
| 代理制度 | 1．中国企业或个人自愿选择，外国企业或个人强制代理；2．委托人申请注册的商标可能存在不得注册情形的，商标代理机构应当明确告知委托人；（可以接受委托）3．商标代理机构知道或应当知道委托人申请注册的商标属于恶意抢注、未经授权的、不以使用为目的的，不得接受其委托；4．商标代理机构除对其代理服务申请商标注册外，不得申请注册其他商标。 |                                                              |
| 注销     | 期满未续，申请注销，权利人死亡或终止（1年内）未转移。        |                                                              |
| 撤销     | 1．自行改变注册商标、注册事项，商标局依职权撤销；2．商标休眠（连续3年不使用），依任何人申请，商标局撤销； |                                                              |
| 宣告无效 | 1．商标本身违法，商标局依职权或经单位或个人申请随时宣告无效；2．注册商标侵犯在先权利，在先权利人申请宣告无效需在商标注册后5年内；3．驰名商标作为在先权利人不受5年限制。 |                                                              |

## 考点88．商标侵权

| 侵权                     | 1．混淆行为：相同商品，相同商标为假冒，商品或商标有类似因素为仿冒；2．善意销售，认定侵权，但无需赔偿，不被行政罚款；使用不侵权；3．伪造、擅自制造他人注册商标标识或销售伪造、擅自制造的注册商标标识的；4．反向假冒：擅自更换其注册商标并将该更换商标的商品又投入市场的；5．故意为侵犯他人商标专用权行为提供便利条件，帮助他人实施侵犯商标专用权行为的。 |
| ------------------------ | ------------------------------------------------------------ |
| 抗辩                     | 1．非商标性使用：特征描述性使用，用途说明性使用；2．先用权：申请商标注册前，他人已经使用，专用权人无权禁止该使用人在原范围内继续使用；3．权利用尽：经商标权人许可或以其他方式合法投放市场的商品，无须经过商标权人许可，即可将带有该商标的商品再次售出或以其他方式提供给公众。 |
| 认定侵权但不承担赔偿责任 | 1．商标休眠：注册商标3年未使用，并且注册商标专用权人不能证明受到其他损失的，侵权人不承担赔偿责任；2．善意侵权：销售不知道是侵犯注册商标专用权的商品，能证明该商品是自己合法取得并说明提供者的，不承担赔偿责任；3．侵犯未注册驰名商标：侵犯未注册驰名商标的，只承担停止侵害、销毁侵权物品等责任，不承担赔偿责任。 |
