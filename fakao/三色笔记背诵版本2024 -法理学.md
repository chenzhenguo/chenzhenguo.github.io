# 理论核心考点

考点01．法的本质

| 正式性     | 1．国家按照法定程序制定或认可；2．由国家强制力保证实施；3．由正式的官方文件加以表现。 |
| ---------- | ------------------------------------------------------------ |
| 阶级性     | 1．法体现的是统治阶级的整体意志和共同意志，而非统治阶级内部各党派集团及每个成员意志的简单相加；2．并非所有法都具有阶级性；3．并非所有法都只体现统治阶级的意志。 |
| 物质制约性 | 法的内容最终受一定社会物质生活条件的决定。                   |

## 考点02．法的特征

| 社会规范       | 社会规范                                                     | 1．调整人与人之间的关系；2．法律规范：由国家制定或认可，并由国家强制力保证实施；3．道德规范宗教规范等其他社会规范，不体现国家意志，没有国家强制力作为后盾，但具有强制性。 |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 技术规范       | 调整人与自然之间的关系                                       |                                                              |
| 自然法则       | 调整自然与自然之间的关系，由自然力保证实施                   |                                                              |
| 效力具有普遍性 | 1．普遍性是指在国家权力所及范围内，具有普遍的约束力；调整对象具有不特定性，反复适用性；2．规范性文件：宪法、法律、法规、规章、司法解释等；--普遍效力3．非规范性文件：结婚证、调解书、法院判决文书、行政处罚决定书、合同、许可证等，只针对特定主体，只有个案效力。 |                                                              |
| 可诉性         | 1．可诉性的判断标志：有诉讼和仲裁；有争议解决；有权利救济；2．只是一种理论上的可能性，并非必然要受到诉讼法的限制。 |                                                              |

## 考点05．法的要素

## （一）法律原则

| 特征       | 内容                                                         | 笼统模糊，假定条件和法律后果都不明确，适用时既关注共性，也关注个性，法官有较大的裁量余地 |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 适用范围   | 具有宏观指导性，适用于某一类行为、某一法律部门甚至全部法律体系。 |                                                              |
| 适用方式   | 以衡量的方式应用于个案；根据原则的分量以及个案的情景判断作用范围，相互冲突的原则可以共存。 |                                                              |
| 分类       | 公理性原则                                                   | 由法律原理构成的原则，是由法律上之事理推导出来的，在国际范围内具有较大的普适性。 |
| 政策性原则 | 一个国家或民族出于一定的政策考量而制定的一些原则，具有针对性、民族性和时代性。 |                                                              |
| 适用       | 特点                                                         | 1．法律原则可以克服法律规则的僵硬性缺陷，弥补法律漏洞，保证个案正义；2．但由于其内涵高度抽象，外延宽泛，所以会赋予法官较大的自由裁量权，从而不能完全保证法律的确定性和预测性。 |
| 条件       | 1．穷尽法律规则，才能适用法律原则；2．除非为了实现个案正义，否则不能舍弃法律规则而直接适用法律原则；3．没有更强理由，不得径行适用法律原则。 |                                                              |

## （二）法律规则

| 特征     | 内容                           | 着眼于共性，目的是削弱自由裁量。 |
| -------- | ------------------------------ | -------------------------------- |
| 适用范围 | 只适用于某一类型的行为。       |                                  |
| 适用方式 | 全有／全无地适用（冲突不共存） |                                  |

| 逻辑结构                                                     | 假定条件                                                     | 生效的时间、空间、对象和情境                                 |                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ---------------------------------------------------- |
| 行为模式                                                     | 权利行为模式                                                 | 可为模式--授权性规则                                         |                                                      |
| 义务行为模式                                                 | 1．应为模式（积极义务）--命令规则2．勿为模式（消极义务）--禁止模式 |                                                              |                                                      |
| 法律后果                                                     | 肯定性后果／否定性后果                                       |                                                              |                                                      |
| 【注意】三个要素在逻辑上缺一不可；但在实践中，法律规则的三要素在法律条文中，都有被省略的可能。 |                                                              |                                                              |                                                      |
| 分类                                                         | 行为模式不同                                                 | 授权性规则                                                   | 可为模式规定人们有权做一定行为或不做一定行为的规则。 |
| 义务性规则                                                   | 1．命令性规则：规定人们的积极义务，即人们必须或应当作出某种行为；2．禁止性规则：规定人们的消极义务（不作为义务），禁止人们作出一定行为。 |                                                              |                                                      |
| 内容具体性                                                   | 确定性规则                                                   | 内容本已明确肯定，无须再援引或参照其他规则来确定其内容的法律规则。 |                                                      |
| 委任性规则                                                   | 内容尚未确定，而只规定某种概括性指示，由相应国家机关通过相应途径或程序加以确定的法律规则。 |                                                              |                                                      |
| 准用性规则                                                   | 内容本身没有规定人们具体的行为模式，而是可以援引或参照其他相应内容。 |                                                              |                                                      |
| 内容强制程序                                                 | 强制性规则                                                   | 内容规定具有强制性质，不允许人们随便加以更改的法律规则。     |                                                      |
| 任意性规则                                                   | 规定在一定范围内，允许人们自行选择或协商确定为与不为的方式以及法律关系中的权利义务内容的法律规则。 |                                                              |                                                      |
| 语言依赖                                                     | 1．法律规则有语言的依赖性：任何语言都需要解释；只要有解释，就有价值判断；2．法律规则的两种语句：（1）规范句：直接规定权利义务的语句；（2）概述句：通过描述的方式，间接规定权利义务的语句；描述句可以改写成规范句。 |                                                              |                                                      |
| 法律规则与法律条文                                           | 法律条文包括规范性条文和非规范性条文。1．规范性条文：法律规则、法律原则，属于法律规范；2．非规范条文：术语界定；公布机关；公布时间；有效日期；属于技术性内容，注意不是技术性规范，因为技术性规范是调整人与自然关系的。【注意】①规则不同于条文：规则是条文所承载的意义；②法律条文和法律规则的关系：并非一一对应，具体情形包括：①一个完整的规则由数个法律条文来表述；②一条规则的内容分别由不同规范性法律文件来表述；③一个条文表述不同的法律规则或其要素；④一个条文仅规定法律规则的某个要求或若干要素。 |                                                              |                                                      |

## （三）法律概念

| 功能                             | 1．法律人适用法律作出判断，需要先对法律概念的意义进行确证的具体化；2．在法律要素上，法律概念可以构成法律规则、法律原则的组成部分；3．法律概念构成了目的论证的界限。 |                                                              |                                              |
| -------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | -------------------------------------------- |
| 分类                             | 根据概念的定义要素是否清                                     | 确定性概念                                                   | 概念本身的语义构成是清晰的。如公民、法人等。 |
| 不确定性概念                     | 1．描述性不确定性概念：不确定由判断标准不明导致；2．规范性不确定性概念：实际适用者的主观评价，本就不存在一个固定的 |                                                              |                                              |
|                                  | 晰                                                           |                                                              | 标准。                                       |
| 根据概念的定义要素之间的关系不同 | 分类概念                                                     | 定义要素之中不存在可以分级的要素，分类概念的所有定义要素对该概念都是充分和必要的，在适用上具有排他性。【注意】分类概念的所有定义要素可以用“和”“并且”以及“或”等词语联结。 |                                              |
| 类型概念                         | 即定义要素中含有至少一个可区分层级的要素的概念。【注意】类型概念具有层级姓和边界的不明确性，在概念上具有开放性和适用的弹性。 |                                                              |                                              |
| 根据概念的功能                   | 描述性概念                                                   | 即描述事实的概念，有真假之分。                               |                                              |
| 评价性概念                       | 即对事实作出价值判断的概念。【注意】评价性概念往往涉及适用者的主观价值判断，无真假之分。 |                                                              |                                              |
| 论断性概念                       | 即基于对某个事实的确认来认定另一个事实的存在的概念。         |                                                              |                                              |

## （四）权利义务

| 法律权利       | 1．即法律规定的行为自由；2．特征：法定性、自主性和利益性、关联性、社会性。 |
| -------------- | ------------------------------------------------------------ |
| 法律义务       | 即法律针对必须作为或不得作为的规定。【注意】义务具备强制履行的性质。 |
| 权利义务的关系 | 1．权利和义务是紧密联系、不可分割的。有权利，就有相应的义务与它同时存在；2．两者的总量是相等的，但具体到某个人时，他所享有的权利与承担的义务不一定相等；3．经历了从浑然一体到分裂对立再到相对一致的过程；4．权利和义务代表了不同的法律精神，它们在历史上受到重视的程度有所不同，在不同国家的法律体系中的地位有主次之分。 |

## 考点07．法律部门与法律体系

| 法律部门         | 1．法律部门之间往往很难截然分开。事实上，有的社会关系需要由几个法律部门来调整；2．单一的规范性法律文件不能包括一个完整的法律部门：大多数规范性法律文件同时包含属于多个法律部门的规范；3．划分法律部门的主要标准是法律所调整的不同社会关系，即调整对象；其次是法律调整方法；4．公法VS私法VS社会法（1）公法旨在限制国家权力，奉行“法无授权即禁止”；（2）私法旨在保护公民权益，奉行“法无禁止即允许”；（3）社会法保护社会利益；（4）公法私法的划分最早由罗马法学家乌尔比安提出，但还没形成普遍可接受的单一的区分标准；（5）社会法是现代社会伴随着“法律社会化”现象的出现而形成，是公私法融合的产物。【注意】私法也要兼顾公共利益。 |
| ---------------- | ------------------------------------------------------------ |
| 法律体系         | 1．法律体系是由部门法构成的，也称作部门法体系；2．法律体系是由一国国内法构成的，不包括完整意义上的国际法，即国际公法；3．构成法律体系的只是一国现行有效的法，不包括已经废止的法律，也不包括尚待制定、还没有生效的法律。 |
| 当代中国法律体系 | 1．当代中国的法律体系主要由七个法律部门和三个不同层级法律规范构成：2．七个法律部门主要是：宪法及宪法相关法、民法商法、行政法、经济法、社会法、刑法、诉讼与非诉法；三个不同层级的法律规范主要是：法律、行政法规、地方性法规、自治条例和单行条例。【注意】宪法相关法主要包括规定如下内容的法律：国家机构的产生、组织和职权。 |

## 考点08．法的效力

| 含义     | 即法的约束力。包括两种情况：1．规范性法律文件的效力：即狭义的法的约束力，仅指法律的约束力；2．非规范性法律文件的效力：指判决书、裁定书、逮捕证等非规范性法律文件的约束力，不具有普遍性。 |                                                              |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 根据     | 十分复杂，一般包括法律本身、道德、社会等方面。               |                                                              |
| 范围     | 对人效力                                                     | 指的是法律对谁有效力，适用哪些人。1．属人主义：指法律只适用于本国公民，不论身在国内还是国外；2．属地主义：指法律适用于该国管辖地区内的所有人，不论是否是本国公民；本国公民不在本国则不受本国法律约束；3．保护主义：指以维护本国利益作为是否适用本国法律的依据。不论国籍和所在地域，只要侵犯了本国利益，均要受到本国法律的追究。 |
| 空间效力 | 一般来说，一国法律适用于该国主权范围所及的全部领域，包括领土、领水及其底土和领空，也可以适用于本国驻外使馆、在外船舶及飞机。 |                                                              |
| 时间效力 | 指法何时生效、何时时效以及有无溯及力。1．法的生效：（1）自法律公布之日起：（2）规定具体生效时间；（3）规定具体生效条件。2．法的失效：（1）明示的废止：即在新法或其他法律文件中明文规定废止旧法； |                                                              |
|          |                                                              | （2）默示废止：即出现新旧法冲突时，适用新法而使旧法事实上被废止。3．法的溯及力：即法溯及既往的效力。指法对其生效之前事件和行为是否适用。（1）法律、法规、规章都以不溯及既往为原则；（2）有些情况为了更好的保障人们的权利，有可能溯及既往。 |

## 考点10．法律责任

| 法律责任概述   | 特点                                                         | 1．最终依据是法律；2．具有国家强制性。                       |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 分类           | 刑事责任／民事责任／行政责任／违宪责任／国家赔偿责任         |                                                              |

## 考点12．法的实施

| 法的实施         | 法的执行（执法）                                             | 1．狭义的执法指行政，即行政机关贯彻执行立法机关制定的法律和其他规范性法律文件的活动；2．性质：以国家的名义对社会进行全面管理，具有国家权威性和国家强制性；3．具有主动性和单方面性，程序性要求没有司法严格和细致；4．执法的主体是国家行政机关及其公职人员。 | 【注意】执法和司法的比较记忆：①执法的范围具有全面性；司法的范围具有特定性，仅限于案件；②执法是执行权，具有主动性；司法是判断权，具有被动性。但是司法也得主动提供“便民服务”；3执法主体是国家行政机关及其公职人员；司法主体是国家司法机关及其公职人员。 |

## 考点13．法适用的一般原理

| 适用目标         | 1．最直接的目标是获得一个合理的法律决定；评价某个决定是否合理，主要包括两方面：该决定是否具有可预测性（形式法治的要求）和正当性（实质法治的要求）；2．可预测性，又称合法性，旨在实现法的安定性，强调严格遵守法律规定，避免恣意和武断；3．可接受性，又称合理性，旨在实现法的合目的性，强调在遵守法律的前提下，综合考虑道德、习惯等。 |
| ---------------- | ------------------------------------------------------------ |
| 可预测性VS正当性 | 1．二者存在紧张关系；这种紧张关系是形式法治和实质法治之间紧张关系的一种体现；2．从整体法治来看，必然要求裁判者在二者之间寻找最佳的协调；3．但对特定时间段内的特定国家的法律人来说，可预测性具有初始的优先性。 |
| 法适用的步骤     | 理论上的步骤：整体来说，法的适用过程在形式上是逻辑三段论推理过程；1．确认小前提：查明案件事实；2．寻找大前提：选择与案件事实相符的法律规范；3．推出结论：从两个前提中推导出法律结论。【注意】①三步骤并非各自独立、严格区分的单个行为，而是可以相互转换“目光流转往返”；②大小前提的区分只是相对区分，不存在绝对的事实判断和纯粹的法律判断，事实的确定离不开法律，法律的解释离不开事实；③考试需要判别事实判断和法律判断，通常而言，法律规定属于法律判断，行为、后果和固果关系属于事实判断。 |
| 法律证成         | 1．所谓的证成就是给法律决定提供充足理由的活动或过程；2．证明法律决定合理性的方式分为外部证成和内部证成；（1）外部证成：对法律决定所依据的前提进行证明，保证推理前提是合理的、正当的；（2）内部证成：按照一定的规则从前提中推出结论，保证结论和推理规则是可靠的；3．内部证成与外部证成的关系：（1）联系：二者都要借助三段论的推理方式；二者都要遵循一定的推理规则；二者都要需求理由来达到说服的目的；（2）区别：内部证成的目的在于结论是否成立，是否符合推理规则；外部证成的目的则在于大小前提是否成立。【解题技巧】只要有法院判决，就一定有法律证成和法律推理。 |

## 考点47．司法与司法制度的概念

| 司法的概念                                 | 演变过程                                                     | 近代以前                                                     | 司法远非一种独立的解纷形态和制度：可能是民间性的调解、仲裁活动，也可能是以国家暴力强制为后盾的官方行为；只要存在社会冲突，即使是古代社会也存在司法。 |
| ------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 近代以后                                   | 1．最初是政治学或法学的概念，代表人物：乔治·劳森，他提出了立法、司法、执法的划分；2．孟德斯鸠第一次全面论述了司法问题和三权分立，1787年被载入美国宪法，进行现实实践；3．司法的概念逐步呈现技术性、程序性特征。 |                                                              |                                                              |
| 实践层面                                   | 1．在三权分立国家，司法与立法、行政相对，司法仅指审判，司法机关仅指法院；检察权隶属于行政，即便设在法院，也受行政指挥；2．我国清末：引进司法制度，司法仅指审判权，检察机关设在法院但归法部领导；孙中山五权分立后，检察机关附属于行政；3．社会主义国家：司法既包括审判，也包括检查；我国受到前苏联影响，亦是如此。 |                                                              |                                                              |

| 被动性                                     | 案件启动及裁判范围依赖于提请或诉求。                         |                                                              |                                                              |
| 交涉性                                     | 整个过程离不开多方利益主体的诉讼参与。                       |                                                              |                                                              |
| 终局性                                     | 1．这是现代司法的根本属性；2．法院、检察院的活动均有终局性。 |                                                              |                                                              |
| 普遍性                                     | 1．形式意义：利用法律解决个案，赋予其普遍的意义；2．实质意义：司法可解决其他机关不能解决的一切纠纷；管辖所有人，范围最广泛；因此司法是纠纷解决最普适的方式；法院是最主要的解决主体。 |                                                              |                                                              |
| 程序性                                     | 法定程序是保证司法机关正确、合法、及时适用法律的前提。       |                                                              |                                                              |
| 司法的功能                                 | 解决纠纷                                                     | 这是司法的主要功能，是审判的主要任务，是司法的普遍特征，它构成司法制度的基础、运作的内容和直接任务，是其他功能发挥的先决条件。 |                                                              |
| 调整社会关系                               | 1．我国司法制度的调整社会关系功能主要通过法院的审判活动实现；2．随着社会的转型，法院的受案范围出现了新的变化，司法在调整社会关系上的功能越来越凸显。 |                                                              |                                                              |

| 形成公共政策                               | 1．现代司法的作用已经不是仅仅局限于就具体纷争事件进行个别表决，而是超越于具体的个别事件，对一般社会主体的利害取向或价值观念，造成事实上的波及和影响；2．我国法院在形成公共政策上的主要表现是，司法对法律与政策没有规定问题的处理方式，需符合法律和政策精神，符合社会公众的愿望，从而促进相关法律、政策的形成。 |                                                              |                                                              |
| 人权保障                                   | 1．司法具有维护和支持其他公权力依法行使、发展人权的作用，又具有防范和制裁公权力恣意行使、侵犯人权的作用；2．需要强化诉讼参与人的权利保障、被告人的权利保障、落实终审和诉讼终结制度， |                                                              |                                                              |

## 考点54．律师执业行为规范

| 律师业务推广规范                                             | 推广原则                                                     | 1．律师和律师事务所可以以广告方式宣传律师和律师事务所以及业务领域和专业特长；2．律师和律师事务所可以通过发表学术论文、案例分析、专题解答、授课、普及法律等活动，宣传自己的专业领域；3．律师和律师事务所可以通过举办或参加各种形式的专题、专业研讨会，宣传自己的专业特长；4．律师可以以自己或其任职的律师事务所名义参加各种社会公益活动。 |

## 考点55．法律援助

| 特征                 | 1．性质：法律援助是一项社会保障制度；提供援助是政府的职责；2．责任主体明确：法律援助的责任主体就是国家与政府；3．工作统一性：对公民的法律援助申请和法院指派的法律援助案件，由法律援助机构统一受理（接受）、统一审查、统一指派、统一监督；4．服务的无偿性：我国的法律援助是绝对“免费”。【注意】法院诉讼费用的缓减免，属于司法救助，不纳入法律援助制度体系。5．范围的广泛性：（1）从受援助对象来看，既包括经济困难符合法律援助条件者，也包括法律有特别规定的残疾者、弱者；（2）从司法程序来看，既有刑事诉讼中的辩护与代理，又有行政诉讼中的代理，还有民事诉讼中的代理；（3）凡是中国公民，具备法律援助条件的都可以申请法律援助；与中国签订法律援助司法协议的外国公民和无国籍人，符合条件的，也可申请法律援助。6．援助形式的丰富性：既包括诉讼法律援助服务，也包括非诉法律援助服务，还包括公证、法律咨询、法律信息资料的免费提供；对于刑事案件，法律援助机构应当指派律师为其提供辩护。【注意】刑事法律援助案件只能指派律师担任辩护人；法律咨询不需要审查经济条件。7．援助人员的多样性：既有律师、法律援助机构的工作人员，又有社会团体、事业单位等社会组织利用自身资源提供法律援助的人员。 |                                                              |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 对象                 | 刑事诉讼                                                     | 1．盲、聋、哑、未成年人（只能指派律师）、尚未完全丧失辨认或控制自己行为能力的精神病人为刑事被告人或犯罪嫌疑人，没有委托辩护律师的，应当获得法律援助；2．其他的残疾人、老年人为刑事被告人或犯罪嫌疑人，因经济困难没有能力聘请辩护律师的，可以获得法律援助； |

| 援助双方利害冲突     | 1援助律师与受援人之间有明显的利害冲突的，应当回避的，免除被指派律师的法律援助义务；2．明显的利害冲突包括：（1）指派的律师是同一案件其他当事人的代理人或法律顾问，且该当事人与受援人存在利害冲突的；（2）指派的律师是同一案件其他当事人的近亲属的；（3）指派的律师与受援人存在其他利害冲突，可能影响公正实施法律援助的；3．法律援助机构发现上述情况后，有权直接决定该律师回避，受指派的律师、受援人也可以向法律援助机构申请回避，由法律援助机构决定是否回避。 |                                                              |
| 终止                 | 1．受援人不再符合法律援助经济困难标准的；2．案件依法终止审理或被撤销的；3．受援人自行委托其他代理人或辩护人的；4．受援人要求终止法律援助的；5．受援人利用法律援助从事违法活动的；6．受援人故意隐瞒与案件有关的重要事实或提供虚假证据的。 |                                                              |
| 救济程序             | 1．检察院审查批准逮捕时，认为犯罪嫌疑人具有应当通知辩护的情形，公安机关未通知法律援助机构指派律师的，应当通知公安机关予以纠正，公安机关应当将纠正情况通知检察院；2．犯罪嫌疑人、被告人及其近亲属、法定代理人，强制医疗案件中的被申请人、被告人的法定代理人认为公检法应当告知其可以向法律援助机构申请法律援助而没有告知，或应当通知法律援助机构指派律师为其提供辩护或诉讼代理而没有通知的，有权向同级或上一级检察院申诉或控告；3．申请人对法律援助机构不予援助或终止援助的决定有异议的，可以向主管该法律援助机构的司法行政机关提出。符合条件的，主管机关应当书面责令法援机构及时援助；不符合条件的，应当维持不予援助的决定，并书面告知申请人。 |                                                              |

| 代理                   | 1．当事人申请办理公证，可以委托他人代理，但申办遗嘱、遗赠扶养协议、赠与、认领亲子、收养关系、解除收养关系、生存状况、委托、声明、保证及其他与自然人人身有密切关系的公证事项，应当由其本人亲自申办；2．无民事行为能力人或限制民事行为能力人申办公证，应当由其监护人代理；法人申办公证，应当由其法定代表人代表；其他组织申办公证，应当由其负责人代表；3．公证员、公证机构的其他工作人员不得代理当事人在本公证机构申办公证；4．居住在香港、澳门、台湾地区的当事人，委托他人代理申办涉及继承、财产权益处分、人身关系变更等重要公证事项的，其授权委托书应当经其居住地的公证人（机构）公证，或经司法部指定的机构、人员证明；居住在国外的当事人，委托他人代理申办前款规定的重要公证事项的，其授权委托书应当经其居住地的公证人（机构）、我驻外使（领）馆公证。 |                                                              |                                                              |

| 公证调解               | 经公证的事项在履行过程中发生争议的，出具公证书的公证机构可以应当事人的请求进行调解；经调解后当事人达成新的协议并申请公证的，公证机构可以办理公证；调解不成的，公证机构应当告知当事人就该争议依法向法院提起民事诉讼或向仲裁机构申请仲裁。 |                                                              |                                                              |

## 考点57．公证员职业道德和职业责任

| 职业道德         | 1．公证员不得同时在两个以上公证机构执业；2．公证员不得从事有报酬的其他职业；3．不得为本人及近亲属办理公证或办理与本人及近亲属有利害关系的公证；4．公证员、公证机构的其他工作人员不得代理当事人在本公证机构申办公证；5．不在名片上印制各种头衔、职称；6．不得以不正当的方式或途径对其他公证员正在办理的公证事项干预或施加影响；7．不得利用媒体或采用其它方式，对正在办理或己办结的公证事项发表不当评论；8．不得从事有报酬的其他职业和与公证员职务、身份不相符的活动；9公证事项应属于法定的可公证范围。 |                                                              |                                                              |

## 考点58．其他法律职业人员职业道德

## 考点63．民国宪法

| 三民主义与《组织大纲》   | 1．三民主义是民族主义、民权主义与民生主义的简称，是孙中山政治法律思想的核心内容；2.1912年1月2日公布《修正中华民国临时政府组织大纲》，成为中华民国第一部全国性的临时宪法性文件；3．受美国宪法影响，基本上采用总统制共和政体；4．中央国家机关权力分配实行资产阶级三权分立原则；5．采取一院制的议会政治体制，参议院是国家立法机关。 |                                                              |
