
# 民诉核心考点

## 考点01．基本原则

| 平等原则     | 当事人享有平等的诉讼权利，承担平等的诉讼义务。               |                                                              |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 同等原则     | 外国当事人在中国法院起诉、应诉，其权和义务与中国当事人相同。 |                                                              |
| 对等原则     | 外国法院对我国当事人的诉讼权利加以限制，我国法院对该国当事人权利实行对等原则。 |                                                              |
| 辩论原则     | 1．时间：整个诉讼过程中，不仅限于法庭辩论阶段，而且包括整个一审程序、二审程序以及再审程序中；2．内容：包括实体和程序两个方面的问题；3．形式：口头和书面均可；4．诉讼程序适用辩论原则，执行程序、特别程序、督促程序、公示催告程序等程序不适用辩论原则。 |                                                              |
| 处分原则     | 1．当事人有权在法律规定的范围内处分自己的民事权利与诉讼权利：处分的范围包括实体权利和诉讼权利，必须在法律规定的范围内实施；2．辩论原则VS处分原则：（1）裁判超事实、证据的，违反辩论原则；（2）裁判超诉求的，违反处分原则；（3）裁判既超事实、证据，又超诉求的，两者都违反。 |                                                              |
| 诚实信用原则 | 1．贯穿于整个民事诉讼过程中；2．不仅约束当事人，也约束法院和其他诉讼参与人；3．恶意诉讼：当事人之间恶意串通，企图通过诉讼、调解等方式侵害他人合法权益的，法院应当判决驳回其诉讼请求，并根据情节予以罚款、拘留；构成犯罪的，依法追究刑事责任。 |                                                              |
| 检察监督原则 | 对象                                                         | 民事裁判文书以及调解书（损害国家、社会公共利益）＋审判人员的违法行为＋执行活动 |
| 方式         | 抗诉（针对生效裁判、调解书，不适用再审的案件，也不能抗诉）＋检察建议（生效裁判、调解书＋审判人员的违法活动） |                                                              |
| 次数         | 当事人申请抗诉／检察建议，以1次为限，且须以申请再审为前置程序。 |                                                              |
| 在线诉讼原则 | 【新修】在线诉讼原则→在线诉讼与线下诉讼具有同等效力原则      |                                                              |
| 方式         | 经当事人同意，民诉活动可以通过信息网络平台在线进行。【提示】采取在线诉讼要经过当事人同意，但并不要求双方当事人都同意；一方同意线上一方不同意，对同意一方可以进行在线诉讼。 |                                                              |
| 效力         | 与线下诉讼活动具有同等法律效力。                             |                                                              |

## 考点02．合议制度

| 合议庭的组成                                                 | 一审                                                         | 审判员／审判员＋陪审员 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ---------------------- |
| 二审                                                         | 审判员                                                       |                        |
| 发回重审                                                     | 原审法院按照一审程序另行组成合议庭，陪审员可以参加。         |                        |
| 再审                                                         | 原来是一审的→审判员／审判员＋陪审员→按照一审程序另行组成合议庭（不能适用简易程序） |                        |
| 原来是二审或上级提审的→审判员→按照二审程序另行组成合议庭     |                                                              |                        |
| 【注意】二审发回重审或再审程序：原审合议庭成员或独任审判员不得参加重审或再审合议庭。（重要考点） |                                                              |                        |
| 特别程序                                                     | 选民资格案件和重大疑难案件由审判员组成合议庭。               |                        |

|                  | 注意                                                         | 1．人民陪审员参加的合议庭，只适用一审程序审理的诉讼程序。（1）一为审级，包括一审程序，也包括按照一审程序审理的发回重审和再审程序；（2）二是案件性质必须是诉讼案件，非诉程序案件中不能有陪审员参加；（3）不论是中院、高院、还是最高法，只要适用一审程序审理的案件，均可以由陪审员参加；（4）陪审员从法院所在城市的基层法院名单中随机抽取；（5）陪审员和审判员权利相同，只是不能当审判长。2．合议庭一般是3人组成，也有例外--必须7人组成的合议庭：（1）公益诉讼案件；（2）涉及征地征迁、生态环境保护、食品药品安全；（3）社会影响大的案件。3．合议庭的评议规则：（1）少数服从多数，不同意见如实记入笔录；（2）形成不了多数意见的，提交审委会；（3）对裁判持不同意见的，必须在裁判书上签字，不得拒签（这是与仲裁的区别）。 |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 独任制适用情形   | 简易程序                                                     | 只能由基层法院（及其派出庭）审理简单的一审的民事诉讼案件适用。 |
| 普通程序         | 一审                                                         | 基层法院审理的基本事实清楚、权利义务关系明确的一审民事案件。【注意】中院审理的一审案件不得适用独任制。 |
| 二审             | 中院对一审适用简易程序审结OR不服裁定提起上诉的二审案件（不予受理、驳回起诉、管辖权异议的裁定），事实清楚、权利义务关系明确，经双方当事人同意。 |                                                              |
| 特别程序         | 宣告公民失踪、死亡，指定遗产管理人、认定公民无、限制民事行为能力，认定财产无主，确认调解协议效力，实现担保物权 |                                                              |
| 督促程序         | 支付令                                                       |                                                              |
| 公示催告程序     | 其中公示催告阶段适用独任制；宣告票据无效应该由审判员组成合议庭 |                                                              |
| 独任制不适用情形 | 1．涉及国家利益、社会公共利益的案件；2．涉及群体性纠纷，可能影响社会稳定的案件；3．人民群众广泛关注或其他社会影响较大的案件；4．属于新类型或疑难复杂的案件；5．法律规定应当组成合议庭审理的案件；6．其他不宜由审判员一人独任审理的案件。 |                                                              |

## 考点03．回避制度

| 适用对象 | 参与本案审理的院长、副院长、审判委员会委员、庭长、副庭长、审判员、助理审判员、陪审员、书记员、翻译人员、鉴定人、勘验人、检察人员、执行人员、法官助理【新增】、司法技术人员【新增】。【注意】证人、诉讼代理人不需要回避。 |                                                              |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 适用情形 | 身份不当                                                     | 1．是本案当事人或当事人的近亲属；2．本人或其近亲属与本案有利害关系；3．担任过本案的证人、鉴定人辩护人、诉讼代理人、翻译人员；4．是本案诉讼代理人的近亲属； |

|                |                                                              | 5．本人或其近亲属持有本案非上市公司当事人的股份或股权；6．与本案当事人或诉讼代理人有利害关系，可能影响公正审理的。 |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 法官行为不检点 | 接受请客送礼的，为本案当事人推荐代理人、律师，违反规定会见当事人、诉讼代理人等不正当行为的。 |                                                              |
| 跨诉讼阶段     | 原则                                                         | 在一个诉讼过程中参与过本案审判工作的审判人员，不得再参加本案其他诉讼程序的审判。 |
| 例外           | 发回重审的案件，在一审法院重审后作出裁判后又进入二审程序的，原二审程序中合议庭组成人员不受前款回避规定的限制。 |                                                              |
| 申请程序       | 方式                                                         | 口头、书面                                                   |
| 时间           | 案件开始审理时提出，但如果在开始审理后才知晓回避理由的，可以在法庭辩论终结前提出。 |                                                              |
| 申请效力       | 1．申请后，回避决定作出前，被申请人回避人应暂停本案工作，但需要采取紧急措施的除外；2．法院作出回避决定后，不影响已经进行的诉讼程序的效力。 |                                                              |
| 回避决定       | 1．审判人员→院长决定；（包括陪审员）2．其他人员→审判长决定；3．院长担任审判长时→由审委会决定；4．检察人员→检察长的回避由检委会决定；检察人员和其他人员的回避由检察长决定。 |                                                              |
| 救济           | 1．申请人的申请被驳回后，可以本院复议1次。2．复议期间，被申请回避人员不停止工作；被决定回避的人员无权申请复议。 |                                                              |

## 考点04．公开审判制度

| 范围                 | 开庭审理的过程＋宣判的过程＋裁判文书           |                                                     |
| -------------------- | ---------------------------------------------- | --------------------------------------------------- |
| 对象                 | 向社会、媒体、公众公开，允许旁听和报道         |                                                     |
| 审理公开             | 原则                                           | 审判过程和结果应当公开。                            |
| 例外                 | 法定不公开                                     | 1．涉及国家秘密；2．涉及个人隐私；3．法律另有规定。 |
| 依申请不公开         | 1．离婚案件；2．涉及商业秘密的案件。           |                                                     |
| 合议庭               | 评议一律不公开＋宣判一律公开                   |                                                     |
| 裁判文书的公众查阅权 | 查阅对象                                       | 生效的判决书、裁定书。--包括民事公益诉讼的调解书    |

| 权限                 | 仅限于查阅。                                   |                                                     |
| 保护                 | 对国家、商业秘密和个人隐私不予准许。           |                                                     |

## 考点06．反诉

| 主体     | 同一性：本诉的被告对本诉的原告提出。--并不意味在本、反诉中的诉讼地位相同 |
| -------- | ------------------------------------------------------------ |
| 牵连关系 | 1．反诉本诉是基于同一法律关系而产生的目的相对抗的不同实体请求。2．基于同一事实产生。3．本诉与反诉请求是基于相互牵连的两个不同的且各自独立的法律关系而产生的目的相对抗的不同实体请求。【注意】牵连关系并不当然得出本、反诉必须基于同一法律关系。 |
| 时间     | 在本诉诉讼过程中提出。【注意】《民诉司法解释》关于各审理阶段提出反诉的处理情形：①一审：应在辩论终结前提出，可以合并审理。②二审：可以调解，调解不成，告知另行起诉，双方当事人同意由二审法院一并审理的，可以合并审理。③再审：（7）原审未合法传唤缺席判决，影响当事人行使诉讼权利；（2）追加新的诉讼当事人；（3）诉讼标的物灭失或发生变化致使原诉讼请求无法实现；（4）当事人提出反诉无法通过另诉解决。符合以上四种情形，当事人提出反诉，法院应当准许。 |
| 管辖     | 向受理本诉的法院提出，且受诉法院对反诉有管辖权（牵连管辖）。 |
| 程序     | 反诉应与本诉适用同一诉讼程序，能合并审理的话就合并审理。     |

## 考点07．级别管辖

| 基层法院 | 一审民事案件                                                 |
| -------- | ------------------------------------------------------------ |
| 中级法院 | 1．重大且涉外案件；2．本辖区有重大影响的案件；3．最高法确定由中级法院管辖的条件：（1）海事、海商案件；（2）公益诉讼案件；（3）专利纠纷案件；（4）仲裁相关：除了国内仲裁的财产保全、国内仲裁的证据保全找基层法院外，其他的仲裁事项皆是找中院。4．重大涉港澳台；5．著作权纠纷：原则上由中院管辖，但各高院可以确定若干基层法院管辖一审著作权纠纷案件；6．商标民事纠纷：原则上由中院管辖，但各高院可以根据辖区实际，经最高法批准，可以在较大的市确定1-2个基层法院管辖商标纠纷一审案件。 |
| 高院     | 本辖区有重大影响的案件                                       |
| 最高法院 | 1．全国范围内有重大影响的案件；2．最高法认为应当由本院管辖的案件。 |

## 考点08．地域管辖

| 一般地域管辖     | 原则：原告就被告                                             | 1．一般顺位：被告经常居住地法院→被告住所地法院→被告原户籍所在地法院；2．特殊顺位：民诉解释规定的三种情形。情形①：优先选择监禁地法院（监禁满1年）→被告住所地法院；情形②：只能由被告居住地法院管辖；情形③：优先选择被告经常居住地法院管辖→被告居住地法院辖（起诉时）。3．民诉解释规定的三种原告就被告的情形：（1）双方当事人都被监禁或都被采取强制措施的→被告原住所地法院。若被告被监禁或被采取强制教育措施1年以上的→被告被监禁地或被采取强制教育措施地法院；（2）双方当事人均被注销户籍的→被告居住地法院；（3）夫妻双方离开住所地超1年，一方起诉离婚的案件→由被告经常居住地法院；没有经常居住地的，→原告起诉时被告居住地法院管辖。 |                                                          |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | -------------------------------------------------------- |
| 例外：被告就原告 | 1．一般顺位：原告经常居住地法院→原告住所地法院→原告原户籍所在地法院；2．特殊顺位：下述八类情形之（6）（7）（8）除了遵循上述一般顺序外，还可以由被告住所地法院管辖。（1）对不在中国领域内居住的人提起有关身份关系的诉讼；（2）对下落不明或宣告失踪的人提起的有关身份关系的诉讼；（3）对正在被采取强制教育措施的人提起诉讼；（4）对正在被监禁的人提起诉讼的；（5）被告被注销户籍的；（6）追索三费（赡养费、抚养费、扶养费）的几个被告住所地不在同一辖区的→原告住所地法院；（7）夫妻一方离开住所地超过1年，另外一方起诉离婚的→原告住所地法院。（8）不服指定监护或变更监护关系的案件→被监护人住所地法院。 |                                                              |                                                          |
| 特殊地域管辖     | 侵权纠纷                                                     | 一般规定                                                     | 由侵权行为地（行为实施地＋结果发生地）或被告人住所地法院 |
| 特殊规定         | 1．产品、服务侵权：产品制造地、产品销售地、服务提供地、侵权行为地和被告住所地法院；2．信息网络侵权：信息网络侵权行为地包括实施侵权行为的计算机等信息设备所在地、侵权结果发生地包括被侵权人住所地；3．侵犯名誉权：侵权行为实施地、侵权结果发生地、被告住所地法院均有管辖权；4．因诉前保全引发的损害赔偿：（1）未在法定期间内起诉或申请仲裁的→采取保全措施的法院。（2）在法定期间内起诉或申请仲裁的，但被保全人因此遭受损失的→受理法院或采取保全措施的法院。 |                                                              |                                                          |
| 合同纠纷         | 因合同纠纷提起的诉讼由被告住所地（恒有管辖权）或合同履行地法院管辖。 |                                                              |                                                          |
| 约定了履行地的   | 1．合同已实际履行的，约定履行地与实际履行地不一致的→约定履行地法院；2．合同没有实际履行的，约定履行地和一方当事人的住所地一致的→约定履行地法院； |                                                              |                                                          |

|                                              |                                                              |                                                             | 3．合同没有实际履行的，约定履行地和双方当事人的住所地不一致的→约定履行地法院无管辖权。 |
| -------------------------------------------- | ------------------------------------------------------------ | ----------------------------------------------------------- | ------------------------------------------------------------ |
| 没有约定履行地或约定不明确的：履行地确定规则 | 1．给付货币的→接受货币一方所在地；2．交付不动产的，不动产所在地；3．交付其他标的，履行义务一方所在地；4．财产租赁合同、融资租赁合同以租赁物使用地为合同履行地→该地有管辖权；5．以信息网络方式订立的买卖合同，通过信息网络交付标的→以买受人住所地为合同履行地；通过其他方式交付标的的，收货地为履行地。 |                                                             |                                                              |
| 其他特殊规定                                 | 公司诉讼                                                     | 公司设立、确认股东资格、分配利润、解散等诉讼→公司住所地法院 |                                                              |
| 保险合同                                     | 一般规定                                                     | 被告住所地、保险标的所在地                                  |                                                              |

## 考点11．裁定管辖

| 移送管辖   | 1．法院受理案件后，当事人在答辩期间届满后未应诉答辩的，法院在一审开庭前发现管辖有误的，应当裁定移送。【注意】①起诉后，受理前发现无管辖权的，应告知原告向有管辖权的法院起诉；②受理后发现主管错误的，应当驳回起诉。2．移送管辖的要件：（1）法院已经受理；（2）发现自己无管辖权（客观无）；（3）认为受移送法院有管辖权；（4）不存在应诉管辖问题。3.1次为限：受移送法院认为自己也无管辖权的，不得再行移送或退回，只能报请自己的上级法院指定管辖。【注意】2个以上法院均有管辖权的案子，先立案的法院不得将案件移送其他有管辖权的法院。 |                                                              |

## 考点16．共同诉讼

| 普通共同诉讼             | 特征                                                         | 1．一方或双方为2人以上；2．诉讼标的是同种类的；3．是数个可分之诉；4．为了实现诉讼经济而合并；5．合并审理，分别判决 |

| 构成要件                 | 1．数个同种类诉讼标的；2．同一法院管辖；3．适用同一程序；4．法院认为可合并；5．当事人同意 |                                                              |
| 内部关系                 | 各诉讼人之间行为独立，对其他诉讼人没有效力。                 |                                                              |
| 必要共同诉讼             | 特征                                                         | 1．一方／双方为2人以上；2．诉讼标的同一；3．法院必须合并审理，合一判决。 |

|                          |                                                              | 8．遗产继承诉讼中：所有继承人；9．无、限人造成他人损害的：无限人＋监护人；10．劳务派遣期间：被派遣的工作人员在工作中致使他人损害的，用工单位为当事人，除非当事人主张派遣单位和用工单位承担责任的；11．原告起诉代理人和被代理人，要求承担连带责任的：以代理人和被代理人为共同被告；12．原告起诉代理人和相对人，要求承担连带责任的：以代理人和相对人为共同被告；13．共有财产受到他人侵害：部分共有人起诉的，其他共有人为共同诉讼人；14．连带保证合同；15．共同侵权、共同危险等。 |
| 必要共同诉讼人的追加     | 1．可以依申请／依职权；2．应追加者已经明确表示放弃的实体权利的，可以不予追加；3．不愿意参加＋不放弃实体权利的→追加为共同原告，其不参加诉讼，不影响审理和判决；4．应追加的被告不参加诉讼→一般可缺席判决，符合拘传条件的可拘传到庭。 |                                                              |

| 参诉方式               | 只能以起诉的方式参加（法院不得主动追加）                     |
| 参诉时间               | 一审法庭辩论终结前。【注意】第三人一审未参与的，经申请法院可以准许参加二审；二审中有独三参与的→可调解，调不成的，撤销原判，发回重审。 |
| 与本诉的关系           | 1．实质是将两个独立的诉合并审理；2．有独三撤诉不影响本诉的进行；3．本诉的原告撤诉，有独三作为另案原告，被告作为另案被告，诉讼继续进行。 |
| 有独三VS原告的区别技巧 | 第一步：找出案件的原被告，再找出原被告之间的法律关系；第二步：看待判断的主体与被告之间是否存在与本案诉讼标的相同／相类似的法律关系：1．若该人与被告之间存在相同的法律关系→则该人和原告属于必要的共同诉讼的共同原告；2．若该人与被告之间存在相类似的法律关系→则属于普通共同诉讼的共同原告；3．若该人与被告之间不存在与本案诉讼标的相同或相类似的法律关系→则为有独三。 |

## 考点18．无独三

| 参诉理由       | 与案件处理结果有法律上的利害关系。                           |
| -------------- | ------------------------------------------------------------ |
| 参诉方式       | 无独三申请参加或法院依职权追加。                             |
| 诉讼与义务     | 1．附条件享有的权利：（1）判决其承担民事责任→有权提起上诉；（2）调解确定无独三承担义务的→需经其同意＋应送达调解书＋签收（签收前反悔的，调解书不生效，须及时判决）；2．无权享有的权利：管辖权异议，放弃、变更诉求（但可以承认诉求）或撤诉。 |
| 可追加的情形   | 1．代位权诉讼：（三角债）代位权诉讼中的债务人；2．债务承担、债权让与、债的概括承受；3．撤销权诉讼；4．保证合同发生纠纷引起债权人反诉的； |
|                | 5．因缺陷产品引起的侵权诉讼；6．劳动争议中用人单位招用尚未解除劳动合同的劳动者，原用人单位起诉劳动者，可列新用人单位为无独三；原用人单位起诉新用人单位的，可列劳动者为无独三。 |
| 不得追加的情形 | 1．与原被告双方争议的诉讼标无直接牵连和不负返还或赔偿义务的人；2．与原告或被告约定仲裁或有约定管辖的案外人，或专属管辖案件的一方当事人；3．产品质量纠纷中原被告法律关系以外的下列人员：（1）证据证明其已经提供了合同约定或符合法律规定的产品的人；（2）案件中的当事人未在规定的质量异议期内向其提出异议的人；（3）案件中的收货方已经认可其提供产品质量的人。（4）已经履行义务，或依法取得了一方当事人的财产，并支付了相应对价的原被告之间法律关系以外的人。 |

## 考点19．第三人撤销之诉

| 起诉条件       | 程序                                                         | 有作为本案有独三或无独三的资格＋未参加诉讼                   |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 实体           | 有证据证明生效裁判调有错误                                   |                                                              |
| 结果           | 损害其民事权益                                               |                                                              |
| 时间           | 知道或应当知道其民事权益受到损害之日起6个月内。              |                                                              |
| 不予受理的情形 | 1．适用特别程序、督促程序、公示催告程序、破产程序等非讼程序处理的案件；2．婚姻无效、撤销或解除婚姻关系等判决、裁定、调解书中涉及身份关系的内容；3．未参加登记的权利人对代表人诉讼案件的生效裁判；4．损害社会公共利益行为的受害人对公益诉讼案件的生效裁判（可以另诉）。 |                                                              |
| 管辖           | 作出该判决、裁定、调解书的法院                               |                                                              |
| 审理程序       | 审查起诉与受理                                               | 1．法院应在收到起诉状和证据材料之日起5日内送交对方当事人（原审双方当事人），对方当事人可自收到起诉状之日起10日提出书面意见；2．书面审查，必要时可以询问当事人；符合立案条件的30日内立案，不符合要求的，30日内裁定不予受理；对于受理后发现不符合起诉条件的，应裁定驳回起诉。 |
| 审判组织和方式 | 应当组成合议庭开庭审理，适用一审普通程序。                   |                                                              |
| 诉讼地位       | 1．提起撤销之诉第三人→原告；2．生效判决、裁定、调解书的当事人→被告；3．没有承担责任的无独三→第三人； |                                                              |
| 法律效果       | 原则                                                         | 不中止执行                                                   |
| 例外           | 1．原告提供相应担保，请求中止执行；2．向执行法院提出案外人对执行标的的异议，异议成立。 |                                                              |
| 处理           | 1．诉讼请求成立的→应当改变／撤销原裁判调；3．诉讼请求不成立的→驳回诉讼请求。 |                                                              |
| 审理与救济     | 应使用一审普通程序审理，所作判决为一审判决，当事人不服可以上诉。 |                                                              |

## 考点20．法定代理人

| 代理权限       | 享有当事人的所有诉权，也应当履行当事人的诉讼义务。           |
| -------------- | ------------------------------------------------------------ |
| 诉讼地位       | 法代不是当事人，当事人仍然是无民事诉讼行为能力人本人。       |
| 如何确定       | 1．无诉讼行为能力人的监护人是其法定代理人；                  |
|                | 2．多位监护人之间相互推诿，由法院指定其中一人；3．事先没有确定监护人的，先协商，协商不成的由法院指定。 |
| 与当事人的区别 | 1．诉讼地位不同：只能以无民事诉讼行为能力人的名义起诉、应诉；2．裁判针对的是当事人，而非法代；3．诉讼中发生事件的后果不同：（1）当事人死亡的→可能导致当事人变更／诉讼终结；（2）代理人死亡的→另行指定法定代理人继续诉讼，不会终结。 |

## 考点22．证据的理论分类

| 本证VS反证 | 标准                                                         | 按照证明责任来划分                                         |
| ---------- | ------------------------------------------------------------ | ---------------------------------------------------------- |
| 分类       | 本证                                                         | 负有举证责任的一方当事人提出的用于证明自己主张事实的证据。 |
| 反证       | 不承担举证责任的一方当事人提出的用以反对对方当事人的主张的证据。 |                                                            |
| 判断技巧   | 第一步：确定该证据的待证事实；第二步：判断该待证事实应该由哪一方承担举证责任；第三步：该证据是由谁提出来的；1．举证责任人提出来的→本证；2．非举证责任人提出来的→反证 |                                                            |
| 直接VS间接 | 标准                                                         | 根据证据与待证事实的关系                                   |
| 分类       | 直接证据                                                     | 能够单独、直接证明案件待证事实的证据。                     |
| 间接证据   | 不能单独、直接证明案件待证事实，而需要结合其他证据才能证明案件事实的证据。 |                                                            |
| 判断技巧   | 1．该分类只与证据内容的完整性有关系，与证据的来源和证明力无关；2．主要通过证据内容的完整性来区分：（1）直接证据在内容上可以完整证明案件待证事实（2）间接证据在内容上只能证明待证事实的一部分、一片断，需要别的证据予以补充。 |                                                            |
| 原始VS传来 | 标准                                                         | 根据是否直接来源于案件事实                                 |
| 分类       | 原始证据                                                     | 直接来源于案件事实的证据。                                 |
| 传来证据   | 又称为派生证据，指的是不直接源自案件事实，而是通过传抄、转述、复制后所获得的证据。 |                                                            |

## 考点23．证据的法定分类

| 书证              | 以其记载的内容／表达的思想来证明案件事实的证据。             |                                                              |
| ----------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 物证              | 以其自身的特征、外形、质量、痕迹等物理属性证明案件事实的证据。 |                                                              |
| 视听资料          | 利用录音磁带或录像所反映的图像、声音来证明案件事实的证据。   |                                                              |
| 电子数据          | 指的是通过电邮、数据交换、网聊、博客、微博客、手机短信、电子签名、域名等方式形成／存储在电子介质中的信息来证明待证事实。【注意】①存储在电子介质中的录音、录像资料，适用电子证据的规定。如数码相机、u盘、手机信息等。②电子数据的制作者制作的与原件一致的副本，或直接来源于电子数据打印件或其他可以显示、识别的输出介质，视为电子数据原件。 |                                                              |
| 证人证言          | 证人资格                                                     | 1．知道案件情况的单位和个人均有义务出庭作证；2．不能正确表达意思的人不能做证人；3．证人本身不是证据，其提供的内容才是证据；4．证人不适用回避制度。 |
| 通知出庭          | 1．举证期限届满前当事人申请／法院依职权通知；2．未经通知不得出庭。（当事人同意＋法院准许的除外） |                                                              |
| 证人应当 出庭     | 1．法院应当要求证人出庭作证，接受询问；2．证人在审理案件前的准备阶段或法院调查、询问等双方当事人在场时陈述证言的，视为出庭作证；3．双方当事人同意证人以其他方式作证，并经法院准许的，证人可以不出庭作证；4．无正当理由未出庭作证的证人以书面形式提供的证言，不得作为定案依据。 |                                                              |
| 可以不出 庭的情形 | 1．健康原因不能出庭；2．路途遥远交通不便，不能出庭的；3．不可抗力不能出庭的；4．其他正当理由。 |                                                              |
| 保证书            | 1．签署与宣读：法院应当要求证人签署保证书，并当庭宣读相关内容，但无、限人除外；证人确有正当理由无法宣读的，可由书记员代为宣读；2．证人拒绝签署的：不得作证＋费用自担。 |                                                              |
| 证人陈述          | 1．证人应当客观陈述亲身感受的事实，不得使用猜测、推断、评论性语言；2．证人作证前不得旁听庭审，作证时不得以宣读事先准备的书面材料的方式陈述证言；3．证人应就其作证的事项连续陈述；询问证人时，其他证人不得在场；法院认为有必要的，可以要求证人之间对质。 |                                                              |
| 当事人陈述        | 禁反言                                                       | 当事人的陈述与此前陈述不一致的，法院应当责令其说明理由，并结合当事人的诉讼能力、证据和案件具体情况进行审查认定。 |
| 保证书            | 法院认为有必要→可要求当事人本人到庭＋接受询间→问之前应当要求其签署并宣读保证书→有正当理由不能宣读的，书记员代为宣读→当事人故意虚假陈述妨碍审理的，依法追究妨害诉讼的责任。 |                                                              |
| 拒绝的            | 当事人无正当理由拒绝到庭／接受询问／签署／宣读保证书的＋待证事实欠缺其他证据证明的→法院应当作出不利于该当事人的推定。 |                                                              |
| 鉴定意见          | 启动                                                         | 当事人申请／法院依职权                                       |
| 鉴定              | 1．签署承诺书；                                              |                                                              |

|                                                              | 准备                                                         | 2．对鉴定材料的质证；3．经法院准许，鉴定人可以调取证据、勘验物证和现场询问当事人或证人。 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 权利与义务                                                   | 权利                                                         | 了解进行鉴定所需要的案件材料；必要时可以询问当事人、证人；   |
| 义务                                                         | 1．提出书面鉴定意见；2．在鉴定书上签名或盖章；3．出庭作证义务，并接受法官、当事人询问。 |                                                              |
| 出庭                                                         | 情形                                                         | 当事人有异议／法院认为应当出庭的→必须出庭；                  |
| 费用                                                         | 败诉方承担。因鉴定意见不明确或有瑕疵需要签定人出庭的，出庭费用由其自行负担。 |                                                              |
| 拒不出庭后果                                                 | 1．鉴定意见不得作为认定案件事实的依据；2．支付鉴定费用的当事人要求退还鉴定费用的，法院应当裁定责令退还；3．建议有关主管部门或组织对相关鉴定人进行处罚。 |                                                              |
| 专门知识的人出庭                                             | 1．依当事人申请：法院通知1-2名出庭；2．作用：对鉴定意见提出意见（代表当事人质证）；对专业问题提出意见（视为当事人陈述）；3．不得参与专业问题之外的审理活动；4．不适用回避；5．费用：申请人承担。 |                                                              |
| 勘验笔录                                                     | 启动                                                         | 认为有必要，可以根据申请／依职权启动                         |
| 【注意】可以要求签定人参与勘验，必要时可以要求鉴定人在勘验中进行鉴定。 |                                                              |                                                              |

## 考点24．自认

| 对象             | 仅限于事实                                                   |                                                      |
| ---------------- | ------------------------------------------------------------ | ---------------------------------------------------- |
| 效果             | 1．该事实免除对方举证责任；2．以自认的事实作为裁判依据。     |                                                      |
| 范围（禁止）     | 1．对于应当由法院依职权调查的事实，不适用自认制度；2．诉讼中当事人为了达成调解协议或和解目的而做出的妥协，所涉及的对事实的认可，不得在其后的诉讼活动中作为对其不利的证据；（但法律另有规定／双方均予认可的除外）3．自认的事实与法院查明的事实不相符的，法院不予确认。 |                                                      |
| 方式             | 明示                                                         | 明确表示承认的                                       |
| 默示             | 不承认也不否认＋审判人员充分说明并询问→仍不明确表示肯定／否定的 |                                                      |
| 委托的承认       | 1．除授权委托书明确排除的事项以外的，视为当事人的承认；2．当事人在场对诉讼代理人的自认明确否认的，不视为自认。 |                                                      |
| 撤销             | 法庭辩论终结前提出＋对方同意／有充分证据证明其受到胁迫或重大误解情况下做出→法院应当以口头或书面裁定的方式准许。 |                                                      |
| 共同诉讼中的自认 | 普通共同诉讼                                                 | 一人或数人作出的自认，只对作出自认的当事人发生效力。 |
| 必要共同诉讼     | 1．一人或数人作出自认而其他共同诉讼人予以否认的，不发生自认的效力。2．其他共同诉讼人既不否认也不承认的，经审判人员说明并询问后仍然不明确表明意见的，视为全体共同诉讼人的自认。 |                                                      |
| 附条件自认的效力 | 一方当事人对于另一方当事人主张的于己不利的事实有所限制或附加条件予以承认的，由法院综合案件情况决定是否构成自认。 |                                                      |

## 考点25．证据标准与证明责任

| 证明标准       | 程序性事实                                                   | 存在较大可能性，即可认定该事实存在。               |                                                        |
| -------------- | ------------------------------------------------------------ | -------------------------------------------------- | ------------------------------------------------------ |
| 实体性事实     | 一般标准                                                     | 高度盖然性（存在高度可能性即可）→一般事实          |                                                        |
| 特殊标准       | 排除合理怀疑→1．欺诈、胁迫、恶意串通；2．口头遗嘱；3．赠与   |                                                    |                                                        |
| 证明责任       | 前提                                                         | 待证事实处于真伪不明                               |                                                        |
| 证明主体       | 当事人，法院不承担举证责任                                   |                                                    |                                                        |
| 证明责任的分配 | 原则：谁主张谁举证                                           | 依据                                               | 谁主张积极事实，谁承担证明责任。                       |
| 合同纠纷       | 1．主张合同成立并生效一方对成立和生效承担举证责任；2．主张合同关系变更、解除、终止、撤销的一方对此事实承担举证责任；3．对合同是否履行发生争议，由主张已经履行的一方承担举证责任；4．对是否存在代理权发生争议的，由主张存在代理权的一方承担举证责任。 |                                                    |                                                        |
| 劳动纠纷       | 因用人单位做出开除、除名、辞退、解除劳动合同、减报酬、计算年限等决定发生争议的，由用人单位承担举证责任。 |                                                    |                                                        |
| 侵权纠纷       | 1．原告就侵权责任构成要件负举证责任：（1）侵权行为；（2）损害结果；（3）因果关系；（4）过错（无过错责任原则的不存在过错要件，无需证明）2．被告就法定免责事由承担举证责任。 |                                                    |                                                        |
| 例外：倒置     | 因果关系倒置                                                 | 环境污染、破坏生态                                 | 由加害方就其行为与损害之间不存在因果关系承担举证责任。 |
| 共同危险       | 由危险实施者就其行为与损害结果之间不存在因果关系承担举证责任。 |                                                    |                                                        |
| 行为倒置       | 专利纠纷案件                                                 | 由侵权产品的生产者对其制造方法有别于专利方法举证。 |                                                        |
| 过错倒置       | 适用过错推定原则：由被告对其无过错承担举证责任。1．无人在幼儿园、学校或其他教育机构学习、生活期间受到人身伤害，幼儿园、学校或其他教育机构应当承担赔偿责任，但是能够证明尽到职责的，不承担责任；2．医疗纠纷中，患者在诊疗活动中受到损害，有下列情形的，推定医疗机构有过错：（1）违反法律、行政法规、规章以及其他有关诊疗规范的规定；（2）隐匿或拒绝提供与纠纷有关的病例材料的；（3）遗失、伪造、篡改或违法销毁病例材料的。3．动物园的动物造成他人损害的，动物园承担侵权责任，但是能够证明尽到管理职责的不承担；4．建筑物、构筑物或其他设施及其搁置物、悬挂物发生脱落、坠落造成他人损害的，所有人、管理人或使用人不能证明自己没有过错的，应当承担侵权责任。 |                                                    |                                                        |

## 考点34．调解

| 适用范围                         | 积极范围                                                     | 1．可能通过调解解决的，应当调解；2．离婚案件，应该先行调解；3．简易程序的案件，应先行调解。 |
| -------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 消极范围                         | 1．适用特别程序、督促程序、公示催告程序审理的案件；2．婚姻、身份关系确认案件以及依案件性质不能调解的民事案件；【注意】仅仅是涉及婚姻关系确认的案件不适用调解，并非所有婚姻案件。3．执行案件：执行程序不适用调解，但可以和解。 |                                                              |
| 适用阶段                         | 一审、二审、再审程序均能调解，执行程序不能调解（但可和解）。 |                                                              |
| 不公开调解                       | 1．调解过程不公开（除非当事人同意）；2．调解协议内容不公开（除非为保护国家、社会、他人利益＋法院认为确有必要的才公开）；3．主持调解以及参与调解的人员，对调解过程以及该过程中获悉的国家秘密、商业秘密、个人隐私和其他不宜公开的信息，应当保守秘密。 |                                                              |

| 审理法院       | 法院启动                                                     | 1．本院启动的：本院审理；2．上级法院启动的：该上级法院提审。 |

|          |                                                              | 3．上级法院按照审监程序提审的，按照二审程序审理，所作裁判为终审裁判。 |

## 考点55．督促程序

| 要件             | 1．给付标的为金钱和有价证券；2．请求给付的标的已到期＋数额确定，并写明了请求所根据的事实、证据；3．债权人和债务人之间无其他债务纠纷；4．支付令能送达债务人；（留置送达，不可公告送达）5．债务人在我国境内且未下落不明。 |
| ---------------- | ------------------------------------------------------------ |
| 管辖             | 债务人住所地基层法院。                                       |
| 支付令效力       | 1．作出之日发生效力；2．债务人应当在收到之日起15日内清偿债务／向法院提书面异议；3.15日内不异议＋不履行：债权人可以向法院申请执行。 |
| 债务人异议       | 异议的效力：1．债务人书面异议＋法院形式审查＋异议成立的→裁定终结督促程序，支付令失效；2．支付令失效后→转入诉讼程序（申请人不同意的除外）3．申请人不同意起诉的，应于收到终结督促程序裁定之日起7日内提出，否则视为同意转；4．督促程序转为诉讼程序的，起诉时间以申请支付令的时间起算，而不是转为诉讼之日起算。 |
| 效力             | 以下三情形，不构成有效异议，不影响支付令效力：1．债务人对债务本身无异议，只提出缺乏清偿能力、延缓清偿等异议的，不构成异议，不影响支付令效力；2．口头异议无效；3．债务人收到支付令后，不提书面异议，只是向其他法院起诉的，不影响支付令效力。 |
| 支付令错误的救济 | 对支付令错误的救济只能由法院依职权撤销，由院长提交审委会计论决定。 |

|                | （2）管辖：申请法院确认的，由他裁协议约定的仲裁机构所在地、协议签订地、申请人住所地、被申请人住所地的中院管辖。（3）时间：对效力有异议的，应在仲裁庭首次开庭前提出。 |
| 仲裁协议的失效 | 法定情形：1．该纠纷已经过实体裁决；（即使之后被法院裁定撤销或不予执行）2．当事人协议放弃了原先签订仲裁协议；（明示或默示）3．对于约定了仲裁协议有限期限的情形，超过该期限则仲裁协议无效。 |

| 依申请撤销仲裁裁决 | 申请主体                                                     | 仲裁申请人、被申请人  |
| 管辖权             | 仲裁委所在地的中院                                           |                       |
| 法定情形           | 1．无仲裁协议的；2．裁决的事项不属于仲裁协议的范围或仲裁庭无权管辖的；3．仲裁庭的组成或仲裁程序违反法定程序的；4．裁决所依据的证据系伪造的：；5．对方当事人隐瞒了足以影响公正裁决的证据的；6．仲裁员在仲裁该案时有索贿受贿、徇私舞弊、枉法裁判等违法行为的。 |                       |
|                    | 审查期限                                                     | 受理申请之日起2个月。 |
| 审理方式           | 须组成合议庭，并询问当事人。                                 |                       |
| 审查结果           | 1．无撤销情形的：裁定驳回撤销申请，该裁定不可上诉→驳回后，原裁决具有执行力；2．有撤销情形的：裁定撤销仲裁裁决：（1）该裁定不能上诉和申请再审；（2）如果属于超出裁决范围的情形＋超裁部分与其他部分裁决可分：可撤销超裁部分；3．有上述法定情形之4和5的：法院可通知仲裁庭重新仲裁＋同时应裁定中止撤销程序＋应当说明重新仲裁的理由→（仲裁庭可以接受／拒绝）接受的，在指定期限内重新仲裁的：法院裁定终结撤销程序；拒绝的，法院恢复撤销程序。 |                       |
| 撤销裁决的后果     | 当事人可以向法院起诉，也可以重新达成仲裁协议。               |                       |

## 考点65．司法对仲裁的监督（不予执行仲裁裁决）



| 被申请人可以申请不予执行仲裁裁决       | 情形                                                         | 与撤销仲裁裁决情形一致。                             |
| -------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------- |
| 管辖                                   | 受理执行申请的法院。                                         |                                                      |
| 申请                                   | 1．当事人申请不予执行仲裁调解书的或根据仲裁和解协议制作的裁决书／申请撤销裁决→驳回申请后，又申请不予执行的→法院不予支持；2．当事人在仲裁程序中未对仲裁协议的效力提出异议，在仲裁裁决作出后以仲裁协议无效为由主张撤销仲裁裁决或不予执行抗辩的，法院不予执行。 |                                                      |
| 救济                                   | 仲裁裁决被裁定不予执行后，可以起诉或重新达成仲裁协议后去申请仲裁。 |                                                      |
| 案外人可申请不予执行仲裁裁决书、调解书 | 情形                                                         | 当事人恶意申请仲裁、虚假仲裁侵害案外人的合法权益的。 |
| 时间                                   | 自知／应知法院对该标的采取执行措施之日起30日内。             |                                                      |
| 救济                                   | 1．基于案外人申请而裁定不予执行的，当事人可以自裁定送达之日起10日内上一级法院申请复议1次；2．法院裁定驳回案外人不予执行申请，案外人可以自裁定送达之日起10日内上一级法院申请复议1次。 |                                                      |
