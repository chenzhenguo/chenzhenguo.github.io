## 刑法核心考点

考点1．解释技巧（制造结论）

| 平义解释 | 按照该用语最平白的字面含义来解释。                           |
| -------- | ------------------------------------------------------------ |
| 扩大解释 | 指对用语解释后的含义大于字面含义，但该含义仍处在该用语可能的含义范围内。 |
| 类推解释 | 将不符合法律规定的情形解释为符合法律规定的情形。【注意】无论是采取何种方法对刑法法条进行解释，都不得采取类推的模式，除非在法律文字违背立法目的的情况下，可以做出有利于被告人的类推解释，但不意味着可以用类推解释使犯罪嫌疑人摆脱罪名。 |
| 缩小解释 | 指对用语解释后的含义小于字面含义。                           |
| 反对解释 | 指根据用语的正面表述，推导出其反面含义。                     |
| 补正解释 | 指刑法条文用语表述有明显错误，只有通过修正、补正来阐明其真实含义。 |

## 考点2．解释理由（论证结论）

| 文理解释 | 根据文法、语法等来论证解释后的含义是否属于刑法用语可能具有的含义。换句话说，需要考查该解释后的含义在文理上是否讲得通。 |                                        |
| -------- | ------------------------------------------------------------ | -------------------------------------- |
| 论理解释 | 目的解释                                                     | 根据刑法规范的目的阐明刑法规范的含义。 |
| 历史解释 | 指根据历史的、发展的眼光从历史沿革的角度为解释的结论提供合理性，这不同于探求立法原意的主观解释。 |                                        |
| 比较解释 | 借鉴国外立法与判例以阐明刑法规范的含义。                     |                                        |
| 体系解释 | 指根据体系逻辑来论证解释后的含义在刑法体系中是否协调合理：1．同一用语含义的相对化：同一用语在不同条文中可以保持不同含义；2．不同用语含义的同一化：几个不同的用语也可以保持相同含义；3．同类解释规则：指当刑法列举了相关事项的同时又设置了概括性规定时，对于附随于确定性词语之后的概括性词语，应当根据确定性词语所涉及的同类事项确定其含义及范围。 |                                        |
| 当然解释 | 1．出罪举重以明轻：不要求有法条规范，允许有利于行为人的类推解释。2．入罪举轻以明重：前提是有法条规范，否则会违反罪刑法定。 |                                        |

## 考点5．非实行行为

| 生活行为       | 生活行为对法益没有创设法律不允许的危险。【提示】本质行为人客观上并没有对被害人实施直接、现实、紧迫的危险行为。 |
| -------------- | ------------------------------------------------------------ |
| 降低危险       | 降低危险的行为不是危险行为。                                 |
| 替代危险       | 开创新的危险，比原有危险的危害程度要低。                     |
| 被害人自陷风险 | 判断第一步：危险的实行者、支配者是谁？1．被害人是危险的实行者、支配者，行为人教唆或帮助被害人自陷风险。【判断标准】若被害人满足下面两个条件，则被害人对结果负责，反之由行为人负责。（1）主观上：被害人对危险有认识能力。如被害人是未成年人或精神病患者，则认为其无认识能力。（2）客观上：被害人对危险有控制能力，也就是基于自己的意志自由，对危险具有控制、消除或避免的能力。2．行为人是危险的实行者、支配者，被害人同意行为人的危险行为。【判断标准】若行为人满足下面两个条件，则行为人对结果负责，反之由被害人负责。（1）主观上：行为人对危险有认识能力。（2）客观上：行为人对危险有控制能力，也就是基于自己的意志自由，对危险具有控制、消除 |

或避免的能力。【注意】被害人同意接受实害结果中的死亡或重伤结果，这种同意或承诺是无效的。【总结】关键看危险由谁实行行为，对危险有没有认识＋实质支配控制危险的发展。

| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| ［先前行为］产生的义务                                       | 先前行为的种类                                               | 只要风险是由先前行为创设、增加的，会使法益陷入危险状态，就有阻止义务：①作为、不作为均可成为先前行为；②紧急避险可以成为先前行为；③防卫过当可以成为先前行为，对正当防卫，如果手段适当，但不救助会导致过当结果，则负有防止过当结果发生的义务；④过失、故意行为均可成为先前行为。 |                                                              |

## 考点7．因果关系

| 适用范围         | 过失犯罪，结果加重犯，既遂均需判断因果关系。                 |                                                              |                                                              |          |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | -------- |
| 讨论前提         | 建立在事实查明的情况下，事实未查明，依存疑有利于被告人原则。【考查】①2人故意行为都能导致死亡或其中个人的故意行为能导致死亡，但查不出谁导致的结果，按存疑有利于行为人的原则处理，2人行为与结果之间都不成立固果关系，2人均成立故意杀人未遂；②过失中，若是人行为与结果之间无固果关系，则都不成立犯罪。 |                                                              |                                                              |          |
| 判断条件         | 故意的作为犯                                                 | 重叠因果关系                                                 | 无意思联络的两个行为，各自单独都不能导致结果发生，但会起到重要作用，叠加在一起，同时发挥作用，共同导致结果发生。→两个行为都与结果有因果关系 | 二因一果 |
| 二重因果关系     | 无意思联络的两个行为，各自单独都能导致结果发生，各自同时导致结果发生，两个因果关系竞合在一起，类似双保险。→两个条件与结果都有因果关系【考查】①查明了真相，两者行为都确凿，对于实害结果，如果只算在一个人头上，不公平；②如果没查明真相，则是小前提事实存疑，有利于被告，都违法，但都不担责。此情形题目会有明确提示。 |                                                              |                                                              |          |

## 考点8．正当防卫

| 起因条件     | 现实的不法侵害。1．人的行为（或人的行为的延伸）；2．仅限于那些具有进攻性、破坏性、紧迫性的不法侵害；3．主要是针对有具体受害人的侵害；【假想防卫】现实上不存在不法侵害，行为人误以为存在。 |      |
| ------------ | ------------------------------------------------------------ | ---- |
| 时间条件     | 不法侵害正在进行（已经着手，尚未停止）。【注意】①在财产性犯罪的情况，虽然犯罪行为已经既遂，但在当场还能够挽回损失的情形下，仍然可以实施正当防卫。②对于财产型犯罪的延长时间防卫，由于不法侵害人对被害人的人身危险不存在，不宜实施造成被害人死亡这一严重后果的防卫。③事先安装防卫装置的，只要没有造成重大损害，仍然可以考虑成立正当防卫。 |      |
| 意思条件     | 具有防卫意识，指防卫人对正在进行的不法侵害有明确的认识，并希望以防卫手段制止不法侵害，保护合法权益的心理状态。包括防卫认识和防卫意志两个方面。【观点一】防卫认识必要说（通说观点）。成立正当防卫，既需要有防卫认识，也需要防卫意志，主观上不具备防卫意识的偶然防卫不成立正当防卫，应以犯罪论处。【观点二】防卫认识不要说。 |      |
| 对象条件     | 针对不法侵害人本人。本人是指实施实行行为的人，包括人身与财产。【对第三人防卫与打击错误】甲故意伤害乙，乙情急捡砖头扔甲，击中了甲的同时也击中了无辜的丙，致其重伤。【观点展示】①周先权观点：乙对丙构成正当防卫，一个行为只能认定为同一个性质。成立正当防卫不要求防卫行为达到制止不法侵害的效果；②张明楷观点：乙对丙成立假想防卫，构成过失致人重伤。③柏浪涛观点：只需要看乙对丙是间接故意还是过失致人重伤，然后认定即可。设必要讨论是否为违法阻却事由。 |      |
| 限度条件     | 没有“明显”超过必要限度造成重大损害。【防卫过当】客观须存在过当的结果＋要求主观上至少是过失。 |      |
| 特殊正当防卫 | 对正在进行行凶、杀人、抢劫、强奸、绑架以及其它严重危及人身安全的暴力犯罪，采取防卫行为，造成不法侵害人伤亡的，不属于防卫过当，不负刑事责任。【注意】这是法律注意规定。 |      |

| 共犯的理解     | 标准要素                                                     | 1．客观行为是否相同；2．主观故意是否相同；3．触犯罪名是否相同。 |                                                              |

考点15．正犯

| 共同正犯         | 共同实行犯                                                   | 成立条件                                                     | 参与意识＋实行行为→违法是连带的。 |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | --------------------------------- |
| 参与行为         | 对法益具有不可或缺的重要作用也可以成为共同正犯。             |                                                              |                                   |
| 参与意识         | 1．必须意识自己在参与他人的违法事实；【注意】参与意识≠正犯自己的犯罪故意。2．一方有参与意识，一方没有：片面共犯；3．双方都没有参与意识：同时犯，不构成犯罪。 |                                                              |                                   |
| 法律后果         | 1．责任承担原则：部分实行，全部责任。2．在狭义共犯中也是如此，除非共犯关系出现过剩或脱离。 |                                                              |                                   |
| 共谋共同正犯     | 成立条件                                                     | 行为人的谋议行为对于共同犯罪起到了实质的支配作用。           |                                   |
| 与帮助犯的区别   | 对于共谋过程中随声附和，又没有参与实行的人，只定为心理的帮助犯。 |                                                              |                                   |
| 与教唆犯的区别   | 前者共同谋议，并决定共同实行；后者一方有犯意，另一方无犯意，经教唆后另一方才有犯意，而且只有其中一方实行犯罪。 |                                                              |                                   |
| 与间接正犯的区别 | 前者共谋并决定共同实行；后者中一方对另一方具有支配作用，另一方是一方的犯罪工具。 |                                                              |                                   |
| 间接正犯         | 成立条件                                                     | 是否对实行者有支配力，这也是与教唆犯的区别。【注意】解决的是间接的行为人能不能评价为正犯，至于与其他人是否构成共犯另当别论。 |                                   |
| 主观             | 必须是故意                                                   |                                                              |                                   |
| 具体情形         | 1．利用未达刑事责任年龄或无刑事责任能力的人；2．利用他人不具有行为性的举动：如梦游；3．利用他人不知情的行为：（1）他人有过失；（2）他人有其他犯罪故意；4．利用他人无身份、无目的的行为：无身份者不能构成真正身份犯的间接正犯；5．利用被害人的自身行为：指利用、控制、欺骗、强迫被害人的情形。 |                                                              |                                   |

## 考点16．狭义共犯

| 教唆犯           | 成立条件                                                     | 行为对象                                                     | 1．被教唆对象要求具有规范意识、责任能力，但不要求达到责任年龄；2．被教唆对象必须特定，如果不特定就是煽动；3．被教唆对象已有一定犯意的情形：（1）他人已有犯意，此时教唆不构成教唆犯，如对他人起到心理帮助→构成帮助犯；（2）他人已有轻罪犯意，教唆他人犯重罪（与轻罪性质相同）→构成重罪的教唆犯； |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
|                  |                                                              |                                                              | （3）如果他人已经有犯重罪的意图，教唆他人犯轻罪（与重罪性质相同）→则构成轻罪的帮助犯，因为对轻罪的实施起了心理上的帮助作用；（4）他人已经有实施基本犯的意图，教唆他人实施基本犯基础上的加重犯→构成基本犯的心理帮助犯；（5）他人打算将来实施犯罪，教唆他人现在就实施→成立教唆犯；（6）他人打算在条件具备时犯罪，为他人创造了条件，促使他人犯罪→成立教唆犯。 |
| 教唆行为         | 指引其他人实施客观违法行为的行为。【注意】①“客观”不要求实行犯主观有故意；②“违法”实行者实施的行为具有法益侵害性。 |                                                              |                                                              |
| 主观             | 必须有教唆的故意                                             |                                                              |                                                              |
| 与间接正犯的区别 | 区别                                                         | 1．客观上：教唆犯对实行者没有支配力，间接正犯有；2．主观上：教唆犯有引起他人实施客观违法行为的故意，间接正犯有支配利用他人实施客观违法行为的故意。 |                                                              |
| 联系             | 二者具有包容评价的关系，间接正犯可以包容评价为教唆犯。【注意】一些真正身份犯，不具备身份的人构成间接正犯，但不能定此罪，可定此罪的教唆犯。 |                                                              |                                                              |
| 帮助犯           | 成立条件                                                     | 帮助行为                                                     | 1．方式：（1）包括物理、心理；（2）包括作为和不作为；2．时间：（1）事前；（2）事中；（3）事前约定事后帮助；3．帮助行为与结果具有因果关系，起到了实质性促进作用；4．帮助行为具有法益侵害性（1）帮助行为降低危险性，不成立帮助犯；（2）中立行为不在于知不知道实行者要犯罪，而在于帮助行为是否起到现实紧迫的促进作用。5．帮助行为的客观侵害性：只要正犯的行为是符合构成要件的违法行为，即使正犯无故意，以帮助故意实施帮助行为者，也成立帮助犯。 |
| 帮助故意         | 1．帮助者要认识到实行行为的内容；2．如果对自己帮助的结果没有故意，而是过失，不构成帮助犯。 |                                                              |                                                              |

## 考点17．共犯的特殊形式

| 承继的共同犯罪 | 成立时间                                                     | 1．原则上，在既遂前如入。2．如果参与进来共同实行（作用大）就是承继的共同正犯，如果仅仅（作用小）进行帮助，就是承继的帮助犯。【注意】不存在承继的教唆犯。 |                                       |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------- |
| 行为性质       | 1．原则上，后行为人参与的行为性质和前行为人参与的行为性质相同；2．在结合犯的情形下，后行为人仅参与后一犯罪的，其不成立结合犯，单独定后一罪。 |                                                              |                                       |
| 责任           | 基本原则                                                     | 1．后行为人只对参与后与其行为有因果关系的结果负责；          |                                       |
|                | 承担                                                         |                                                              | 2．先行为人对前后的行为结果均要负责。 |
| 延伸原则1      | 能证明后行为人参与后共同引发结果的，两个人均对此结果担责     |                                                              |                                       |
| 延伸原则2      | 后行为人参与进来后，只能证明结果是一个人造成的，但无法查明具体是谁，那么直接归责于前行为人（前后结果均要负责），后者不担责（存疑有利于被告）。 |                                                              |                                       |
| 既遂之后的参与 | 原则                                                         | 犯罪既遂之后不再成立共犯，只可能成立窝藏包庇罪和掩饰隐瞒犯罪所得罪。 |                                       |
| 例外           | 1．在继续犯的情形中，既遂后参与进来，因为继续犯在犯罪既遂后，违法行为一直持续，所以既遂后参与进来仍然可能成立共同犯罪，如参与进来作用大的，成立承继的实行犯，作用比较小的，成立承继的帮助犯；2．在结合犯的情形中，以绑架罪为例，绑架罪的实行行为是非法拘禁，因此绑架罪属于继续犯，所以在其既遂后参与非法拘禁行为（或索财），成立绑架罪的承继的共同犯罪；若参与进来后只参与了后一杀人行为，则不成立绑架罪的承继的共同犯罪，仅成立后一犯罪，故意杀人罪。 |                                                              |                                       |
| 片面共同犯罪   | 含义                                                         | 一方认识到两人共同犯罪，另一方未认识到的情形。               |                                       |
| 分类           | 片面实行                                                     | 1．肯定说：认为片面的实行者构成片面共同正犯；2．否定说：认为片面实行者不构成片面共同正犯，那是因为在片面实行的场合，片面犯这一方的行为已经完全可以独立定罪，不需要像片面教唆帮助一样，必须得挂靠到实行一方。 |                                       |
| 片面教唆       | 片面教唆是否构成片面教唆犯，存观点展示，肯定说是多数说。     |                                                              |                                       |
| 片面帮助       | 1．片面共犯说明，成立共同犯罪所要求的意思联络可以是双向或是单向的；2．片面帮助犯的帮助只限于物理性的帮助，不能是心理性的帮助。 |                                                              |                                       |
| 责任承担       | 1．知情一方与不知情一方成立共同犯罪，对知情一方适用共同犯罪的规定；2．不知情一方与知情一方不成立共同犯罪，不知情一方不适用共同犯罪的规定。 |                                                              |                                       |

## 考点41．盗窃罪

| 行为模式       | 数额较大                                                     | 成立                                                         | 主观上认识到非法占有之物属于数额较大之物＋实施了盗窃（包括预备、实行） |

| 处罚           | 原则成立就处罚，但实际上对预备、未遂、中止并不一定追究刑责，只对本罪既遂绝对追究刑责。【例外】基于司法解释的规定，以数额巨大的财物（3万元）或珍贵文物为盗窃目标的，盗窃未遂也须刑事处罚。 |                                                              |                                                              |
| 多次           | 1．计算次数的标准：同一时间、同一地点、同一被害人，违反三个“同一”，就是另一起盗窃；2．不要求每次行为都成立盗窃罪，更不要求既遂；3.2年内3次盗窃，应认定为多次盗窃。 | 对于该4类型盗窃，成立本罪只要行为人带着非法占有值得刑法保护的财产的目的去实施相应行为即可，不要求必须既遂。 |                                                              |
| 入户           | 是一种行为类型，而不是加重情节，区别于抢劫罪。               |                                                              |                                                              |
| 携带凶器       | 1．包括性质和用法上的凶器；2．不要求显现使用，如果使用就成立抢劫罪；3．要求具有危及他人人身安全的危险。 |                                                              |                                                              |
| 扒窃           | 1．要求在公开场合，不要求秘密性；2．要求是他人随身携带的的财物，包括贴身，但不包括近身。 |                                                              |                                                              |
| 既遂标准       | 取得控制说：行为人建立起了对财物的占有，已经将财物置于自己的控制范围内。 |                                                              |                                                              |
| 特殊盗窃       | 盗用机动车                                                   | 盗用他人机动车，导致机动车丢失或非法占有的，成立盗窃罪或和其他犯罪并罚。 |                                                              |
| 盗窃（借）欠条 | 1．普通型盗窃的，不论得手与否，只要被害人的财产没有因此而减少，就不成立盗窃罪；2．入室、多次、扒窃型盗窃欠条的，均成立盗窃罪；只要带着非法占有值得刑法保护的财物的目的实施四类型行为即可成立本罪。 |                                                              |                                                              |
| 盗打电话、盗用 | 1．以牟利为目的，盗接网络、复制他人网络号码或明知是盗接、复制的电信设备、设施而使用的，成立本罪； |                                                              |                                                              |

|                    | 网络                                                         | 2．将电信卡非法充值后使用，造成电信资费损失数额较大的，成立本罪；3．明知是非法充值后就相当于给电信公司设置了债权，按流程需要购买才可使用，没有支付对价直接使用，成立本罪；4．盗窃他人上网账户、密码上网，造成资费数额损失较大的，成立本罪；5．以虚假、冒用的身份证件办理入网手续并使用移动电话，造成电信资费损失较大的，成立诈骗罪；6．盗窃银行卡，形不成稳定的占有，不成立本罪。 |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 微信支付宝信息资料 | 1．盗用他人微信、支付宝信息资料→一律定盗窃罪；【张明楷观点】【注意7】微信、支付宝的信息资料≠银行卡信息资料，不适用2018司法解释。【注意2】盗窃手机破解相关账号进行盗划、盗刷，前后盗窃属于连续犯，定一个盗窃罪，数额累加。2．盗用他人蚂蚁花呗→一律定盗窃罪；--理由同上【张明楷观点】3．盗用他人蚂蚁借呗→一律定贷款诈骗罪【提示】借呗服务中不存在信用卡问题，不适用2018司法解释。 |                                                              |
| 罪数               | 1．盗窃普通财物后出卖、毁坏财物的，属于事后不可罚的行为，但是如果通过销售行为欺骗他人财物的，成立诈骗罪，与盗窃罪并罚；2．盗窃文物、假币、毒品、淫秽物品等违禁品后又持有、出售的，数罪并罚；3．盗窃文物后毁坏的，数罪并罚。 |                                                              |

| 主观                                                | 非法占有为目的：该目的必须是形成自己占有的事实之后产生的。   |                                                              |           |                    |

## 考点43．抢劫罪

| 构成要件               | 行为结构                                                     | 1．带着抢劫的故意→对人实施暴力、胁迫等强制手段→2．压制被害人反抗，被害人不能、不知、不敢反抗→3．被害人因无法反抗而放弃财物→4．行为人取得财物 |
| ---------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 抢劫对象               | 1．抢劫的对象是数额不大因但是值得刑法保护的财物；2．特别的对象问题：（1）赌资、赌债：抢劫赌资、赃款赃物的，定抢劫罪；如果抢劫自己所输赌资、所赢赌债的，一般不以抢劫罪论处；（2）抢劫到期合法债务的，不以抢劫罪；（3）为个人使用，抢劫家庭成员或近亲属的，不成立本罪；但教唆或帮助他人抢劫家庭成员或近亲属的，可以以本罪论处；（4）14到16周岁的人，强拿硬要少量物品，一般不以本罪论处。（5）欠条①抢劫欠（借）条，有抢劫行为，即成立抢劫罪，但并不既遂（假设未造成人身伤害的）。②抢劫欠（借）条后，又逼迫债权人在收条上签字，达到免除债务目的的，成立抢劫罪既遂。 |                                                              |
| 行为                   | 暴力                                                         | 1．不法使用有形力：殴打、捆绑、伤害、禁闭、故意杀害（爆炸、放火）；2．对无关的第三人实施暴力取得财物，不成立犯罪。 |
| 胁迫                   | 要求以暴力恶害相威胁。1．不包括非暴力的恶害相威胁；2．以暴力相胁迫，并不要求行为人真正具有有加害能力或加害意思，只要使被害人认为有发生可能即可；3．暴力和胁迫的对象：（1）只包括对人实施，不包括对物实施；（2）对象包括财物的占有者、占有的辅助者、财物的（准）保护者（具有保护意思和保护行为的人），不包括无关第三人。 |                                                              |
| 其他方法               | 本质是使被害人不知、不能反抗。                               |                                                              |
| 法定刑升格条件         | 入户抢劫                                                     | 1．入户的含义（1）入：要求必须具有非法性，也即进入时必须带着实施人身或财产犯罪的非法目的；（2）户：指与外界相对隔离的家庭住所，必须侵犯了家的安宁。2．必须是在户内实施了暴力、胁迫等抢劫行为。【注意】入户后实施的抢劫包括非法入户后转化来的抢劫。3．必须认识到是他人的家庭住所。 |
| 在交通工具上抢劫       | 1．具有公共性和营运性的大中型交通工具，在非运营的交通工具上抢劫不属于交通工具上抢劫；2．必须处于运营状态；3．包括在公共交通工具上转化来的抢劫；4．在公共交通工具上携带凶器抢夺，虽然成立抢劫罪，但不属于在公共交通工具上抢劫。 |                                                              |
| 抢劫银行或其他金融机构 | 1．必须是抢劫银行自己的经营资金，不包括办公用品等物品，也不包括储户身上的资金；2．抢劫正在使用中的运钞车的，视为抢劫银行或其他金融机构。 |                                                              |
| 多次抢劫或抢劫数额较大 | 1．多次指的是3次以上；2．应当以每次抢劫均成立犯罪为前提。    |                                                              |
| 抢劫致人重伤死         | 1．对于重伤和死亡的加重结果，主观上可以是故意也可以是过失；  |                                                              |

|                                | 亡的                                                         | 2．必须是抢劫行为本身导致的；3．不限于被害人，还包括前来来阻挡的第三人，以及行为人的同伙。 |
| ------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 冒充军警抢劫的                 | 1．包括无此身份冒充有此身份，也包括有此种军警身份冒充他种军警身份。2．判断标准：结合穿着、持有证件等判断是否足以使他人误以为行为人是军警人员。 |                                                              |
| 持枪抢劫的                     | 1．枪必须是真枪，但不要求是实弹；2．枪必须向被害人展示或使用。 |                                                              |
| 抢劫军用物资或抢险救灾救济物资 |                                                              |                                                              |
| 事后转化抢劫                   | 法律拟制                                                     | 犯盗窃、诈骗、抢夺罪后，为窝藏赃物、抗拒抓捕或毁灭罪证而当场使用暴力或以暴力相威胁，以抢劫罪论处。 |
| 成立条件                       | 主体                                                         | 满16周岁的人。【注意】14周岁＜年龄＜16周岁的，不转化。       |
| 三大行为                       | 盗窃、诈骗、抢夺1．只要行为具有财产犯罪的属性，且符合三罪的构成要件即可。2．三罪只要求着手，不要求既遂。 |                                                              |
| 三大目的                       | 使用暴力是为了窝藏赃物、抗拒抓捕或毁灭罪证。                 |                                                              |
| 当场使用暴力或暴力相威胁       | 当场                                                         | 时间上的当场性，即要求三行为和暴力使用之间具有时间联系的紧密性。 |
| 使用暴力                       | 1．暴力对象不限于被害人，包括其他妨碍人，但不能是加害人自己或动物；2．要达到足以压制反抗的程度，与抢劫罪的暴力相当；【注意】对于以摆脱的方式逃脱抓捕，暴力强度较小，未造成轻伤以上后果的，可不认定为“使用暴力“，不以抢劫罪论处。--司法解释3．暴力致对方重伤、死亡的，成立转化抢劫的结果加重犯；4．关键词是当场使用暴力。也就是说暴力的方式不影响转化，如果暴力的方式还渉及其他犯罪，就要注意评价。 |                                                              |
| 罪数                           | 法律拟制                                                     | 携带凶器抢夺的、事后转化的抢劫罪。                           |
| 结果加重                       | 抢劫行为本身致人重伤、死亡的。                               |                                                              |

## 考点45．诈骗罪

| 行为结构             | 欺骗行为→被害人产生或维持错误认识→被害人基于错误认识处分财产→行为人取得财产→被害人损失财产 |                                                              |                                                              |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 构成要件             | 欺骗行为                                                     | 1．使对方从无到有产生错误认识→作为方式；2．维持对方已有的错误认识→不作为方式（前提：行为人须负有告知义务，且告知义务的来源与作为义务相同） |                                                              |
| 受害人产生或维持错误 | 1．受骗者必须是具有意思自治能力的人；【注意】欺骗儿童、精神病人是盗窃；机器不能被骗。2．受骗者有所怀疑，行为人进一步强化其认识错误的，成立诈骗。 |                                                              |                                                              |
| 处分财物             | 处分行为                                                     | 1．处分指的是将财物脱离被害人后让行为人实现排他性的占有。包括直接交付、承诺转移财产性利益以及免除行为人债务等，但不包括出借行为；2．处分的方式包括受害人直接转移占有给行为人和受害人在行为人可以支配的领域内放弃占有。 |                                                              |
| 处分意识             | 1．被害人有意思自治能力和意志自由；2．害人对财物有具体的、充分的认识，知道自己在处分财物。 |                                                              |                                                              |
| 取得财物             | 既可以是行为人本人取得，也可以是第三人取得。                 |                                                              |                                                              |
| 被害人遭受损失       | 此损失应侧重于单向个别评价，要考虑被害人处分财物的交易目的是否实现。1．无效债权的丧失，不算财产损失，不成立诈骗罪；2．即使行为人提供了相当对价，但是未实现被害人处分财物的目的的，成立本罪。3．捐款型诈骗，如果行为人没能实现被害人捐款目的的，也成立诈骗。 |                                                              |                                                              |
| 既遂                 | 1．要求被害人有遭受财产损失的危险性和可能性；2．要求行为人基于欺骗最终取得财物，被害人遭受损失。 |                                                              |                                                              |
| 罪数                 | 法条竞合                                                     | 与合同诈骗罪和金融诈骗罪                                     |                                                              |
| 牵连犯               | 为了诈骗而伪造有关证件的，择一重，只定诈骗罪。               |                                                              |                                                              |
| 想象竞合             | 1．侵犯著作权罪：制作销售假冒他人署名的美术作品；2．招摇撞骗罪：冒充国家机关工作人员诈骗；3．销售伪劣产品罪：如果销售的是毫无价值的商品，仅成立诈骗罪。 |                                                              |                                                              |
| 区分                 | 与敲诈勒索罪的区分                                           | 1．诈骗是被害人基于错误认识处分财产，敲诈勒索是被害人基于恐惧交付财物；2．诈骗罪中虚构的事实含有暴力恶害的内容时，两罪可想象竞合。 |                                                              |
| 与盗窃罪的区分       | 1．两角关系比较：关键在于判断受害人有无处分财产的意思和处分财产的行为，有就是诈骗，无就是盗窃；2．三角诈骗与盗窃罪间接正犯比较：关键在于受骗人是否具备处分权，是否和处分人重合。（1）三角诈骗：行为人→受骗人（同时须是处分人）→受害人；（2）盗窃罪间接正犯：行为人→受骗人（不是处分人）→受害人。 |                                                              |                                                              |
| 三角诈骗             | 1．三角诈骗和两角诈骗的区分（1）三角诈骗受骗人和受害人非一人；（2）三角诈骗的受骗人只能是受害人的辅助占有人，不能是占有人。2．三角诈骗的类型：（1）直接正犯：干洗店老板甲来到乙家，对保姆丙说，你家主人让我来拿衣服干洗，保姆丙不知情，把衣服拿给了甲，在这个三角诈骗关系里，甲成立 |                                                              |                                                              |
|                      |                                                              |                                                              | 诈骗罪的直接正犯。（2）间接正犯：干洗店老板甲欺骗员工丁说，乙某来电说让我去他家里那件衣服干洗，丁来到乙家，对保姆丙说，你家主人让我来拿衣服干洗，丙不知情，便把衣服拿给了丁，甲是三角诈骗的间接正犯。 |
| 与侵占罪的区分       | 行为人一开始是否具有非法占有目的。1．事前骗：行为人开始就有非法占有的故意，欺骗对方转移财物给自己，成立诈骗罪；2．事后骗：（1）行为人一开始无非法占有的目的，在正常保管他人财物后，才产生不法占有的的目的，据为己有的，成立侵占罪；（2）行为人构成侵占罪后，被害人请求返还时，行为人又采取欺骗的手段免除债务的→【观点展示】观点一：事后欺骗是为了维持占有的事实，属于事后不可罚；观点二：事后欺骗属于诈骗财产型利益，成立诈骗罪，择一重处罚（吸收犯）。 |                                                              |                                                              |

| ------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 使用方式            | 1．使用伪造的信用卡（包括变造）；2．使用以虚假身份证明骗领的信用卡（先骗领再使用）；3．使用作废的信用卡； | （1）这三种行为使用的信用卡是非真实（假）的卡；（2）包括自己／利用他人使用；（3）须以信用卡通常的使用方式使用。如果利用假信用卡进行质押担保，只可能构成普通诈骗罪或合同诈骗罪。 |

## 考点65．逃税罪
